# 每日新聞 podcast — 2026年4月29日（週三）— EP56

歡迎回來，今天是4月29日週三，這集 EP56。今天有兩條主線值得你花時間：第一條是地緣政治的「禮儀外交與槍擊餘波」，英王查理三世到白宮，國宴規格拉到 2007 年來最高；同一週白宮記者協會晚宴的槍擊嫌犯被起訴企圖暗殺總統，這兩件事疊在一起，看起來是巧合，其實是 Trump 政府在重新調度形象。第二條是科技端的「平台分流」，NVIDIA 推出新的多模態模型，Anthropic 把 Claude 拉進 Adobe、Blender、Autodesk 這些創作工具，但同一天，Google 簽下了一份五角大廈的機密合約，條件是 AI 安全濾鏡可以由政府調整，Anthropic 拒簽。一邊是創作工具串連，一邊是國家機器選邊站，今天就從這兩條線進去。

先講時事。

第一個主場是白宮的英王訪美。4月28日 Trump 在白宮東廳設國宴款待 Charles 三世跟王后 Camilla，這是自從 2007 年小布希款待伊麗莎白女王之後，白宮第一次舉辦正式白領帶國宴。Charles 同一天到美國國會聯席會議發表演說，這是英國君主史上第二次在國會發言。演說內容你大概可以想像，沒什麼意外，重點集中在英美兩國同源的歷史脈絡，還有未來合作的方向。Trump 公開稱演說「very great」。

不過你要看的不是儀式本身，而是時間點。WHCD 也就是白宮記者協會晚宴的槍擊案才剛過 4 天。同一天 4月28日，那位 31 歲的加州教師兼工程師，叫 Cole Tomas Allen，被聯邦法院正式起訴 3 項重罪，包括「企圖暗殺總統」。檢方拿到一份他犯案前幾分鐘寄給家人的聲明。Trump 政府的做法是直接拿這個案子做槓桿，公開推動白宮舞廳的新建工程，理由就是「保護總統免於暗殺攻擊」。批評者指出，這是繞開 GSA 公開審查程序的政治操作。

把這兩件事併起來看：王室造勢的場面是給澳洲、加拿大、紐西蘭、英國這個「Anglosphere 軸心」打信號，要鞏固對抗中俄的協同陣線；同時白宮舞廳工程則是把一個安全事件直接轉成基建合約。Trump 政府很懂得把每一個衝突轉成政治資本，這次也不例外。

第二個重點往北看，加拿大 Carney 端出春季經濟更新，大動作是設立加拿大史上第一個主權財富基金，叫 Canada Strong Fund，初始聯邦出資 250 億加幣，跟民間資本配對，定向投資戰略產業跟重大基礎建設。同一天還包了一輪新的油氣補貼，包含碳捕集稅務抵免，要拉動 Alberta 的油砂產量。Carney 這個策略其實很明白：用主權基金把油氣稅收跟礦產收益鎖進長期投資池，對沖大宗商品景氣循環，讓加拿大不要繼續被大宗價格抓著走。

但這套組合也有反對聲音。環保社群跟 NDP 不買單，他們質疑碳捕集補貼根本是給化石燃料產業續命的政治禮物，這跟 Carney 過去幾年在 Bank of England 推碳定價的立場是矛盾的。保守黨則從另一個角度開槍，說 250 億的規模對加拿大現在的財政承受能力過大。對 Carney 來說，這份春季更新等於是他上任後第一份完整的經濟綱領，市場關心的是「有沒有錢」，社運關心的是「補貼了誰」，兩邊在不同的指標上看同一份文件。Carney 同時公布了一個外交動作：5月2日到4日訪問亞美尼亞葉里溫，出席第 8 屆歐洲政治共同體峰會，是首位非歐洲領袖獲邀加入這個機制，意義在於加拿大要把自己定位成「歐洲體系的太平洋延伸」，這跟主權基金的長期布局是同一條線。

第三個主場到南邊。墨西哥 4月28日宣布逮捕 Jalisco New Generation Cartel，CJNG，第二階級的高階首腦 Audias Flores Silva，綽號叫 El Jardinero——「園丁」。墨西哥動了 500 多名兵力突襲 Nayarit 州把人帶走。Flores Silva 是 CJNG 老大 Oseguera Cervantes 的安保總管，掌管 Nayarit、Jalisco、Mexico State 跟 Zacatecas 四州的毒品生產跟運輸網路。Bloomberg 揭露這次突襲動用了美方 CIA 級的情資。

但事情有第二層。Sheinbaum 4月27日已經就 Chihuahua 州的事件發了一份正式外交照會。背景是：兩名 CIA 探員 4月19日在 Chihuahua 一場反毒行動中因為車禍喪生。Sheinbaum 警告美方，未經授權的美方人員不得在墨西哥領土執行行動。Sheinbaum 的紅線講得很清楚：情資合作可以，人員進場執行不行。Trump 政府那邊，傾向把 CJNG 直接劃進 Foreign Terrorist Organization 名單，要求墨方放寬作戰權限。所以你會看到一個矛盾：戰術層面美墨合作還在，El Jardinero 就是合作成果；戰略層面 Sheinbaum 在重新定義主權邊界，把美方的可動作空間框起來。這兩個層面平行運作，問題會堆到下一個衝突點再爆。

第四件大事是日銀。4月28日日銀政策會議結束，維持基準利率 0.75% 不動，但這次的 6 票對 3 票，是 Ueda 任內最大的內部分裂。投反對票的三個人——Nakagawa、Takata、Tamura——主張一次升到 1.0%。日銀同步下修本年度 GDP 預測，從 1.0% 砍到 0.5%，差不多砍一半；同時上調本年度核心 CPI 預測到 2.8%，比市場預期高。Ueda 的講法是，「中東戰爭外溢、預測確定性下降」，所以維持不動，但他沒關上 6 月升息的門。

市場讀出來的訊號是「鷹派的 hold」。隔夜 swap 押注 6月16日升息機率拉到大約三分之二。日圓對美元在記者會後短線逆轉，JGB 殖利率全期上揚。對你來說重點是：6 比 3 這個分裂，其實是日銀內部從共識制走向博弈制的一個分水嶺。Ueda 任內第一年大家都跟他走，現在三票直接跟他唱反調，下一次會議只要油價再壓下來、伊朗戰爭看到出口，這三票就有機會變成多數。

第五件事再回到歐洲。英國 Starmer 4月28日在下議院迎來一場政治攻防戰：MP 辯論要不要把他移送特別委員會，理由是 Mandelson 的駐美大使任命過程出問題。具體說，是 Starmer 之前在國會說「due process 已經走完」，但實際上 Mandelson 的安全審查還沒做完，當時的 Cabinet Secretary Simon Case 已經建議延後公布。Starmer 9 月已經把 Mandelson 解職，因為新揭露了 Mandelson 跟 Epstein 的友誼細節。前幕僚長 Morgan McSweeney 4月28日同一天公開致歉，承認推薦 Mandelson 是「嚴重錯誤」，但否認施壓官員忽略安全顧慮。保守黨開砲，說這是內閣級系統性失當，Starmer 應該主動辭職而不是用委員會程序拖延。Starmer 滿意度已經是戰後新低，這次調查投票過不過，會直接影響 Labour 政府的存活機率。

第六件事，巴西 10 月大選二輪民調。AtlasIntel 4月22到27日訪問 5,008 名選民，誤差正負 1 個百分點，結果 Flavio Bolsonaro 拿 47.8%、Lula 拿 47.5%，差距在誤差範圍內，統計上平手。從 12 月那個 12 個百分點的差距，到現在的平手，這條曲線你要記著。Flavio 接續他被軟禁的父親 Jair 4月16日的背書效應，民調動能還在，跟你預期的世代交替相反，老 Bolsonaro 的影響力反而在他被困住之後更純粹地轉移到兒子身上。Lula 這邊，巴西 9 到 10% 的通膨疊加阿根廷外溢震盪，把他從進攻拉回守勢。市場 BRL 跟 Ibovespa 對 Bolsonaro 勝率上升都已經有反應了。

第七件事，菲律賓 Duterte 在 ICC 的案子今天有新進度。ICC 4月28日正式公告 Duterte 案的審判分庭組成：英國的 Joanna Korner、南韓的 Keebong Paek、法國的 Nicolas Guillou，三位法官就位。4月23日 pre-trial chamber 已經一致通過反人類罪三項控罪，今天進入審判前置程序。涵蓋期間從 2011 年 11 月 1 日到 2019 年 3 月 16 日，橫跨他擔任公職的整個時期。實際開審日要看接下來的 status conferences 排期，預估在 2026 年 10 月到 2027 年 2 月之間。

第八件事，韓國李在明在 4月28日的內閣會議上提出一個新詞：「相互尊重外交」。他說盟友間的問題應該基於相互尊重跟基本原則來解決。這句話表面溫和，但脈絡裡是直接回應美方對情資共享、防衛自主、還有對美企管制的施壓。李在明同步要求啟動「主權自信外交」跟「無外國駐軍時的自主防衛準備」，這個措辭被解讀成是在為 USFK 也就是駐韓美軍的長期變數做預設。同一天他批准 AI 政策秘書 Ha Jung-woo 跟發言人 Jeon Eun-soo 為了投入補選辭職。整體而言，李在明在用更克制的語言重畫他的對美姿態。

接下來是延續區，今天有幾條線繼續在走。伊朗那邊，第 59 天，伊朗最新提案是用「美方解除海軍封鎖、結束戰爭」交換 Hormuz 海峽重啟，但拒絕對核計畫即時讓步；白宮 4月28日表態存疑，油價跳升、股指下挫。以黎停火這幾天又走到崩盤邊緣，IDF 對南黎多座村莊發強制撤離令、Hezbollah 隧道被大規模引爆，美方公開要求以色列「壓住反應」。烏克蘭那邊，無人機擊中 Tuapse 油庫，火勢沒撲滅又被續擊；俄烏雙向無人機數量分別是 94 跟 123 架。歐盟 4月28日把 20 號制裁包執行細則發布，第一次對「便利規避制裁的第三國」吉爾吉斯啟動 anti-circumvention 機制，愛沙尼亞外長表態 21 號制裁包重點是俄羅斯能源資源。匈牙利那邊，Sulyok 總統 4月21日已經敲定，5月9日召開國會新會期首日，當天同步選任跟宣誓 Magyar 為新總理，1990 年來第一次採用「同日選任就職」。阿根廷 Milei，IMF 4月14日完成第二次審查、釋出 10 億美元，同時把阿根廷 2026 GDP 預測從 4.0% 砍到 3.5%；Q1 通膨累計 9.4%，已經吃掉政府全年預算 10.1% 目標的 93%；Milei 滿意度跌到 35%。蘇丹 el-Fasher，RSF 拘留逾千平民，Sudan Doctors Network 形容條件「dire」；WHO 估算全戰場死亡逾 4 萬。賴清德訪 Eswatini 4月21日正式取消，原因確認是模里西斯、馬達加斯加、塞席爾以「一中原則」為名撤銷越領空權。AUKUS 那邊，英國下議院國防委員會 4月28日報告，點名 AUKUS 交付不力，呼籲 Starmer 政府跟澳、美緊急協調以避免關鍵里程碑被錯過。Tamil Nadu 邦選 4月23日完成投票、84.69% 投票率寫下獨立後新高，5月4日上午 8 點開始計票。最後一條輕的：Ruto 4月28日在 Nairobi 礦業投資論壇澄清「奈及利亞英文需要翻譯」這句話被脫離脈絡，稱奈及利亞人是「我的姻親」，請奈及利亞礦業部長 Alake 代為向 Tinubu 致意。Alake 同場回敬「奈及利亞人講得一口好英文」，事件兩天降溫。

時事大致就到這。接下來我們把鏡頭轉到科技。

科技端今天的主題我前面已經點到，「平台分流」。先看 NVIDIA。

NVIDIA 4月28日釋出 Nemotron 3 Nano Omni，這是一個 30B 的 MoE 多模態模型，叫 hybrid MoE，每 token 啟用 3B 參數，context 拉到 256K。簡單講，它把視覺、語音、文字、影片這四個模態整合進同一個 30B 的 MoE 模型裡。NVIDIA 主打的場景是 edge agent：文件分析、自動語音識別、長 audio-video 推理、agentic computer use 也就是讓 agent 直接操作電腦。發布一上線就有 Foxconn、Palantir、Eka Care、ASI 採用，Dell、Oracle、Docusign、Infosys、Zefr 在評估。它同步上架 Hugging Face、OpenRouter、build.nvidia.com，加上 25 個以上 partner 平台。

要小心的點：NVIDIA 主打的「9 倍吞吐量」這個數字，是用他們自己挑的對手跟精度配置去比的，獨立評測還沒出來。30B MoE 在 omni 賽道現在已經是擁擠戰場，Qwen3-Omni、MiniCPM-o 都打同一塊，所以這個版本能不能跑出真實的成本優勢，要看下個月開發者把它接到實際 production 之後的回饋。

第二個是 Anthropic 的 Claude for Creative Work。同樣 4月28日同步公布，Anthropic 一次拉了五家進來：Adobe、Blender、Autodesk Fusion、Ableton、Splice。Adobe 那條 connector 拉進 50 個以上 Creative Cloud 工具的素材，包括 Photoshop、Premiere、Express。Blender 走自然語言對 Python API。Autodesk Fusion 用對話建 3D 模型。Ableton 把 Live 跟 Push 的官方文件灌進 Claude，做 grounding。Splice 給音樂製作者用。教育端，Anthropic 同步跟 RISD 也就是羅德島設計學院、Ringling College、還有 Goldsmiths，三所藝術學院的計算藝術課程合作。

不過 Anthropic 自己還有一個尷尬要處理。同一週 Claude system prompt 出 bug 的事情，已經在 GitHub Issue #49363 上被使用者報出來，HN 也跟著起飛。Bug 細節指向 system prompt 配額計算錯誤，把 managed agent 直接 brick，更糟的是會多扣付費用戶的錢。Anthropic 4 月才推 Managed Agents 商業化，第一個高曝光 incident 就是「真的扣到錢」，對 enterprise 採用是負面訊號。所以 Claude for Creative Work 連線講起來漂亮，使用者真正會問的是：你連線連得這麼廣，但你連自己 system prompt 配額都算錯，creative pipeline 出問題誰負責？

第三個是 Google 簽五角大廈。Reuters 4月28日揭露，Google 同意美國國防部在機密網路上把 Gemini 用於「任何合法政府用途」，而且必須「按要求協助政府調整公司 AI 的安全設定跟濾鏡」。合約跟 OpenAI、xAI 既有的五角大廈合約並列。措辭裡保留了一句「不得用於本土大規模監控或無人類監督的自主武器目標選擇」，但同條款也聲明 Google 沒有否決政府合法作戰決策的權利。Anthropic 拒絕鬆綁自主武器跟本土監控的護欄之後，被國防部標記為供應鏈風險。

這三件事併起來看，你會看到一張很清楚的地圖。NVIDIA 跟 Anthropic 走 ecosystem 路線，把 model 接進其他人的 production 工具；Google 走 vertical 路線，直接把 model 接進國防系統。Anthropic 拒簽這件事之所以重要，不是因為它會改變什麼具體的軍事決策，而是它證實了一件事——所謂「可調整的安全濾鏡」，本質上就是把 model alignment 的政策外包給政府，誰要 model alignment 鬆得開，誰就拿到合約。

第四件事是 Hugging Face 的 Transformers 5.7.0。這個版本把 Poolside 的 Laguna 系列合進 mainline。Laguna XS.2 是 33B 總參、3B 啟用的 MoE coding model，Apache 2.0 授權，主打「long-horizon agentic coding」——就是寫程式、跑測試、看 failure、然後自製工具。架構上有個創新點，叫 per-layer head counts，不同 decoder layer 可以有不同 query-head 數量。這讓我覺得開源 coding 模型的創新節奏在加快，Poolside 從一個拿錢的 coding agent 公司，現在自己把模型開源，用模型換 distribution。

第五件是 SBF。4月28日 Lewis Kaplan 法官駁回 Sam Bankman-Fried 的重審申請，措辭很直接，說 SBF 的指控是「異想天開的陰謀論」，原文是 wildly conspiratorial。上訴還在進行，但這個駁回基本上把 retrial 這條路堵死了。對加密貨幣產業來說，這是 FTX 案的一個確定性節點。
