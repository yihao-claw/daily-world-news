## 🧠 LLM

• 🔥13 **Hugging Face Transformers v5.7.0 把 Poolside 的 Laguna 系列合進 mainline** — 33B 總參、3B 啟用的 MoE coding model（Laguna XS.2，Apache 2.0），主打 long-horizon agentic coding：寫程式、跑測試、看 failure、自製工具。同時併進架構創新：per-layer head counts，不同 decoder layer 可以有不同 query-head 數量。Laguna M.1 走商用授權路線。

• 🔥10 **talkie-1930-13b：用 1931 年以前歷史英語訓練的「復古語言模型」** — Nick Levine、David Duvenaud、Alec Radford（GPT、Whisper 共同作者）合作的研究專案；base 53.1 GB、260B tokens 訓練量，instruction-tuned 版 26.6 GB 用 pre-1931 reference works 萃取的 instruction-response pairs 微調。學術角度有趣，但實用性接近零——這是「LLM 史學」，不是新一代 frontier model。
  📎 來源：[Simon Willison](https://simonwillison.net/2026/Apr/28/talkie/)

• 🔥13 **ollama v0.22.1**：mlx runner 跨多 sequence batch sampler、tokenizer 修 multi-regex BPE offset、mlx 支援 NVIDIA TensorRT Model Optimizer import、修桌面 app 啟動會殺掉現役 ollama 的 bug。延續軌跡（前一晚 0.21.3-rc0 才釋出，週期已壓到 24h 級）。

• 🔥10 **pip 26.1 推 lockfiles 與 dependency cooldowns** — Richard Si 主導；首次在 pip 內建 lockfile 機制，cooldown 讓你延後採用剛發佈的套件版本（避開 supply-chain attack 高風險窗口）。同時放棄 Python 3.9 支援。Simon Willison 用 Python 3.14 測過。
