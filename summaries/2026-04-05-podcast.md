先從中東開始。

美伊戰爭打到現在第三十六天了，第六週。如果你有在追這個系列，你會知道上禮拜的情況已經夠混亂了——F-15 被擊落、伊朗飛彈轟炸海灣國家煉油廠。但這週末的事情又把緊張度拉到一個新的層次。

川普直接對伊朗下了一個48小時最後通牒。他的原話是：如果荷姆茲海峽在週一之前沒有重新開放，「all hell will rain down」——全面的地獄般打擊會降臨在伊朗身上，目標包括橋梁和電廠。聽起來像是在談判，但實際上更像是在威脅。

伊朗的回應也很有意思。他們的中央軍事指揮部用了一個很特別的措辭——他們說這是一個「無助且緊張的」威脅。你想想看，被最後通牒的那一方，居然回過來說你是「無助且緊張的」，這在外交辭令上其實是一種很強的訊號——他們在暗示美方已經進退兩難了。

從軍事面來看，最新的數字是這樣的：美軍累計陣亡十三人、受傷三百六十五人。上週四，一架 F-15E 在伊朗領空被擊落，飛行員已經獲救了，但武器系統官到現在還失蹤。搜救過程中，又有一架 A-10 在荷姆茲海峽附近墜毀。同一天，伊朗革命衛隊的飛彈攻擊了科威特的海水淡化廠和煉油廠，造成八個人死亡。沙烏地、阿聯酋、巴林的基礎設施也遭到攻擊。週五，布什爾核電廠的一棟輔助建築附近傳出爆炸，具體情況還在調查中。

這件事最值得關注的角度，其實不是軍事行動本身，而是各方的政治算計。

美國這邊，川普在下最後通牒的同時，白宮向國會提交了2027財年的國防預算——一點五兆美元，創美國歷史新高。這個時間點不是巧合，他需要用戰時氛圍來推動這個天文數字的預算。

海灣國家是最痛苦的——他們被夾在中間，自己的民用設施被伊朗攻擊，但又不敢得罪美國。科威特、沙烏地、阿聯酋、巴林四國聯合譴責伊朗，但同時也是最急切呼籲停戰的。

歐洲那邊，馬克宏的態度很直白。他批評川普的戰爭目標「不斷在變」，而且他在亞洲訪問的時候公開說了一句很重的話：「我們的目標不是成為兩個霸權的附庸。」這個「兩個霸權」指的是美國和中國，等於是在暗示歐洲不想被美國的戰爭策略拖下水。

聯合國安理會那邊也在動。一個關於「保障荷姆茲海峽航運自由」的決議案拿去投票了，但俄羅斯、中國、法國三個常任理事國都不買帳——他們擔心決議的措辭會變成美軍軍事行動的法律背書。所以基本上又卡住了。荷姆茲海峽的通行量已經從每天一百五十艘船暴跌到只有十到二十艘。英國召集了三十五個國家開視訊會議，但也沒有什麼實質進展。

明天，四月六號，是川普最後通牒的到期日。同一天，OPEC+ 要開會討論是否放鬆自願減產。如果通過的話，會部分緩解油價壓力。但如果最後通牒到期什麼都沒發生，Goldman Sachs 預測油價可能直接突破一百一十五美元。WTI 目前已經到一百一十一塊多了。

我想特別提一下油價的脈絡。你如果看數字的話，WTI 原油已經從今年年初翻了將近一倍了。荷姆茲海峽這個地方，平常每天有一百五十艘船通過，現在只剩十到二十艘。IEA 已經把這次危機定性為「石油市場史上最大的供應中斷」——比七零年代的石油危機還嚴重。但有趣的是，美國本土的燃油供應其實沒怎麼受影響，因為繞好望角就好了。最痛的是歐洲和亞洲，特別是日本和台灣這些高度依賴中東能源的國家。航空業也很慘，本來今年預計全球航空業會創下四百一十億美元的紀錄利潤，但噴射燃油價格翻倍之後，這個預測基本上作廢了。

好，從中東出來，我們來看歐洲。

距離匈牙利大選只剩一個禮拜了——四月十二號投票。這場選舉被 Politico 歐洲版稱為「歐盟2026年最重要的選舉」，這個說法一點都不誇張。

奧班執政十六年了，他現在面臨的是上台以來最大的威脅。反對黨 Tisza 黨的領袖 Péter Magyar，民調大概在五成左右，領先奧班的 Fidesz 大約十個百分點。Magyar 自己說，這次選舉是「匈牙利在世界中定位的公投」。

為什麼這對歐盟這麼重要？因為奧班這些年來基本上成為歐盟決策的最大障礙。他否決了烏克蘭入盟談判、阻擋了九百億歐元給烏克蘭的低利貸款、拒絕共同庇護政策、繼續進口俄羅斯石油和天然氣。歐盟的外交和國防政策需要全票通過，而奧班一個人就可以癱瘓整個機制。

但也不能太樂觀。智庫 CSIS 的分析點出了一個重要的問題：就算 Tisza 黨贏了，匈牙利深層的制度問題——司法獨立受損、媒體環境惡化——不太可能在一屆政府內修復。所以真正的轉變可能需要很多年。

另一件歐洲的事是馬克宏的亞洲行。他去了日本和韓國，這趟行程最引人注意的是他公開表態的姿態——他呼籲各國建立一個獨立於美中的多邊合作框架。他跟韓國總統李在明會談的時候，主要聊的是氣候、核能跟印太安全。從法國的角度來看，馬克宏一直在嘗試塑造歐洲作為「第三極」的角色，不跟著美國走，也不靠向中國。但現實是，在伊朗戰爭的壓力下，歐洲的自主外交空間其實越來越小。

好，快速帶過幾條亞洲的消息。

韓國現在的經濟狀況蠻令人擔心的。韓元持續貶值，KOSPI 上週單日暴跌了百分之六點五，政府正在準備追加預算案，但在貨幣貶值的壓力下，擴張財政和穩定匯率變成了兩難。

日本那邊呢，FY2026 正式開始了，九兆日圓的國防預算啟動。日本宣布已經達到 GDP 百分之二的國防支出目標，比原訂時程提前了兩年。

台灣有幾件事值得注意。國民黨主席鄭麗文應習近平邀請訪問中國，行程到四月十二號。但整件事非常不透明——會談議程沒有公開，沒有立委同行，搭的是中國航空公司的飛機，媒體採訪只限少數在北京有駐點的機構。民進黨批評這形同「朝貢外交」。

另一件是台灣跟美國簽了新的 LNG 合約，每年一百二十萬噸。這很重要，因為伊朗衝突之前台灣大概三分之一的 LNG 來自卡達，現在已經陸續從澳洲和美國取得替代供應。能源安全在戰爭時期變得格外關鍵。

在南亞，巴基斯坦和阿富汗塔利班政府正在中國烏魯木齊進行和談。北京在斡旋，中國外交部說「磋商正在穩步推進」。不過說實話，之前在卡達和土耳其也談過，結果二月底巴基斯坦就對喀布爾發動空襲，停火瞬間破裂。這次能不能真的談出什麼，還很難說。

以色列入侵黎巴嫩也進入了第二個月。自從三月二號以色列對真主黨發動攻勢以來，已經造成黎巴嫩一千三百一十八人死亡、近一百萬人流離失所——佔黎巴嫩全國人口的百分之二十。以軍已經對南黎巴嫩超過一百個城鎮發出撤離令，而且計劃佔領利塔尼河以南的全部地區。上週五，以軍空襲損壞了提爾市的一間醫院。加拿大、法國、德國、義大利、英國五國都呼籲以色列避免擴大地面行動，但以色列國防部長的回應是：只要真主黨還是威脅，就不允許流離失所者返回。這個態度基本上等同於長期佔領。

非洲那邊有一件讓人開心的事——剛果民主共和國五十二年來第一次闖進世界盃，全國放假一天慶祝。但開心的事不多，更多的是油價衝擊帶來的痛苦：迦納汽油漲了百分之十五、柴油漲百分之十九，馬拉威更誇張，漲了超過百分之三十。南非是唯一逆勢削減燃油稅的國家。布吉納法索的軍事領導人 Traoré 公開說國家應該「忘記民主」，大選無限期延後。

好，聊完國際局勢，我們來看看科技圈今天有什麼動靜。

先說加密貨幣這邊一個大消息。Charles Schwab——如果你不熟悉的話，這是美國最大的券商之一，管理的客戶資產將近十一點九兆美元——他們正式開放了比特幣和以太坊現貨交易的候補名單。

這個服務叫「Schwab Crypto」，計劃上半年推出，透過旗下的 Charles Schwab Premier Bank 提供。目前除了紐約州和路易斯安那州之外，全美客戶都可以申請候補。

為什麼這很重要？因為這代表加密貨幣的「主流化」又往前走了一大步。之前是 Coinbase 拿到聯邦信託執照，現在是傳統金融巨頭直接開放現貨交易。但也不要太興奮——看空的人會指出，這個服務只限 BTC 和 ETH 兩個幣種，而且是透過銀行子公司而不是一般券商帳戶提供，實際的轉換率可能遠低於市場期待。更何況，現在的加密市場剛經歷了自 2018 年以來最差的一個季度。

不過 IMF 這邊也發了一個值得注意的報告。他們警告代幣化金融在市場壓力期間可能放大系統性風險，呼籲各國以央行錨定資產為基礎來建立監管框架。DeFi 社群當然不同意，他們認為 IMF 的分析太悲觀了，忽略了鏈上透明度帶來的風險可見性優勢。

網路安全方面，這個禮拜的情況蠻嚴重的。

Chrome 今年第四個零日漏洞被發現了——CVE-2026-5281，是 WebGPU 的 Dawn 元件裡面的一個 use-after-free 漏洞，而且已經有人在實際利用了。Google 已經緊急修補。如果你還沒更新 Chrome，趕快去更新。

CISA 也連發了兩道緊急修補命令。第一個是 Cisco Secure Firewall Management Center 的最高嚴重度漏洞，允許未授權的攻擊者直接拿到 root 權限。第二個是 Langflow——一個 AI 工作流平台——的漏洞，可以讓攻擊者在你的 AI 部署環境裡執行任意 Python 程式碼。如果你的團隊有在用 Langflow，趕快檢查版本。

另外還有一件供應鏈攻擊事件。Axios，就是那個很多 JavaScript 專案都在用的 HTTP 函式庫，有兩個惡意版本被發布到 npm 上，是透過入侵維護者帳號做的，疑似北韓國家級駭客所為。這可能已經洩漏了數十萬個 secrets。npm 生態系的帳號安全問題，已經不是第一次了。

太空方面有一個讓人興奮的消息。Artemis II 的四位太空人明天就要繞月了！

他們四月一號發射，四月二號完成了 TLI 點火離開地球軌道，現在正在往月球的方向飛。明天，四月六號，他們會以八字形繞月飛行，最遠距離地球四十萬六千七百多公里。這個距離會打破 1970 年 Apollo 13 創下的最遠人類飛行紀錄。這是人類五十四年來第一次飛到月球附近。如果一切順利，他們幾天後會降落在聖地牙哥外海。

在 LLM 和 AI 這邊，今天沒有什麼重磅的模型發布，但有兩篇很值得讀的文章。

Simon Willison 寫了一篇關於 coding agents 對開發者認知影響的深度分析。他的核心觀點是：當 AI 代理處理越來越多的實作細節，開發者的角色從「撰寫者」變成了「審查者」，但問題是——你怎麼在不完全理解程式碼的情況下判斷它是對是錯？這是一個新的認知負擔，而且很容易滑向對 AI 生成程式碼的盲目信任。

Sebastian Raschka 也發了一篇拆解 coding agent 架構的文章，把 coding agent 拆成四個核心模組：規劃、工具使用、記憶、跟自我修正。他比較了 Cursor、Windsurf、Claude Code 這些工具在這四個維度上的設計差異。比如說，有的工具偏重規劃能力，先把任務拆解再動手寫；有的則偏重自我修正，寫完之後自己跑測試、發現問題再修。如果你是做 AI 工具的人，這篇值得細讀。

說到 AI 工具，有一個小但值得一提的消息。Ollama 發布了 v0.20.2，主要的改變是啟動時預設開啟新對話介面。如果你記得的話，上禮拜 Ollama 才剛發布 v0.20.0 加入了 Gemma 4 支援，現在已經進入穩定性迭代階段了。

Amazon 那邊也有動作。四月四號，Atlas V 火箭成功發射了二十九顆 Kuiper 網路衛星，是這款火箭有史以來最重的載荷。Amazon 正在加速建設自己的衛星網路，跟 SpaceX 的 Starlink 正面競爭。現在的衛星網路賽道已經不只是科技公司的事了——在戰爭時期，衛星通訊的戰略價值變得非常明顯。

生技那邊也有大動作，兩筆大型併購同一週公布。Biogen 花了五十六億美元收購 Apellis Pharmaceuticals，拿到兩個已核准的眼科藥物 Syfovre 跟 Empaveli。Eli Lilly 花了六十三億買下 Centessa Pharmaceuticals，主要是看上他們針對嗜睡症的 OX2R 靶向候選藥物。這兩筆交易在季末最後一天推動生技指數飆漲了百分之七——可以說是把整個季度的低迷一口氣補回來了。不過批評者也指出，這種高額溢價收購長期來看可能壓縮投資報酬率。

最後快速提一個安全方面的事，德國聯邦內政部公布了歐盟 eIDAS 數位身分錢包的實作方案，結果要求使用者必須有 Apple 或 Google 帳號才能啟用。歐洲的隱私倡議者當然炸鍋了——歐洲公民的數位身分基礎設施，居然建立在美國科技巨頭的平台上？這跟歐盟一直強調的數位主權完全矛盾。
