# EP38 — 2026年4月8日

早安，歡迎回到每日世界新聞 Podcast，今天是四月八號，星期三。我是你的主持人。今天要跟你聊的大事非常集中，國際局勢這邊，美伊戰爭進入第三十八天，Trump 丟出了一個讓人倒抽一口氣的最後通牒；烏克蘭情資又爆出俄羅斯跟伊朗合作的新細節，讓整個戰局的版圖重新被改寫；台灣這邊呢，國民黨主席鄭麗文踏上中國土地，是十年來第一次，時機非常敏感。科技圈今天也熱鬧，Google 的 Gemini 3.1 Ultra 正式發布、芝加哥商品交易所 CME 宣布加入兩支加密期貨、以及一個全球資安圈都在討論的 GRU 駭客事件。好，我們一個一個來聊。

那我們先從最緊繃的地方開始，美伊戰爭。

到今天已經打第三十八天了。昨天 Trump 在白宮的記者會上，把伊朗提出的那個所謂十點和平方案，評為「是重要的一步，但還不夠好」。你可能會問，那個方案講什麼？根據半島電視台的整理，伊朗提的是階段性的，四十五天的分階段停火、逐步重新開放荷姆茲海峽、還有雙方都不再打對方的能源設施。聽起來已經是伊朗大幅讓步了，對吧？但 Trump 認為，方案裡面沒有包含對伊朗核計畫的長期監管條款，所以他說不夠。

然後 Trump 做了一件讓整個國際社會都警惕起來的事情——他在自己的社群平台上，給出了一個最後通牒。他說，伊朗必須在美東時間週二晚上八點前，完全重新開放荷姆茲海峽，否則——我原話翻譯給你聽——「整個文明今晚可能就會死去，永遠無法復原。我不想這樣，但很可能會發生。」他還說，「伊朗每一座橋樑將在美東週三午夜前被夷平，每一座電廠都將被摧毀、燒毀、永遠無法使用」。

這是什麼等級的發言？這已經不是傳統的外交施壓語言，而是直接威脅要對一個國家的民用基礎設施進行毀滅性打擊。更讓人不安的是，Trump 在被記者追問戰爭目的時，他說了一句「To the winner belong the spoils，勝者為王」，他說如果由他決定，他會「接收伊朗的石油」，因為「我首先是商人」。

好，這時候你就會看到一個很典型的國家間敘事差異。我們一個一個看。

美國這邊，白宮幕僚把這個最後通牒包裝成「迫使伊朗上談判桌的必要施壓」，強調外交窗口沒有關閉。伊朗方面，其總統 Pezeshkian 打電話給歐盟理事會主席 António Costa，他的原話是「如果獲得戰爭不會重啟的保證，伊朗有必要意願結束戰爭」。你注意到沒有，伊朗這邊在用的是「結束戰爭」這個詞，而不是「投降」。他們把自己塑造成被迫應戰的一方。

歐洲的反應最有意思。西班牙總理 Sánchez 公開把自己定位為 Trump 在歐洲最堅定的反對者，他直接反對任何「以武力奪取他國石油」的說法。這在歐盟內部其實是少見的，因為德法通常會比較謹慎。

而中國呢？外交部發言人毛寧在四月七號的例行記者會上，對美方最後通牒表達「嚴重關切」，她呼籲各方克制，但特別強調了中方對荷姆茲海峽航運安全的「正當商業利益」。這句話很關鍵，等於是在告訴華盛頓：你動荷姆茲，等於動中國的能源命脈。

那市場怎麼看這件事呢？其實市場比媒體冷靜很多。Polymarket 上面的四十五天停火合約，YES 的機率只有百分之一到四。換句話說，絕大多數的交易員都認為這個最後通牒只是個談判姿態，不是真正要開打的序曲。原油市場也是一樣，WTI 從戰爭高點的 112 美元，一路跌到 95.44，單日跌超過十五個百分點；Brent 更誇張，跌了十六個百分點。這其實很清楚告訴你，交易員不相信 Trump 真的會炸電廠。但反過來講，如果市場猜錯呢？退役將領警告，對伊朗電廠的大規模打擊會造成嚴重的平民人道災難。所以我們繼續觀察。

好，聊完伊朗，我們把鏡頭拉到另一個被這場戰爭擾動的戰場——烏克蘭。

你可能以為美伊戰爭跟烏克蘭沒什麼關係，但昨天烏克蘭國防情報總局 GUR 發布了一份評估報告，把兩件事直接串起來。報告說，俄羅斯的軍事衛星在三月二十一號到三十一號這十天內，對中東十一個國家進行了至少二十四次詳細的影像測繪，涵蓋四十六個「目標物」，包括美軍基地、其他外國軍事基地、機場、還有油田。更勁爆的是，烏克蘭情資指出，俄羅斯軍情局 GRU 向伊朗交出了一份清單，上面列著以色列五十五處關鍵能源基礎設施的精確座標——伊朗最近對以色列能源網的精準飛彈打擊，就是靠這份清單執行的。

這個情資一出來，以色列 Mossad 的負責人直接公開表態，說要「把俄羅斯視為伊朗戰爭的共同方」。這是什麼意思？這等於是在說，以色列可能不再把俄羅斯當成中立國，未來以色列對敘利亞俄軍基地的態度就會變化。

那這對烏克蘭自己呢？PBS 跟 Bloomberg 都報導，俄軍已經開始對烏克蘭東部所謂「要塞帶」城市發動春季攻勢的早期階段。克里姆林宮否認是利用中東戰爭擴大攻勢，說這是「既定計畫」，但時機點就是這麼湊巧。更糟的是，基輔警告他們的 Patriot 攔截彈因為要支援中東，已經出現短缺。Zelenskyy 上週末跑了一趟中東，跟沙烏地阿拉伯和卡達簽了「價值數十億美元」的十年防務協議，用烏克蘭累積了三年多的反無人機技術，去換波灣國家的援助跟對俄施壓。這個叫做「技術換安全」的外交算盤打得很精，但能換多少，就看美伊戰爭會拖多久。

Foreign Policy Research Institute 的分析就很直接，他們說如果中東戰爭超過九十天，美方 Patriot 產能就吃不消了，到時候華府很可能會被迫要烏克蘭讓渡東部的一些城市，來保全基輔的空防。這是一個很殘酷的推演，但它反映了一個現實——全球彈藥產能是有極限的，兩條戰線同時燒，就會有人要縮。

再來一個跟伊朗戰爭無關、但今天一定要講的事情，就是台灣。

昨天四月七號，國民黨主席鄭麗文抵達中國，她是十年來第一位以現任黨主席身分訪問大陸的國民黨領袖。她的行程被外界視為有機會會晤習近平，這就是一場重量級的政治訪問。時機怎麼會挑得這麼敏感？你要知道，台灣立法院現在正擱置政府提出的四百億美元額外國防預算案，美方對台軍售因為國會卡關也一併受影響。習近平會在這個時候接見她，意圖其實很明顯——鞏固鄭在黨內的地位、順便干擾華府對台軍售的節奏。

賴清德這邊的回應很克制，他再次重申「願意對話，但台灣有權決定自己的路」。你注意他用的詞，他沒有反對鄭麗文訪陸，但他強調的是「自主權」。這三個字是台灣對中對美外交的最大公約數。

另外還有一件事讓台灣很緊張——WSJ 跟 Taipei Times 追蹤到，中國罕見地在台灣以北，從黃海一路延伸到日本西側東海海域，預留了一塊大面積空域，**而且是四十天**。重點是，北京沒有宣告具體演習。過去解讀中國軍事動態，大家習慣看「演習開始、演習結束」這個框架，但這次是什麼演習都沒宣告，就先把空域圈起來四十天。部分台灣軍事評論者指出，這可能是用來掩護實際兵力推進，或者是新裝備測試的掩體。傳統的判讀方式已經不管用了，這是一個新的「持續性戰備姿態」。日本自衛隊已經把西南諸島的戰備級別調高。

好，聊完國際局勢這幾場硬仗，我們來看看科技圈今天有什麼動靜。

今天科技圈最大的新聞，是 Google DeepMind 終於把年度最重量級的模型 Gemini 3.1 Ultra 正式發布了。這是什麼等級的模型？兩百萬 token 的上下文窗口——聽起來 GPT-5.4 跟 Claude Mythos 5 也差不多對吧？但 Gemini 3.1 Ultra 的賣點不在窗口大小，而在於**它是從訓練階段就針對文字、圖像、音訊、影片四種模態同時推理去設計的**。過去你用 GPT-4o 或者 Gemini 1.5，多模態其實是把不同的編碼器拼在一起，讓模型去「翻譯」不同模態的資訊。但 Gemini 3.1 Ultra 是一開始就在四種資料上聯合訓練，所以它在跨模態的基準像 MMMU 跟 Video-MME 上刷新了紀錄。

那這個東西有多強？講一個實際的例子，用戶可以丟給它一段十分鐘的影片，同時加上一份 PDF 論文跟一段音訊訪談，問它「這三個資料裡哪個人講的話跟論文的結論矛盾」，Gemini 3.1 Ultra 可以直接指出來。這在過去是要三個步驟處理的事情。

當然也有批評的聲音。研究者指出，兩百萬 token 的實際可用範圍其實大約在一百萬以內，中段資訊的提取還是會有那個經典的「needle in haystack」衰減問題。另外 Ultra 版的 API 定價到現在都還沒公布，中小開發者短期內是看得到吃不到。

跟它同期競爭的還有 OpenAI 的 GPT-5.4 Thinking。OpenAI 今天也揭露了一個重要數字——GPT-5.4 Thinking 在桌面電腦操作基準 OSWorld-Verified 上拿下 75.0 分，**首次超越人類平均水準**。OSWorld 是什麼？它是測試 AI 能不能像人一樣操作電腦、開軟體、填表單、處理 email 的基準。這個分數的意義是，未來幾個月你會看到更多企業把「自動化員工」這個概念變成真正的產品。但同樣地，批評聲音也很即時——OSWorld 的任務是預先錄製的封閉環境，真實企業場景的長尾錯誤率遠高於 benchmark，OpenAI 沒有揭露失敗模式。

Anthropic 那邊呢，延續昨天 Claude Mythos 5 資訊外洩的新聞，今天進一步確認 Mythos 5 是首個被廣泛承認的十兆參數前沿模型，主打高風險環境下的網路安全分析跟複雜程式碼重構，並同步推出了一個輕量版叫 Capabara。更值得注意的是，Anthropic 的 MCP 協定——就是 Model Context Protocol，你可以理解成 AI 版的 USB-C——三月單月累計安裝量突破九千七百萬次。OpenAI、Google、Mistral、Meta 全都已經推出 MCP 相容的 SDK 或 server。這代表什麼？代表 Agent 工具調用的事實標準，已經由 Anthropic 拿下。

但這裡也有分歧。社群討論認為 MCP 的協定設計偏向 Anthropic 自家 Claude Desktop，很多人呼籲要更開放的傳輸層治理。這其實是每一個產業標準誕生時都會遇到的權力爭奪。

加密貨幣這邊，有兩件大事。第一件，芝加哥商品交易所 CME 宣布加入 AVAX 跟 SUI 期貨，五月四號上線，等 CFTC 審核完。但更重大的是，CME 宣布從五月二十九號開始，它家全系列的加密期貨產品——Bitcoin、Ether、Solana、XRP、AVAX、SUI——**全部轉為二十四小時不間斷交易**。這意味著什麼？意味著傳統金融市場跟加密現貨市場之間「週末時差套利」的空間會被壓縮。機構入場變得更方便，但同時批評者警告，如果 24/7 沒有對應的流動性，週末的波動會被放大，受傷的還是散戶。

第二件是 Polygon 今天下午 UTC 兩點、區塊高度 85,268,500 要執行 Giugliano 硬分叉。這是 PoS 驗證人集合輪換跟 gas 計價的優化。但要提醒的是，Polygon 過去兩次硬分叉都出現四到六小時的非計畫性區塊停滯，市場對這次的執行穩定性還有疑慮。

最後一個科技新聞，我覺得你必須要知道——Krebs on Security 披露俄羅斯 GRU 的駭客組織 APT28，從 2025 年底一路到 2026 年三月，長期針對家用跟小型辦公室的路由器植入持久性的 implant。他們攻的是 TP-Link、Asus、MikroTik 這幾家常見型號。目的是什麼？把被攻陷的路由器當跳板，去偷企業用戶的 Microsoft Office 365 refresh token，然後長期存取你的 email 跟 SharePoint。CISA 跟 FBI 已經聯合發布 TA26-097A 公告。這件事最諷刺的地方是，大量被攻陷的路由器都是已經停止韌體更新的 EOL 型號，硬體廠商的 end-of-life 政策，最後是由終端使用者買單。

好，接下來是今天的市場快報。

美股方面，昨天四月七號的收盤，S&P 500 在六千六百一十六點，微漲了零點零八個百分點；Nasdaq 上漲零點一；道瓊則是微跌零點一八。三大指數在 Trump 最後通牒前夕其實是分化的，市場沒有全面緊張。

亞洲股市今天大部分還受到清明跟地區假期影響，上證、恆生、台灣加權這幾個市場最新數據都是四月二號到三號的，所以今天沒有新的盤面可以講，我就不重複昨天已經講過的數字了。日經 225 最新是四月六號的五萬三千四百點，小漲零點五五。

好，今天就聊到這裡。一句話總結——世界今天的主旋律，就是在等一個八點鐘的最後通牒會不會真的兌現。如果兌現，你明天會聽到一整套完全不同的故事；如果只是姿態，那麼烏克蘭的 Patriot 彈藥、台灣的國防預算、跟 Gemini 3.1 Ultra 的 API 定價，這些事情就會重新站回新聞頭條的中央。

在結束前，還有幾件今天雖然不是頭條，但值得你知道的事情，我快速跟你交代一下。

第一件是歐洲。德國、義大利、西班牙、葡萄牙、奧地利這五個歐盟成員國的能源事務主管，聯名寫信給歐盟，要求對能源業徵收超額利潤稅。為什麼？因為中東戰爭把電價推得很高，一般家庭的荷包在燒，各國財政也吃不消。這五國希望用能源業的超額利潤，去補貼消費者跟公共預算。同一天，歐洲議會也批准了跟美方的新貿易協議，接受美方對歐洲商品課徵百分之十五的關稅。這等於是歐盟在貿易上對美方退了一步，同時把希望寄託在加速推進 Mercosur、還有更多非美市場的協議上。你可以看到，歐盟的邏輯是：對美國我忍、對俄羅斯我防、對中國我防——三條戰線都在繃。

第二件，德國還發生了一件讓人心碎的事。在 Schleswig-Holstein 邦一個叫 Mittelangeln 的小鎮，週一辦了一場社區復活節尋蛋活動，結果強風導致一棵大樹倒下，砸中了人群，兩名兒童跟一名成年人當場死亡，還有一人重傷。德國氣象局已經把近日強風列為紅色警報。我講這個不是要煽情，而是要提醒你，在所有大國博弈、AI 模型發布這些宏大敘事底下，真實的生活還在，真實的悲劇還在。

第三件跟你我荷包有關——台灣交通部民航局昨天核准了長程航線燃油附加費大幅調漲百分之一百五十七，單程最高會到新台幣三千七百三十九塊。這是疫情以來最大的調升幅度，暑假要飛歐美的朋友，機票預算要重新算一次。這個漲幅的直接推手，就是中東戰爭引發的油價波動——雖然原油昨天大跌，但航空公司的燃油對沖合約通常是三到六個月前鎖定的，所以漲價的時間差會比現貨來得慢。
