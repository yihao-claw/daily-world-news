# 每日世界新聞 Podcast — EP46

嘿，你好，歡迎回到每日世界新聞，我是你的主持人。

今天禮拜天，我們把時間花在兩條最值得你追的主線：一條是美伊停火剩下兩天，伊朗又把荷莫茲海峽關上了，全球海運保險費、能源價格、匯率走向全部綁在這個節奏上；另一條是印度國會在週五上演了 Modi 執政十二年來第一次憲改敗選，這不只是一場輸贏，是南北印政治版圖重組的起點。科技那邊，我們會聊 Ollama 推出一個會自己學你工作流的 Hermes Agent、Kelp DAO 跨鏈橋被駭走將近三億美元、還有 PyTorch 連三個核心提交都標著「Authored with Claude」，Meta 等於把 Anthropic 的模型納入正式工程鍊。好，我們開始。

我們先從中東開始講。你昨天聽我說荷莫茲海峽「完全開放」，川普還公開感謝伊朗——這個狀況連三十六小時都沒撐過去。伊朗軍方週六發了一份新的聲明，說荷莫茲的控制權「回到先前狀態」，而且對試圖通行的船隻開火示警。理由是什麼？德黑蘭說，美國根本沒有停下來，對伊朗港口的封鎖還在繼續，「你沒放手我就不放手」。

這件事你要記住兩個關鍵日子：一個是週三，也就是四月十五，那一天美伊在伊斯蘭馬巴德的第二輪談判，總共談了二十一個小時，一個小時都沒睡，最後還是沒有共識。美方要求伊朗二十年內不得鈾濃縮，伊朗說最多五年。另一個日子是週二，也就是四月二十一，兩週的停火就在那一天到期。如果這兩天之內沒有新的協議、或者沒有至少延長停火框架，中東再起戰火的機率就會陡增。

同一時間，以色列跟黎巴嫩那邊，所謂的十日停火在週五東岸時間下午五點生效，結果黎巴嫩軍方禮拜六清晨就已經指控以色列多次砲擊南部邊境村莊。真主黨雖然沒有正式參與協議，但高層放話「手指在扳機上」。你感覺到這種氛圍了嗎？一份紙上的停火，跟地面上的停火，是完全兩回事。聯合國自己的統計是，從上一次黎巴嫩停火生效以來，已經累積超過一萬次違規、好幾百名黎巴嫩人死亡。

加薩那邊我也不想放過。你可能會覺得「加薩的停火不是早就開始了？」是，表面上是，但從四月中到現在的這幾天，每一天都還在死人。週四，北加薩的 Beit Lahiya 被以色列無人機空襲，有兩兄弟當場身亡。同一天，加薩城 Zeitoun 區一個九歲的男孩 Saleh Badawi 被以軍擊斃。Khan Younis 南邊也有一名三十八歲的男子喪命。一個數字你聽起來會很沉：自從 2023 年十月以來，加薩死亡人數已經超過七萬兩千三百人；光是停火之後就超過七百六十五人。人權辦公室也記錄到，三月是約旦河西岸有紀錄以來，定居者暴力最致命的一個月。

好，我們再往東走。印度這邊週五發生一件很罕見的事——你記得我昨天提到印度下議院會表決憲法修正案嗎？結果出來了：Modi 政府推的《第 131 條憲法修正案》被否決了。要修憲需要三分之二多數，也就是三百五十二票，最後只拿到二百九十八票，反對二百三十票，差了五十四票。這是 Modi 執政十二年來第一次憲改敗選。

這個法案表面上是要把下議院席次從五百四十三席擴增到八百五十席，同時落實女性保留席次，預計 2029 年上路。聽起來很進步，對不對？但反對黨——Congress 的 Priyanka Gandhi、南印 DMK 的領導層——全都看穿了，這個 850 席的版本是「北方邦、北方印度搶奪話語權」的工具。你要知道，印度南北印經濟、人口結構差很多，北印生育率高，如果照人口比例重劃，南印州在國會的相對份量會被稀釋。所以南印幾乎是全線反對，加上 Congress 系統整合，最後讓 Modi 吃下他掌權以來最重的一次立法挫敗。政府事後也只好把《劃區法案》撤回。

這件事不是單一法案的勝負，它預告的是：印度政治從 2026 年下半年開始，將會進入南北印正式的拉鋸期。Modi 想靠「人口紅利 + 劃區」鎖住下一個十年的政權——這個如意算盤現在被國會正式打掉了。

換個方向，你如果對歐洲有興趣。匈牙利這邊你大概記得，四月十二 Magyar 的 Tisza 黨把 Orbán 拉下台，這週六新的計票出來，Tisza 從原本估的一百三十八席再上修到一百四十一席，在一百九十九席的國會裡，這是匈牙利現行選制史上最大的多數。Magyar 本人禮拜三已經和匈牙利的國家元首會晤，政權移交敲定在五月第一週——Orbán 十六年的執政就此收官。

布魯塞爾現在最關心的，是新政府上任後能不能幫歐盟解開兩件事：一個是九百億歐元的對烏貸款，一個是對俄第二十輪制裁案。但你要知道，擋第二十輪制裁的國家其實不是匈牙利而是斯洛伐克，原因是 Druzhba 原油管線被俄方打壞之後，斯洛伐克要求烏克蘭書面保證「清晰、透明、可驗證」的恢復供油時程，否則不讓制裁過關。澤倫斯基預期 Druzhba 可以在四月底修好——如果順利，歐盟在四月的最後一週就有可能雙利多：對俄加碼制裁，同時對烏放行貸款。你可以把這個當成一個錨點來盯。

烏克蘭那邊週四凌晨吃了一次年度最大規模的夜襲。俄軍出了將近七百架無人機、十九枚彈道飛彈，從基輔一路打到奧德薩。至少十八人罹難、一百一十八人傷，其中包括一個十二歲的男孩。俄國防部的說法是「對烏克蘭深入打擊我方煉油廠跟軍工設施的報復」。這個攻擊規模你要放在哪個脈絡看？烏克蘭空軍週五就已經開始喊防空彈藥吃緊，部分原因就是以伊戰爭期間，美方把防空庫存抽到中東去了。

亞洲再給你兩條重點。台灣這邊，內政部主導的「首次反能源封鎖演習」已經進到第八天，從四月十二跑到四月二十一。這次跟以往最大的不同是，它不是只有軍方演，而是內政部、海巡署、海軍、經濟部一起聯合執行。四月十八，美國在台協會的副代表親自到現場觀摩民防動員環節，公開講這是「台灣全社會韌性工程的重大一步」。國安會總協調官顧立雄也強調，反封鎖演習要常態化。這個訊號非常清楚：對華府跟台北來說，中國對台封鎖已經不是假設，而是必須演練到肌肉記憶的劇本。

同一個週日，賴清德下禮拜三也就是四月二十二要出訪史瓦帝尼，是他就任以來第一次外訪。這是一趟直飛、不過境任何第三國的行程；代表團裡林佳龍和潘孟安都會同行。史瓦帝尼是台灣在非洲唯一的邦交國，這次一方面是鞏固邦誼、慶祝史瓦帝尼國王登基四十週年，一方面也是向非洲其他有搖擺可能的國家發一張「台灣仍在場」的宣告。北京肯定會回應，但關鍵是這趟訪問結束後，短期之內會不會有第三國像過去那樣選擇切換承認。

菲律賓這邊，下禮拜三還有另一件大事：海牙國際刑事法院會針對 Duterte 管轄權上訴做最終裁決，海牙時間上午十一點、馬尼拉時間下午五點宣判。去年十月，ICC 的預審分庭已經裁定「菲律賓退出 ICC 前的案件可以繼續追究」。Duterte 律師再上訴到上訴庭，就是這禮拜三要出結果。八十一歲的 Duterte 已經三度請求不出庭，理由是「體重驟降跟認知障礙」。如果上訴庭維持原判，他就正式進入 ICC 的反人類罪審判階段，這對整個東南亞未來各國執政者面對的人權追訴案來說，是一個會被反覆引用的標竿判決。

接下來我們進到科技。今天的科技也很有意思，因為你會發現「AI 工具跟 AI 使用者」之間的界線在快速模糊。

---

第一個要跟你聊的是 Ollama 發了 v0.21.0 正式版，裡面有一個新功能叫 Hermes Agent。這個 Agent 最特別的地方不是它有多厲害，而是它會從你每次跟它互動的過程中，自動擷取新的技能、累積到你的個人化 skill library。你可以想像成是「你用得越多，你這台機器上的模型就越懂你」。同一個版本還把 Gemma 4 帶上 Apple Silicon 的 MLX runtime——如果你是用 Mac 在跑本地模型，速度跟能耗都會明顯改善。

這件事對本地 AI 的意義你要聽清楚：過去本地模型的優勢一直是「隱私 + 成本」，但跟雲端比功能永遠輸。Hermes 這個設計把戰場換了——本地模型現在拿來跟雲端比的不是 raw 能力，而是「個人化深度」，因為雲端模型沒有辦法長期留存你所有的 prompt 跟 code。但你也要有風險意識：Hermes 把你的 prompt、你的 code、你的 commits 全都變成「自訂 skill」的訓練素材，對於有機密資料的工程環境，這幾乎等於給模型一份永久記憶。企業用戶要上這套，得先補 redaction 跟離線 audit 的配套才能放心。

---

第二條，你要留意的是一個加密資安的大案件：Kelp DAO 的 rsETH 跨鏈橋，在 UTC 四月十八深夜被利用 LayerZero 跨鏈訊息層的漏洞給打穿了，總共被抽走將近三億美元——準確數字是兩億九千兩百萬。Kelp 的緊急 pauser 多簽名機制在攻擊成功之後四十六分鐘內凍結了核心合約，阻止了兩次後續追擊。

這件事你要注意的不只是金額，而是 LayerZero 這個名字——它是目前最大的通用跨鏈訊息層，非常多個 LST 協議都接在它上面。如果 LayerZero 的核心真的被攻破，那同構的 Renzo、EtherFi 這些協議都有可能踩到同一個坑。加密圈這幾天的情緒有點像 2022 年的 Cross-chain Bridge 黑色系列再現——當時 Ronin、Wormhole、Nomad 連續出事。現在這波很可能是新一輪橋的安全審視循環。我會把眼睛盯在接下來七十二小時，看看其他接 LayerZero 的協議有沒有自主停機檢查。

---

第三條，你要聽我說一下 PyTorch 跟 Claude 的關係變化。PyTorch 核心在四月十八連發三個 viable/strict 的提交。一個是 aarch64 上跳過 fbgemm 才能跑的量化 benchmark——這個 bug 讓 m8g ARM runner 整整壞了一個多月。第二個是利用 PR 171443 新加的 torch.Stream.is_capturing 把 optimizer 的 graph capture 邏輯重構。第三個是修 XPU 上 hard-code CUDA 的測試失敗。這三個 commit 都有一個共同點——它們的 commit message 裡面都標著「Authored with Claude」。

你懂我意思嗎？這不是 Meta 內部員工「偶爾用 Claude 幫忙」，這是 Meta 在核心工程鏈上，把 Claude 當成正式的共同作者在 check-in。如果你還在觀望 Claude Code 能不能進生產工程，這就是最硬的背書之一。

同一時間，LangChain 發了 langchain-core 1.3.0 正式版、langchain-anthropic 1.4.1 補完 Opus 4.7 所有新 feature 的對接——也修掉了 compaction block 裡空的 encrypted_content 會讓整支 API 炸掉的 bug。LangChain 這次還有另一件值得注意的事：text-splitters 1.1.2 跟 huggingface 1.2.2 都加了 SSRF 硬化。這是做 agent 的人現在的日常——你放一個 agent 出去抓網頁，如果沒處理 SSRF，就等於給了攻擊者一張進你內網的門票。LangChain 這一輪等於系統性在補這塊。

---

第四條，我想帶你看一個「想法對撞」的事情。以太坊共同創辦人 Joseph Lubin 公開批評 OpenAI、Anthropic、Google 這三家把前沿 AI 的算力壟斷——他呼籲用區塊鏈基礎的「去中心化推論」來對抗。乍聽你會覺得「嗯，有點道理」，但我必須給你反面的角度：Lubin 自己是多個 web3-AI 相關基金的 LP，這番話有非常明顯的利益衝突。而且去中心化推論過去兩年已經有 Bittensor、Akash 這類專案在做，都沒有真的撼動 GPU 集中化的結構。你要看到這一點：去中心化推論在技術上是可行的，但跟頂級閉源模型的能力差距還是數量級的。把它當成「替代品」是太樂觀；把它當成「某些場景的補位」才比較實際。

前沿科技這邊還有一個你應該留一眼的訊號：中國的智元機器人（AgiBot）在一個月內量產突破一萬台，四月十七發表了六個自研 AI 模型 + 七個垂直生產力方案，正式把自己從「硬體公司」轉成「具身智能平台公司」。今天禮拜天，北京亦莊還有一場機器人馬拉松，高德的第一款四足機器人「途途」首次亮相。這是中國具身智能賽道的集中一役。同一天，海康威視公佈 2025 年報：營收九百二十五億人民幣、淨利增長 18.5%、經營現金流暴增 91%，海外業務即便在制裁壓力下仍然成長 7.5%。這些都是在告訴你，中國的「硬體 + AI」賽道不是在退，反而在這種制裁跟脫鉤的壓力下長出新的韌性。

---

最後給你一個比較冷門但影響深遠的點。NASA 在四月十六敲定用 SpaceX 的獵鷹重型火箭，在 2028 年幫歐洲航天局發射「羅莎琳德·富蘭克林」火星車。這件事的背景是 2022 年俄羅斯退出 ExoMars 計畫之後，這台火星車一直沒有發射載體——現在正式由美方的 Falcon Heavy 接手。這代表的不只是太空科技，它是「俄烏戰爭之後美歐太空合作重組」的一個具體落地案例。你要把這件事跟前面 Hormuz、跟歐洲制裁俄國綁在一起看——今天全球的供應鏈不是在「去風險」，而是在「重新選邊站」，太空、能源、AI 算力、跨鏈橋，每一層都在重新畫地圖。

好，今天的節目到這裡差不多了。如果你只能帶走三件事，我會希望你記得這三個：第一，四月二十一美伊停火到期是這週最重要的倒數；第二，印度國會週五的挫敗不只是 Modi 一個人的輸贏，是南北印政治版圖重整的開端；第三，Kelp DAO 被攻擊的三億美元背後，是跨鏈橋安全的下一波系統性風險，如果你手上有 LST、有錢放在 LayerZero 上面接的任何協議，這週請多看幾眼。
