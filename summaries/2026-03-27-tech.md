## 🧠 LLM / 大型語言模型

• 🔥13 **Gartner 預測：LLM 推論成本 2030 年下降 90%** — Gartner 於 3 月 25 日發布報告，預測到 2030 年，擁有一兆參數的 LLM 每進行 1,000 億 token 推論的成本將低於 1 美元，較當前水準降低 90%（效率提升最高達 100 倍）。主要驅動因素包括：客製化晶片普及、模型壓縮與量化技術成熟、競爭加劇壓低邊際成本。
⚠️ 批評觀點：成本下降不等於民主化——頭部廠商壟斷算力、資料和人才的結構性優勢將持續存在，成本優勢可能進一步集中於大公司而非扶持小型開發者。
📎 https://www.hpcwire.com/aiwire/2026/03/25/gartner-forecasts-90-drop-in-llm-inference-costs-by-2030/
📎 https://www.sdxcentral.com/news/gartner-llms-to-be-up-to-100x-more-cost-efficient-by-2030/

• 🔥13 **美國 CDAO 向 Anthropic、Google 等提撥 6 億美元 AI 合約** — 美國國防部首席數位暨 AI 辦公室（CDAO）於 3 月 26 日宣布向 Anthropic、Google 等多家 AI 公司提撥共 6 億美元合約，用於軍事 AI 工具部署。同日，Axios 報導 Anthropic 此前因安全疑慮被凍結的五角大廈合約「可能重啟」。參議員 Mark Warner 等人聯名致函，要求 AI 公司說明其在 DoD 系統中的安全防護機制及模型部署細節。
⚠️ 批評觀點：DefenseOne 研究指出，LLM 工具可能損害軍事人員的獨立判斷力，「不是killer robots 問題，而是讓人類決策者更依賴 AI 建議」；國會要求透明度，但軍方以保密為由拒絕完全公開。
📎 https://meta-defense.fr/en/2026/03/26/anthropic-dod-llm-dangers-800-million/
📎 https://www.axios.com/2026/03/26/anthropic-pentagon-ai-deal
📎 https://www.defenseone.com/technology/2026/03/military-ai-troops-judgement/412390/

• 🔥13 **Ollama v0.18.3 正式發布：VS Code 深度整合** — Ollama 發布 v0.18.3，核心更新是與 GitHub Copilot Chat / VS Code 的原生整合，開發者可在 IDE 內直接使用本地 LLM，無需切換工具。Copilot Chat 現在優先讀取 Ollama 模型的 `general.basename` 參數實現無縫對接。
⚠️ 批評觀點：本地 LLM 整合雖提升隱私，但大多數開源模型在程式碼補全品質上仍遜於 GPT-4o / Claude 等商業模型，對生產力的實際提升有限。
📎 https://github.com/ollama/ollama/releases/tag/v0.18.3

## 🛡️ 資安

• 🔥13 **LiteLLM PyPI 套件遭 TeamPCP 植入後門——AI 供應鏈首次大規模攻擊** — 黑客組織 TeamPCP 在 LiteLLM 1.82.7 和 1.82.8 版本中植入惡意程式碼，目的是竊取 API 憑證（OpenAI、Anthropic 等密鑰）。LiteLLM 是廣泛使用的多模型 API 統一介面，全球影響數萬 AI 開發者和企業。ITPro 報告稱這是針對 AI 基礎設施的供應鏈攻擊升級，攻擊者「甚至不需要 hack AI 本身，只需攻擊它依賴的套件」。
⚠️ 批評：AI 開發社群過度依賴 PyPI 單一生態，安全審計機制嚴重不足；事件揭露 AI 工具鏈存在比傳統軟體更嚴重的信任假設問題。
📎 https://www.itpro.com/security/litellm-pypi-compromise-everything-we-know-so-far
📎 https://futuresearch.ai/blog/litellm-attack-transcript/

• 🔥5 **Mandiant M-Trends 2026：攻擊者存取移交僅需 22 秒** — Mandiant 年度報告揭露，2025 年攻擊者在入侵系統後平均 22 秒即可完成存取憑證移交給下一道攻擊鏈。勒索軟體也出現「recovery-denial」新型態——不再加密，而是破壞備份使受害者無法自行恢復。
📎 https://complexdiscovery.com/twenty-two-seconds-to-hand-off-inside-mandiants-m-trends-2026-findings/
