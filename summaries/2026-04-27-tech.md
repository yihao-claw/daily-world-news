• 🔥18 **Anthropic Claude Mythos「太危險不發布」的模型遭外洩，第三方承包商員工用 Mercor 洩漏憑證猜出 API URL**
  Anthropic 把 Mythos Preview 限定給 52 家受審企業（Apple、AWS、Cisco、JPMorgan、NSA），明說模型有自主挖零日的能力所以不公開。Bloomberg 與 Tom's Hardware 4月22日起連續報導：一名 Anthropic 第三方承包商員工，把先前 Mercor 資料外洩取得的內部命名規律拿來猜 Mythos 的 API 端點，結果從 4月7日 持續存取至今。Anthropic 已啟動調查，但這次外洩不是「被駭」，是 SaaS 配置失誤——攻擊面從模型權重外擴到供應鏈中介人。
  🔻 Zscaler：「這不是 hack，是 SaaS misconfiguration——命名規律可預測 + 缺零信任邊界，等於把鑰匙留在門口。」
