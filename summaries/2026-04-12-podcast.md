# EP41 每日新聞 Podcast — 2026年4月12日

嘿，週日早安，歡迎回來，這裡是每日新聞 Podcast 第四十一集。今天是復活節週日，但全世界可沒有放假的意思。匈牙利正在投票——奧班執政十六年可能走到終點，然後美國跟伊朗的談判在伊斯蘭堡進入了技術細節階段，荷姆茲海峽這個全球石油咽喉到底開不開，現在還沒有答案。好，我們直接開始。

先從歐洲說起。匈牙利今天投票。你如果有在追歐洲政治的話，應該知道奧班這個人——2010 年他上任成為總理，到現在十六年了，中間跟歐盟吵架、跟普丁走得很近、把國內司法體制改得面目全非。但這次，他碰上了真正的對手。反對派 Tisza 黨的 Péter Magyar，一個前體制內的人，去年跳出來組黨，現在民調領先幅度從八個百分點到二十三個百分點不等。比較可靠的 Medián 民調給的數字是 Tisza 五成八、Fidesz 三成三。但問題是匈牙利的選制非常不公平——單一選區的設計對執政黨有利，就算反對派拿到比較多的票，最後議席分配可能還是倒過來。所以今晚開票結果到底怎樣，真的很難說。

不過你想想看，如果 Magyar 真的贏了，影響不只是匈牙利自己。歐盟凍結了好幾十億歐元的撥款，因為匈牙利在法治上面一直不達標。新政府上來的話，這些錢有可能鬆綁。更重要的是匈牙利對俄烏戰爭的立場可能會大轉彎——從「不要制裁俄羅斯」變成「站在歐盟這邊」。這對整個歐洲的地緣政治格局都會有影響。

說到俄烏，今天有一個復活節停火。普丁宣布從莫斯科時間下午四點開始，為期三十二小時。澤連斯基其實幾個禮拜前就在推這個提議了，透過美方轉達給俄羅斯。但你也知道，這種停火的紀錄不太好。果然，停火才剛生效，俄方就說烏克蘭砲擊了赫爾松地區的 Nova Kakhovka。所以到底能不能撐過這三十二小時都是問題，更別說延長成更廣泛的和平談判了。不過至少象徵意義上，雙方還願意做出這個姿態，這一點還是值得注意的。

好，接下來是今天最大的國際焦點——美伊談判。

這個事情的背景是這樣的：四月八號美伊達成兩週停火，巴基斯坦居中調停。但停火宣布的同一天，以色列就對黎巴嫩發動了他們自己稱為「永恆黑暗行動」的大規模空襲，五十架戰機在十分鐘內投了一百六十枚炸彈，三百零三人死亡。黎巴嫩說這是屠殺，伊朗因此威脅如果以色列不停手就要再打回去，而且把荷姆茲海峽關了。

荷姆茲海峽，你可能知道，每天全球大概兩成一的石油從這裡過。關了的話，油價會暴漲，全球供應鏈都會出問題。所以美國急了。副總統 Vance、特使 Witkoff、還有 Kushner，三個人飛到伊斯蘭堡，跟伊朗國會議長 Ghalibaf 和外長 Araghchi 面對面談。美方說所有軍事目標都已經達成了，現在要求伊朗開海峽。伊朗那邊帶了一個十點方案過去，說荷姆茲的主權問題不能談，開不開要跟全面停火一起處理。

到了週六——也就是昨天——美軍做了一個很有意思的動作：兩艘驅逐艦強行通過了荷姆茲海峽，這是開戰以來第一次。而且 CENTCOM 確認已經開始掃雷作業了。所以你看，一邊在談，一邊在展示軍事實力。這個談判到底能不能談出東西來，接下來幾天非常關鍵。

轉到亞洲。台灣這邊有一個很大的事情——國民黨主席鄭麗文在四月十號跟習近平見了面。地點是北京人民大會堂。這是將近十年來中共跟國民黨最高層級的正式會晤。習近平講的那套你大概可以想像：「同根同源」、「一家人」。鄭麗文說她這趟是「和平使命」，說兩岸制度不同但應該相互尊重、超越政治對抗。

島內的反應當然是兩極的。賴清德政府直接說了：「歷史告訴我們，向威權強權妥協只會犧牲主權跟民主。」呼籲國民黨支持國防預算。批評者說這是投降外交，支持者說這是維持對話管道的務實做法。你如果從更大的框架來看這件事，中國最近四十天持續進行空域預留——這是一種軍事施壓的手段，同時又願意接見台灣在野黨的黨主席。這種「一手硬一手軟」的策略其實是北京的慣用手法。而鄭麗文這次訪問的時機也很微妙，正好在 Trump 跟習近平預計五月會面之前。所以兩岸關係的走向，接下來幾個月會非常關鍵。

再來看一下菲律賓。馬可仕總統的支持度在第一季跌到了百分之十九，創下新低。主要原因有幾個：水災治理出了問題、杜特蒂前總統被移送海牙引發爭議、油價上漲。杜特蒂的女兒、Sara Duterte 副總統 的支持度也在下滑，從百分之三十四降到了百分之二十八。ICC 將在四月二十二號決定是否正式審判老杜特蒂的反人類罪指控——他的「反毒戰爭」據估計造成了超過六千人死亡。

另外在非洲這邊，幾個值得注意的事。吉布提總統 Guelleh 以百分之九十七點八的得票率連任第六個任期，去年才修憲取消年齡限制讓七十八歲的他可以再選。唯一的對手拿了百分之二點一九。這種選舉結果你也不用太認真看，但吉布提的地緣位置——紅海入口、各國軍事基地林立——讓它在國際政治中的分量遠超過這個小國的體量。

教宗 Leo 十四世明天開始他就任以來第一次非洲之行，四個國家、十一個城市、超過一萬一千英里。阿爾及利亞、喀麥隆、安哥拉、赤道幾內亞。特別值得注意的是他會在阿爾及爾訪問全球最大的清真寺之一，然後在喀麥隆英語區的 Bamenda 舉辦和平會議——那裡從 2017 年以來就有分離主義武裝衝突，死了幾千人。梵蒂岡的說法是「教宗在就任初期就去非洲，就是要讓世界看到非洲的重要性。」

蘇丹的難民危機持續惡化。查德境內現在有一百三十萬蘇丹難民，四點二八億美元的資金缺口恐怕會讓救援被迫大幅縮減。查德東部每三個人裡面就有一個是難民。世界糧食計劃署用不到所需一半的資源在運作，已經被迫削減大部分難民的口糧了。

最後在時事的部分補一個氣候的消息。WMO——世界氣象組織——發布了《2026 全球氣候狀況》報告，裡面有一個很嚇人的數字：地球的能量失衡達到了六十五年來的最高紀錄。什麼意思呢？就是地球吸收的太陽能量遠遠超過它向外釋放的量，這個差距越來越大。其中九成的多餘能量被海洋吸收了，所以你會看到海水溫度持續上升、珊瑚白化加劇、沿海城市面臨更大的洪水風險。全球超過三十億人依賴海洋和沿岸資源維生，這不是一個遙遠的問題，而是正在發生的危機。

好，時事的部分差不多了。我們來聊聊科技。

今天科技最大的新聞其實不是什麼新產品，是一個制度面的改變。日本內閣在四月十號通過了修正案，正式把加密貨幣從「支付工具」升級為「金融商品」，跟股票、債券同級。這什麼意思呢？就是以後在日本交易加密貨幣，跟交易股票一樣，受一樣的規範。禁止內線交易、發行方要做年度揭露、無照經營最高關十年。但好處是稅率簡化了——從以前最高百分之五十五的累進稅，降到大概百分之二十的單一稅率。日本的加密交易帳戶已經超過一千兩百萬了，客戶持倉五兆日圓，這個法案如果本會期通過的話，二零二七年四月生效。這對亞洲乃至全球的加密貨幣監管走向都會有影響。

LLM 這邊，GLM-5.1 的後續值得關注。Z.ai——就是以前的智譜 AI——四月七號以 MIT 授權全面開源了這個七百四十四億參數的 MoE 模型。每個 token 只啟用四百億參數，二十萬的 context window，十二萬八千的 output。在 SWE-Bench Pro 這個程式碼基準測試上拿到了百分之五十八點四，超過 Claude Opus 4.6 的百分之五十七點三，成為第一個在這個榜上奪冠的開源模型。不過你也不用太興奮，SWE-Bench Pro 只是一個面向，真正在 Agent 場景下跑起來穩不穩定，還需要更多驗證。

Stripe 的 Minions 也值得聊一下。這是 Stripe 內部的自主 Coding Agent，現在每週產出超過一千三百個 PR——全部零人工撰寫程式碼，但還是要人工 review。主要處理的是重複性工作：升級依賴、API 遷移、跨 codebase 重構。它的架構叫 Blueprint，混合了確定性節點跟 Agent loop。你可以把它想像成不是寫新功能，而是處理那些工程師不想做但又必須做的維護工作。一天一百八十五個 PR，這個規模放在任何公司都很驚人。

微軟這邊動作也不少。Agent Framework 1.0 正式版出了，合併了 Semantic Kernel 和 AutoGen 的概念，支援 MCP 1.0 和 A2A 1.0 雙協定，給了長期支援承諾。同時 Agent Governance Toolkit 也以 MIT 授權開源了，號稱是第一個覆蓋 OWASP Agentic AI 全部十項風險的治理工具包。如果你在做企業級的 AI Agent 部署，這兩個東西是值得認真看的。

生物科技方面，Anthropic 花了四億美金收購 Coefficient Bio，一個不到十個人的團隊。共同創辦人出身 Genentech 的 ML 部門。Anthropic 去年十月才推出 Claude Life Sciences，現在又收了一家做藥物研發 AI 的公司，很明顯是要認真進入生命科學領域。Sanofi、Novo Nordisk、AbbVie 這些大藥廠已經在用 Claude 了。

太空方面，Artemis II 的四位太空人在四月十號晚間成功濺落太平洋，結束了將近十天的繞月之旅。四月六號他們創下了距地球二十四萬八千六百五十五英里的人類最遠紀錄，打破了 Apollo 13 五十五年前的舊紀錄。USS John P. Murtha 把太空人接了回去。而就在 Artemis II 回來的隔天，四月十一號，SpaceX 又用 Falcon 9 發射了第二艘 Cygnus XL，搭載五噸補給送往國際太空站，預計後天抵達。太空物流的節奏越來越快了。

資安方面有兩個值得注意的。Smart Slider 3 Pro 這個 WordPress 外掛的更新系統被攻擊者劫持了，推了一個含後門的版本出去。這種供應鏈攻擊特別危險，因為使用者以為自己是在做正常的軟體更新，結果實際上是在安裝後門。如果你有用這個外掛，趕快檢查版本號是不是 3.5.1.35，有的話立刻移除。另外北韓的 Contagious Interview 攻擊活動越來越猖獗，原本主要瞄準 npm 的 JavaScript 生態系，現在擴展到了 Go、Rust、PHP。手法是在套件庫裡放惡意套件，偽裝成正常的開發工具。開發者如果不小心安裝了，後門就直接進到你的開發環境裡面。這種攻擊對開源生態系的信任基礎是一個很大的挑戰。

順便提一下，SEC 在四月十六號——也就是下週三——要開一個 CLARITY Act 的圓桌會議。參議院四月十三號復會之後，銀行委員會可能在四月底進行 markup。CLARITY Act 基本上要做的事情是把加密貨幣的監管權分清楚：CFTC 管數位商品現貨市場，SEC 管投資合約資產。Bessent 財政部長已經公開敦促國會儘速通過了。搭配日本剛通過的法案來看，你可以說全球主要經濟體正在加速建立加密貨幣的監管框架。

最後快速看一下市場。今天是週日，股市、匯市、商品全部休市。加密貨幣是全年無休的——昨天的數據，BTC 七萬三千六百七十四美元，漲了不到百分之一。ETH 兩千三百一十七美元，漲了超過百分之三，表現比較好一些。整體來說美伊談判的不確定性還在壓著風險資產，但加密市場似乎已經消化了大部分的衝擊。

好，今天就到這裡。明天匈牙利應該會有開票結果，伊斯蘭堡的談判也會繼續。我們明天見。
