早安，我是 April，今天是 2026 年 3 月 23 日，星期一，早上好。

先說說今天的大綱。國際局勢方面，我們要聊三件事：第一，美-以-伊朗戰爭打到第 23 天了，昨天以色列在被伊朗飛彈打中阿拉德之後展開報復，伊朗無人機攻擊 UAE 周邊的船隻，22 個國家同時發聯合聲明，但誰都沒有跟著打仗的意思。第二，烏克蘭的和平談判繼續——在邁阿密，在談判桌上；同一時間在烏克蘭，砲擊繼續，還是四個人死了。第三，全球央行在油價暴漲的情況下，三個主要央行同時選擇維持利率不變，這背後有個很微妙的邏輯。科技圈方面，Anthropic 更新了 Claude 的安全政策，允許提供「公開可得」的武器資訊，引發大量批評；OpenAI 要把員工翻倍；還有一個開源安全工具遭到供應鏈攻擊，影響可能非常大。我們一件一件來聊，你準備好了嗎？

---

首先說中東。

你要理解現在這場衝突，得先想一下它是怎麼開始的。時間是 2 月 28 日，美國和以色列聯手對伊朗展開空襲，目標包括伊朗的核設施和軍事指揮體系，行動中擊殺了伊朗最高領袖哈梅內伊。這是一個歷史性的事件——哈梅內伊在位超過三十年，是伊朗伊斯蘭共和國的精神支柱和最高政治權威。他的死讓伊朗政治體系進入了一個從未有過的空缺期，而當時的伊朗是不可能就這樣吞下去的。

從 3 月 1 號開始，伊朗展開了系列報復。飛彈打以色列城市，無人機攻擊美軍在中東的基地，更重要的是——伊朗的伊斯蘭革命衛隊對霍姆茲海峽發出警告，禁止船隻通過。這條海峽平常承載著全球 20% 的石油運量。封了霍姆茲，就是把全世界的能源安全都掌握在手裡。

這就是目前的背景。現在說昨天。

3 月 22 日，戰爭第 23 天。伊朗飛彈擊中了以色列城市阿拉德——你可能沒有聽說過這個城市，它在以色列南部的内蓋夫沙漠邊緣，距離約旦邊境不遠，人口大概 3 萬多人。以色列宣佈展開報復性打擊。

同日，伊朗的無人機攻擊了沙迦——就是 UAE 的一個城市——外海 15 海里的一艘船隻。英國海事貿易行動中心說，全體船員都平安，但這顯示伊朗已經開始對 UAE 構成直接威脅。

22 個國家——UAE、英國、法國、德國、日本、巴林，等等——同時發出聯合聲明，譴責伊朗封鎖霍姆茲海峽，以及攻擊商業船隻的行為。這份聲明的意義很微妙。你看，22 個國家聯合譴責，聽起來聲勢浩大，但這些國家沒有一個說要跟著美國打。他們願意罵伊朗，但不願意跟著上戰場。這反映了一個很現實的計算：伊朗的能源武器讓很多國家不敢得罪太深，萬一霍姆茲真的完全封死，那些依賴波斯灣石油的國家首先倒楣。

川普方面，他說的是「考慮收兵」。但他特別強調這不是停火。有點像是說：我在想要不要打少一點，但我沒有說停。同時他威脅說，如果伊朗不重開霍姆茲，美軍就攻擊伊朗的電廠。這是一種升級警告，用更高的威脅來換取伊朗在霍姆茲問題上讓步。

但川普同步在做一件很有意思的事：他悄悄解凍了讓伊朗賣原油的臨時許可證。什麼意思呢？就是說，伊朗的石油原本因為制裁無法出售，有大量原油滯留在海上——大概有一億四千萬桶。川普允許這批油賣出去，目的是希望增加全球石油供應，把布蘭特油價從 119 美元的高位拉下來。

你有沒有看出這裡面的矛盾？川普一邊威脅伊朗，一邊幫伊朗的油找市場。這個操作讓很多盟友搞不懂方向，也讓批評者質疑：這到底是在打壓伊朗，還是在跟伊朗做生意？

伊朗方面，他們的態度是確定的——不相信川普的「考慮收兵」。伊朗外長阿拉格齊的說法是，霍姆茲目前沒關，但如果美方攻擊電力基礎設施，就會完全關閉。伊朗把能源基礎設施的安全當做一個籌碼：你不打我的電廠，我就不關霍姆茲。這是一個很直接的交換條件。

阿拉格齊同時說，正在和日本談判，讓日本的相關船隻可以通過海峽。這裡很有意思——伊朗在這個時候主動向日本示好。為什麼是日本？因為日本和伊朗長期有能源合作，日本一直努力維持和伊朗的外交關係，沒有完全跟著美國的立場走。所以伊朗找日本，是在找一個可以溝通的橋梁，同時也是在分化美國的亞洲盟友陣營。

日本現在很被動，但也很主動地在運用自己的角色。外相茂木說，停火後日本考慮協助掃雷——霍姆茲那邊的水雷問題是一個潛在威脅。同時波斯灣裡有 22 艘日本相關船隻，政府正在安排撤離。日本在美伊之間的這個角色，是一個非常細緻的外交操作，既沒有得罪美國，又保留了和伊朗的溝通管道。

英國允許美軍用英國基地打伊朗的飛彈基地——這是軍事上的支持，但英國同時公開批評以色列的激進軍事行動，說以色列「拖著美國」進了一個很難收場的衝突。這種一面支持、一面批評的態度，體現了英國在大西洋同盟框架下試圖保持自己的獨立性。

讓我說一個蠻讓人感慨的細節。Oman 的外長提到，伊朗在 2 月 27 日——也就是被攻擊的前一天——其實已經同意永不儲備濃縮鈾，並接受 IAEA 的核查驗證。如果這個協議成立，和平「唾手可得」。但 2 月 28 日美以就動手了，談判的窗口就這樣被打碎了。這段話讓很多人回頭想：這場戰爭是不是本來可以避免的？

我沒有辦法告訴你答案，但我覺得這個問題值得放在心裡。

傷亡數字現在是：伊朗境內超過 1,444 人死亡，以色列 18 人，美軍 13 人，波斯灣周邊地區 21 人。能源市場上，布蘭特原油從戰前的大約 83 美元漲到現在的 119 美元，漲了超過四成。這個油價如果持續下去，對全球的影響是巨大的，航空、製造業、汽車，每個行業都在算這個成本。

說完中東，我們把目光轉到歐洲。

烏克蘭的情況這週有個新發展——澤倫斯基把政治代表團送到美國邁阿密，希望推進和平談判。美方的特使威特科夫和庫什納——對，就是川普的女婿庫什納——見了烏克蘭代表，說會議「具建設性」，聚焦在縮小分歧和推進全面和平協議。聽起來不錯，但你要知道，克里姆林宮對這一輪談判至今沒有正式回應。俄羅斯連桌子都還沒坐到。

更能說明問題的是：就在談判進行的同一天，俄烏雙方還在互相砲擊，造成至少四人死亡。戰場沒停，談判也在繼續，這兩件事同時存在，讓人感覺整個和平進程有點虛幻。

你可能想問，為什麼談判還在進行，雙方還在打仗？這有幾個層面的原因。第一，俄羅斯沒有派軍事代表出席，這意味著即使政治上達成某種共識，執行機制——也就是如何讓雙方軍隊停下來——根本沒有人在討論。沒有執行機制的停火是沒有意義的。第二，俄羅斯可能在等待更好的時機——等美國被中東拖住、等烏克蘭在戰場上更疲憊、等冬天讓烏克蘭的能源基礎設施雪上加霜——然後在談判桌上要到更多。

西方分析師的擔憂是，普亭可能趁美國注意力分散，在烏克蘭戰場加大壓力，把軍事上的優勢轉化成談判桌上的籌碼。這個邏輯，在過去幾年裡我們見過多次了。

歐盟方面，有一個蠻讓人感慨的新聞。歐盟和川普的關稅談判，最後的結果是歐盟讓步了。接受 15% 的對等關稅，而且要承諾購買額外 7,500 億美元的美國能源——主要是天然氣。歐盟原本說要用「核選項」，也就是全面報復性關稅來抵制，但最後退讓了。

為什麼退讓？理由很現實——在中東戰爭的背景下，歐洲的能源安全本來就已經很脆弱，如果再跟美國打貿易戰，政治和經濟上都承受不了。但批評者說，這個協議讓歐洲「替美國安全保護付租金」——買美國的天然氣換取美國的安全承諾，而且這些能源購買承諾和歐盟的氣候目標根本矛盾。歐洲議員批評這是「屈辱性妥協」，但決策已經做了。

全球央行這週的決定也值得說一說。美聯儲、歐洲央行、日本銀行全部維持利率不變。這個集體不動的背後，其實是一個共同的困境：油價飆漲理論上是通膨壓力，傳統的回應是升息；但油價飆漲同時也在抑制經濟成長和消費，如果這時候升息，可能讓衰退的壓力更大。所以三個央行都選擇先觀望。Bloomberg 的分析說，如果油價持續維持在 119 美元以上，英格蘭銀行和日本銀行在第二季可能會被迫升息。美聯儲今年預計只會降息一次。

這對普通人意味著什麼？借貸成本不會降，房貸利率還是高的，企業融資成本還是高的，同時你去加油站的費用在漲，你買的東西在漲。這是能源危機在日常生活裡的具體感受。

東亞這邊，說幾件事。

日本和韓國的關係，有一個我覺得很值得記錄的趨勢。Bloomberg 這週有一篇分析，標題說的是「川普和習近平在把日本和韓國推向更近」。這兩個國家因為歷史問題——主要是二戰時期的強制勞役和慰安婦問題——長期以來在外交上有很多摩擦，安全合作也有很多障礙。但是現在，面對川普要求盟友「自己負擔防務費用」的壓力，加上解放軍持續在台灣海峽和南海的動作，日本和韓國開始走得越來越近。

日本首相高市早苗和韓國總統李在明今年一月就舉行了雙邊峰會，這個畫面在幾年前很難想像。防務資訊共享、聯合演習的深度都在增加。當你不確定你的保護傘還在不在的時候，你就會找離你最近的人抱團。這是很直觀的地緣政治邏輯。

台灣這邊，國防部說中國的軍事威脅「持續且迫切」，而且中國的軍事建設「從未停歇」。但同時，追蹤數據顯示解放軍三月份飛越台灣防空識別區的次數，比二月明顯下降了。這個反差怎麼解讀？分析人士認為，北京可能在觀察中東戰爭的走向，暫時保持戰略收斂——不是因為威脅減少了，而是在等待更好的時機。

中國一直在玩的是一種「灰色地帶」戰略。封鎖演習、南海巡邏、對菲律賓漁民施壓、忽然讓海警船出現在塔斯曼海——這些動作都在不觸發軍事回應的閾值之下，但每一個動作都在蠶食對方的心理抵抗力。現代戰略理論把這叫做「勝而不戰」的邏輯——讓對方慢慢相信抵抗是徒勞的。

美國的分析師強調，台灣製造全球 90% 的先進半導體，台海穩定對美國的國家安全是「存在性利益」。但現實情況是，美國深陷中東，在台灣問題上的注意力和資源都在分散。這就是北京在靜待的窗口。

中國這邊還有一個值得注意的動作。國務院總理李強最近承諾給外資企業「國民待遇」，在和輝瑞、阿斯特捷利康等五大跨國藥廠的會議上，承諾強化智慧財產保護和提升政策透明度。這個動作是在說，中國願意對外資更開放。但 FT 的批評是，過去類似的承諾很多，但實際落地受到各種隱性障礙——監管要求、數據本地化、不透明的審批流程。承諾和實際執行之間的差距，一直是外資進中國市場的核心障礙。

韓國那邊，有一個很不幸的消息——一處汽車零件工廠發生嚴重火災，14 人死亡，60 人受傷。消防人員持續搶救。韓國製造業的安全問題，在這種事故後又會有一輪審視。

南亞方面，我想說一個可能影響深遠的新聞。美國國家情報總監塔爾西·加巴德在 3 月 18 日向參議院提交的年度威脅評估，把巴基斯坦的核武和彈道飛彈計劃列為對美國的重大威脅——而且用了一個過去從來沒有出現過的措辭，說巴基斯坦的先進彈道飛彈射程「可能已涵蓋美國大陸」。這把巴基斯坦和俄羅斯、中國並列，是一個非常嚴厲的定性。

巴基斯坦方面當然強烈反駁，說他們的核武完全是為了應對印度的威脅，美方的說法是雙重標準。印度方面，媒體報導說官員「暗自滿意」——因為這等於美方正式承認了巴基斯坦作為核威脅的嚴重性，符合印度長期以來的立場。

但戰略分析師指出這個時機非常敏感。美國在中東戰爭高峰期公開點名巴基斯坦核威脅，這可能有兩個目的：一，施壓巴基斯坦切斷對伊朗的任何後援；二，重新定義美-巴關係的籌碼分佈。無論如何，在全球多個衝突同時進行的情況下，多一個核大國被列為直接威脅，是讓人不安的信號。

澳洲這邊，燃料供應還是穩定的，能源部長說暫時沒有配給計劃，儘管中東戰爭對全球石油供應有衝擊。但北邊的麻煩是熱帶氣旋 Narelle 正在逼近北領地，當地已經啟動緊急應變。

好，聊完國際局勢，我們換個頻道，來看看科技圈。

今天科技最大的話題，我想說 Anthropic 的政策調整。

你可能知道 Anthropic 這家公司。創辦人 Dario Amodei 和 Daniela Amodei 當初是從 OpenAI 出來的，理由就是他們認為 OpenAI「不夠重視 AI 安全」。所以他們創立 Anthropic，把「負責任的 AI 開發」作為整個公司的核心定位。估值 615 億美元，最近還在和 Palantir 合作，成為唯一一個被用於美軍機密任務的 AI 模型。

就在這幾天，Anthropic 悄悄更新了 Claude 的使用政策。新政策允許 Claude 在資訊「已公開可得」的前提下，回應涉及武器、爆炸物及危險材料的問題。這句話聽起來很有道理——公開資訊嘛，你自己去搜尋引擎也找得到。但問題就在這裡。

「資訊已公開」這個標準，幾乎可以覆蓋任何事情。網路上什麼資訊找不到？更根本的問題是：AI 的危害不只是它「知道」什麼，而是它能夠把零散的公開資訊整合起來，用清晰的、步驟化的方式呈現，讓原本需要大量專業知識才能執行的操作變得容易很多。這個整合和簡化的能力，才是 AI 在危險資訊問題上的核心風險。

AI 安全研究圈對這個政策調整反應很激烈。批評者說，如果連 Anthropic 都在鬆綁，那整個 AI 行業的安全底線就很難維持了。這也讓監管機構的介入顯得更加合理——當業界自律失敗，法規就會出現。

我覺得這件事值得我們去想：在 AI 商業競爭的壓力下，安全承諾能撐多久？OpenAI 要把員工翻倍，Anthropic 在鬆綁政策，Google 也沒有停步，三家公司都在拼速度、拼規模。當速度優先，安全就容易成為被犧牲的第一個。

說到 OpenAI 的消息——《金融時報》報導，OpenAI 計劃在 2026 年底前把員工人數從目前的 4,500 人擴張到 8,000 人，幾乎翻倍。這發生在整體科技業裁員潮的背景下，反映了 AI 核心賽道的人才需求仍然非常旺盛。OpenAI、Anthropic、Google 三家在搶同一批頂尖研究員，人才是最稀缺的資源。批評者的問題是：急速擴招會不會稀釋公司文化，特別是在 OpenAI 組織架構還在從非營利到營利轉型的過渡期？

開源工具這邊，Ollama 發布了 v0.18.2，修了一個 model 選取的 bug，讓本地跑大型語言模型更穩定了。LangChain 也更新了，langchain-anthropic 升到 1.4.0，和 Claude 的整合更順了。這些工具的持續迭代讓本地 AI 部署的門檻越來越低，開發者的生態系越來越豐富。

AI Agent 這個方向，有兩個值得記錄的消息。第一，一家叫 Manifold 的新創公司完成了 800 萬美元的種子輪融資，由 Costanoa Ventures 領投。Manifold 做的是 AI Agent 的偵測與回應平台——也就是說，他們在保護企業環境裡的 AI Agent 不被攻擊。隨著 Agentic AI 進入生產環境，攻擊面在急速擴大，這個垂直賽道正在爆發。

第二，一個叫 0G 的區塊鏈基礎設施項目宣佈定位為「AI Agent 的底層區塊鏈」。他們的邏輯是：如果 Agentic AI 時代真的到來，AI Agent 需要進行鏈上的資產控制和決策，就需要一個低延遲、高吞吐量的底層。但批評者的問題是：真正需要鏈上結算的 AI Agent 用例，至今仍然是少數；去中心化本身帶來的延遲和複雜性，對多數企業應用來說反而是阻礙。AI × Crypto 的敘事被市場反覆炒作，但落地情況一直沒有達到敘事的高度。

接下來說一個我覺得所有在用 CI/CD 流程的開發者都應該知道的事情。

Trivy——這是一個非常廣泛使用的開源漏洞掃描工具，很多公司的 GitHub Actions 都在跑它——遭到了供應鏈攻擊。攻擊者組織叫 hackerbot-claw，他們利用 GitHub Actions 裡一個叫 `pull_request_target` 的工作流程漏洞，竊取了一個 Personal Access Token。拿到這個 token 之後，他們劫持了 Trivy 的 GitHub repository，篡改了 75 個 release 標籤，在裡面植入了 infostealer——就是一種會竊取你的登入憑證和密鑰的惡意程式。

受損版本大概存活了三小時，但被篡改的 GitHub Actions 標籤維持活躍長達 12 小時。在這段時間裡，所有在 CI/CD pipeline 裡跑這些 Actions 標籤的系統，都可能把自己的 API 密鑰、SSH 金鑰、資料庫密碼等等洩漏出去了。

Trivy 團隊後來發現了問題，輪換了所有密鑰，推出了乾淨版本。但這件事暴露了一個系統性的漏洞：開源生態裡，我們對這些 Actions 標籤的信任幾乎是盲目的。你的 CI/CD pipeline 拿著你的最高權限，去執行一個你從來沒有驗證過完整性的外部腳本。

如果你現在用的是浮動版本標籤——就是像 `uses: aquasecurity/trivy-action@v0.18` 這樣的——你應該立刻把它改成指定 commit hash，比如 `uses: aquasecurity/trivy-action@a1f8b3e4...`。Hash 是不可篡改的，標籤可以被重新指向任何地方。

Oracle 這邊也有一個緊急修補需要注意。他們的 Identity Manager 和 Web Services 有一個 CVE-2026-21992 的漏洞，CVSS 評分 9.8，幾乎是滿分，屬於未驗證遠端程式碼執行。意思就是說，攻擊者不需要任何憑證，就可以在你的伺服器上執行任意程式碼。如果你的系統有暴露在網路上的 Oracle Identity Manager 實例，應該立即套用 Oracle 發布的緊急補丁。

太空這邊有一個讓我覺得很值得分享的消息。第三顆有史以來被觀測到穿越太陽系的星際天體，名字叫 3I/ATLAS，在離開太陽系之前，被超過 12 個 NASA 科學任務密集觀測。這創了記錄——我們第一次拿到一顆星際彗星的完整資料集。這些天體不是我們太陽系裡形成的，它們帶著的是其他恆星系的物質，分析這些資料可能讓我們了解宇宙裡其他地方的化學組成是什麼樣子，甚至可能找到其他行星系形成的線索。科學家預計分析成果會在未來數年陸續發表。

還有一個天文謎題被解開了：小麥哲倫雲，距離地球大約 20 萬光年，是一個矮星系，長期以來裡面的恆星運動模式讓天文學家困惑——它們的方向和速度非常混亂，不像一個正常演化的星系應該有的樣子。現在利用歐洲太空總署的蓋亞衛星數據，科學家發現這個混亂是因為數百萬年前，小麥哲倫雲和旁邊的大麥哲倫雲發生了一次星系尺度的碰撞，把整個結構都打亂了。宇宙就是這樣，有時候混沌的背後，有一個非常暴力的故事。
