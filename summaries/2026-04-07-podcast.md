早安，各位聽眾朋友，今天是四月七號，星期二，歡迎收聽每日新聞 Podcast 第三十八集。

今天有幾件大事要跟你聊。國際局勢方面，美伊戰爭來到了一個可能改變整場戰爭走向的關鍵時刻——今天晚上就是川普那個最後通牒的到期時間，而伊朗正式遞交了一份十點反提案，同時三國斡旋的四十五天停火框架也在談判中。歐洲這邊，歐盟執委會的雲端系統被一個非常巧妙的供應鏈攻擊攻破，影響了將近三十個歐盟機構。匈牙利大選倒數五天，反對黨民調領先繼續擴大到了歷史性的二十三個百分點。台灣這邊，航空燃油附加費今天正式調漲一百五十七個百分點，國民黨主席鄭麗文也在今天啟程訪中。

科技圈方面，比特幣因為停火消息暴漲突破六萬九，一個 CVSS 九點一的重大資安漏洞正在全球被利用，Anthropic 和 OpenAI 都有新的動態。太空那邊，Artemis II 昨天完成繞月，正在回家的路上，後天還有 ESA 跟中國科學院合作的 SMILE 衛星要發射。內容很多，我們一個一個來。

先來看今天最重要的——美伊戰爭。這場戰爭今天進入第三十八天，而今天晚上的最後通牒到期可能是開戰以來最關鍵的一個時間節點。讓我把目前的情勢幫你理清楚，因為同時有三條線在跑。

第一條線是伊朗的正式回應。伊朗透過巴基斯坦向美方遞交了一份十點反提案。這份文件的內容相當完整，包括荷姆茲海峽的安全通行協議——也就是允許商船在某些條件下恢復通行、戰後重建框架、以及解除制裁的時間表。但最關鍵的一點是——伊朗明確拒絕了任何形式的臨時停火。他們的立場很清楚：要談就談永久結束戰爭，不接受那種「先停幾天看看」的安排。從伊朗的角度來看，這其實有道理——如果接受臨時停火，等於承認美國有權隨時重啟軍事行動，這在國內政治上是賣不出去的。

第二條線是多國斡旋。埃及、巴基斯坦和土耳其三國正在推動一個四十五天的停火框架，這是目前桌上最具體的和平方案。巴基斯坦的版本是先立即停火，然後在十五到二十天內完成全面和平談判。但雙方的底線差距太大——美國要求伊朗先重開海峽再談其他，伊朗堅持先談整體方案再開海峽。這基本上就是「誰先讓步」的問題，而到目前為止，雙方都不願意先退一步。

第三條線是川普的最後通牒。他昨天說伊朗的提案「重要」但「不夠好」，維持今晚東部時間八點的期限。伊朗那邊的回應也很強硬——他們警告如果川普真的動手，伊朗的回擊將會「更嚴厲、更大範圍」。你看，這個語言其實已經在為可能的升級做鋪墊了。

那市場怎麼看呢？Polymarket 上面，四月七號停火合約只有百分之一到四的 YES。也就是說，聰明錢幾乎不相信今天會達成停火。但反過來說，如果真的出了什麼意外的突破，市場的反應會非常劇烈，因為所有人都已經 price in 了「不會停火」這個結果。油價昨天的走勢就很能說明問題——WTI 早盤一度衝到一百一十五美元，之後因為停火消息回落到一百一十二左右。今天晚上不管結果如何，波動只會更大。

好，再來看中東的另一條戰線。以色列在黎巴嫩的戰爭已經持續超過一個月了。這幾天最大的新聞是以色列的戰略目標發生了公開的轉變——國防部長 Israel Katz 宣布計畫摧毀多個黎巴嫩邊境城鎮，並維持對南黎巴嫩的長期佔領。部分以色列官員甚至主張把邊界北移將近十英里到黎巴嫩境內。這是一個非常重大的轉變，因為最初以色列說的是「解除真主黨武裝」，現在變成了「重建安全區」，這聽起來更像是一個長期佔領的計畫。

MERIP——中東研究與資訊計畫——刊登了一篇來自前線的報導，引述了以軍士兵的話說「我們的指南針壞了」，意思是連自己的軍人都開始質疑這場戰爭的方向。Foreign Affairs 的分析更悲觀，說黎巴嫩正面臨國家崩潰的臨界點——一百二十多萬人流離失所，佔全國人口百分之二十。

好，來看歐洲。匈牙利大選倒數五天，局勢越來越有趣了。Medián 最新民調顯示反對黨 Tisza 領先幅度擴大到二十三個百分點——百分之五十八對百分之三十五。21 Research 的數據也差不多，五十六對三十七。如果這個差距在選舉日兌現，Tisza 甚至有可能拿到三分之二的國會多數，也就是修憲多數。想想看，奧班執政十六年，建立了整套符合自己利益的制度——從選區劃分到媒體環境，而現在反對黨的領先幅度大到可能連這些制度優勢都扛不住。

不過我要提醒你，這裡面有一個很大的疑問。親政府的民調機構 Nézőpont 給出的數字跟獨立民調完全不一樣——他們說 Fidesz 領先六個百分點。政治學者 Gábor Török 說，政府民調和非政府民調之間這麼大的差距，在匈牙利政治史上是前所未見的。到底誰說的對？四月十二號就知道了。但不管結果如何，這場選舉對歐盟的意義巨大——奧班過去幾年一直在否決各種歐盟提案，從烏克蘭入會到共同庇護政策，如果他下台，歐盟的決策效率可能會出現質的變化。

另外，今天是世界衛生日。WHO 發起了一個叫「與科學同行」的全年運動，在全球反智浪潮此起彼伏的時候，這個運動的政治意味其實蠻重的。法國以 G7 輪值主席的身份在巴黎辦了 One Health 高峰會，八十多個國家近八百個科學機構參加。

台灣這邊有兩個重要新聞。第一，從今天開始，航空燃油附加費正式大幅調漲。短程從五百六十三元台幣漲到一千四百四十八元，長程從一千四百六十四元漲到三千七百六十六元，漲幅一百五十七個百分點。這完全是荷姆茲封鎖的直接後果——全球航空燃油價格隨原油一起飆漲。台灣現在正好要進入旅遊旺季，這個漲幅對旅遊業的衝擊會很明顯。第二個新聞是國民黨主席鄭麗文今天啟程前往北京，受習近平邀請進行為期六天的訪問。會談議程還是沒有公開，在美中關係緊張、伊朗戰爭持續的大背景下，這個訪問的時機和內容都值得密切關注。

好，聊完國際局勢，我們來看看科技圈今天有什麼動靜。

加密貨幣市場是今天科技圈最大的新聞。前面提到的那個美伊停火談判消息出來之後，BTC 在短短幾個小時內就漲了超過百分之三點二五，突破六萬九千美元，二十四小時交易量超過二百九十億美元。為什麼漲得這麼快？一個字——軋空。之前的市場情緒是伊朗戰爭開打以來最悲觀的水準，大量交易者做空加密貨幣，空頭倉位極度擁擠。當 Axios 那篇獨家報導一出來說美伊在討論四十五天停火，這些空頭就被集體軋倉了。

但你不要被這個漲勢迷惑了。市場對停火的實際達成還是極度懷疑。而且更有趣的是，就在比特幣漲的同時，Bloomberg Intelligence 的分析師 Mike McGlone 重新發表了他的極端看空觀點——他認為比特幣如果不能站穩七萬五千美元，長期目標是回到一萬美元。他的邏輯是比特幣的長期均衡價格遠低於目前水準。這個觀點你可以不同意，但他確實代表了機構投資者中一部分人的看法。

另一個加密貨幣的消息是 Gnosis Chain 宣布將在四月十四號啟動 Fusaka 硬分叉。這是跟進以太坊主網去年十二月的同名升級，核心功能是 PeerDAS，用概率性的數據可用性抽樣來取代完整的 blob 下載，聽起來很技術，但簡單說就是讓節點運行得更輕量、更省頻寬。

資安方面。歐盟執委會的雲端被駭這件事我覺得每個搞技術的人都應該認真看一下，因為攻擊手法非常精巧。攻擊的源頭是一個叫 Trivy 的開源漏洞掃描工具——你可能用過或者聽過，它是 Aqua Security 維護的，在 DevSecOps 圈子裡非常普遍。TeamPCP 這個駭客組織做了什麼呢？他們在三月十九號透過 Trivy 的正常更新管道散佈了一個木馬化的版本。也就是說，當你像平常一樣更新 Trivy 的時候，你其實已經把後門裝進自己的系統了。

歐盟執委會在三月二十四號偵測到 AWS 環境有異常 API 活動，調查之後發現已經被偷走了超過三百 GB 的資料。受影響的範圍非常廣，四十二個執委會內部部門和至少二十九個其他歐盟機構。三月二十八號，勒索組織 ShinyHunters 把偷來的資料公布在暗網上。Mandiant 的評估更可怕——他們說這波攻擊已經波及了超過一千個 SaaS 環境，意味著受害者遠不只歐盟。

這件事的教訓非常清楚：供應鏈安全不只是嘴巴說說。歐盟自己在推 NIS2 指令、DORA 法規這些網安規範的時候講得頭頭是道，但自己的基礎設施卻因為沒有充分驗證開源工具的更新而被打穿。如果連歐盟都中招，其他組織更要檢視自己的軟體供應鏈流程。

另一個重大的資安事件是 Fortinet 的漏洞。FortiClient EMS 的零日漏洞 CVE-2026-35616，CVSS 評分九點一，從三月三十一號開始就已經在野外被利用了。這個漏洞讓未授權的攻擊者可以繞過 API 認證、提權並執行任意程式碼。CISA 在四月六號已經發了緊急令，要求聯邦機構在四月九號之前完成修補。如果你的組織在用 FortiClient EMS 7.4.5 或 7.4.6 版本，真的要趕快處理。

AI 方面有幾個值得一提的消息。Anthropic 最近的狀況有點多。上週 Claude Code 的原始碼意外洩漏之後，這週又有內部文件流出，揭露了正在開發的新一代 Claude Mythos 模型的資訊。據說這個模型在推理、編程和網路安全能力上有重大升級，但洩漏的材料同時也警告這個模型可能帶來新的網安風險。另外一個比較重大的動作是 Anthropic 用大約四十億美元的股票收購了 AI 生技新創 Coefficient Bio，顯示他們在生命科學領域的野心。

OpenAI 那邊也有大消息。GPT-5.5 代號 Spud 的訓練已經完成，正在做安全評估，預計數週內就要發布了。有研究者說四月可能是 AI 模型史上最關鍵的月份，因為多個重量級模型——不只 OpenAI 的——預計在同一個時間窗口推出。如果真是這樣，四月底到五月初可能會出現一波密集的模型發布潮。

企業 AI 應用方面，Stripe 內部使用的 AI 編程代理曝光了。他們叫它 Minions，基本上就是用 LLM 搭配藍圖和 CI/CD 管線來生成可以直接上線的程式碼，同時保留人工審查機制。這是第一家大型支付公司公開分享 AI 代理開發流程的實例。另外 Qualys 推出了一個叫 Agent Val 的安全代理，可以自動發現系統漏洞、判斷威脅並自主修補。AWS 也在 GitHub 上成立了 Strands Labs，專門做代理式 AI 的實驗性開源專案。你可以看到，Agent AI 正在從概念走向實際部署的階段。

太空方面有兩個消息。Artemis II 昨天成功完成了繞月飛行——四位太空人最遠離地球四十萬六千七百多公里，觀測了月球背面，還目睹了一場太空日食。現在飛船正在利用地球引力的彈弓效應返航，預計四月九號到十號在太平洋聖地牙哥外海濺落。另一個太空消息是後天四月九號，歐洲太空總署和中國科學院合作的 SMILE 衛星要從法屬圭亞那發射。SMILE 是第一個用 X 射線來全面觀測地球磁場如何回應太陽風暴的衛星。有意思的是，在當前地緣政治環境下，歐洲和中國的太空合作仍然在推進。
