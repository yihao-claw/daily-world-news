嗨，歡迎回來，這裡是每日新聞 Podcast，我是你的主持人。今天是四月十三號，週日，復活節假期，但世界可一點都沒有放假的意思。今天最重要的兩件事：Trump 直接下令封鎖荷莫茲海峽，然後匈牙利正在進行一場可能改寫歐洲政治版圖的選舉。我們直接開始。

先講最震撼的。美國跟伊朗在巴基斯坦伊斯蘭堡的談判，徹底破裂了。副總統 Vance 帶隊跟伊朗人談了超過二十一個小時，結果在核武議題上完全談不攏。Trump 的反應非常激烈，直接在 Truth Social 上宣布：美國海軍即刻封鎖荷莫茲海峽，所有進出的船隻都要被攔截，同時開始清除伊朗布下的水雷。

你要知道，荷莫茲海峽大概承載了全世界百分之二十的原油運輸量，每天差不多兩千萬桶原油要經過那個窄窄的水道。這不是一個小動作，這幾乎是在全球能源市場上投了一顆炸彈。伊朗革命衛隊馬上回嗆，說海峽依然開放，視封鎖為戰爭行為。你想想看，如果美國海軍真的開始攔截商船，伊朗會怎麼反應？他們有能力用快艇和反艦飛彈騷擾美軍船隊，這個升級的風險是真實存在的。中國外交部也出來喊話要各方克制，因為中國大約四成進口石油要走荷莫茲，北京的焦慮非常真實。分析師現在在討論的是，如果封鎖真的全面執行，油價會不會衝破一百美元一桶。上週四收盤的時候 WTI 原油差不多九十六塊多，已經很高了，加上封鎖溢價，破百不是危言聳聽。加密市場已經先反映了，比特幣跌破七萬一，以太坊跌破兩千兩百，二十四小時內清算量暴增將近百分之九十。有看多的分析師認為比特幣在六萬五有強支撐，但地緣風險持續壓制風險資產的意願，短期內恐怕還是逆風。

而且有一個很有意思的副作用。因為戰爭的關係，很多大宗商品交易商被銀行切斷服務了。合規銀行不願意碰跟伊朗有任何沾邊的交易。結果這些交易商被迫轉向穩定幣，用 USDT、USDC 來結算。你可以說，戰爭正在加速加密貨幣滲透進傳統貿易的速度，這是一個非常值得關注的結構性變化。

好，我們離開中東，去歐洲。匈牙利今天舉行了國會大選，這場選舉太關鍵了。奧班執政十六年，Fidesz 黨看起來第一次真的可能會輸。反對派 Tisza 黨的領袖 Péter Magyar 選前民調一直領先。今天的投票率衝到了百分之七十七點八，打破了二零零二年的歷史紀錄。Magyar 說他「審慎樂觀」，但如果票數太接近，完整計票可能要到下週六才會出來。

為什麼這場選舉對整個歐洲都重要？因為奧班一直是歐盟裡面最大的反對派，動不動就用否決權擋住歐盟的決策，不管是對俄制裁還是烏克蘭援助。如果他下台，歐盟的決策效率可能會有質的飛躍。當然保守派的觀點是，Magyar 上台代表中歐會失去一個敢於對抗布魯塞爾的聲音。

再講另一個歐洲的事。俄烏的復活節停火，嗯，基本上就是一個笑話。Putin 宣布三十二小時停火，從昨天下午四點到今天午夜。結果呢？烏克蘭方面記錄到兩千兩百九十九次俄軍違規，包括攻擊、砲擊、無人機。俄方也不甘示弱，說烏克蘭違規了一千九百七十一次。基輔市民還是在戶外慶祝復活節，但你訪問他們，每個人都說「持久和平？我們不太抱希望。」這已經不是第一次了，去年復活節 Putin 也宣布過停火，結果一模一樣。

然後中東還有另一條線。以色列跟黎巴嫩預定週二在華盛頓談判，但 Netanyahu 已經先放話了：不會討論停火。他說以色列只談「解除真主黨武裝跟建立和平關係」。問題是，黎巴嫩方面說雙方之前已經就停火達成共識了。兩邊的說法完全矛盾。自從四月八號的「永恆黑暗行動」以來，黎巴嫩已經有將近兩千人喪生。那天以色列出動五十架戰機，在十分鐘內對至少四十八個區域發動攻擊，包括貝魯特的住宅區，光那一天就超過三百五十人死亡。國際特赦組織把這稱為無差別攻擊。

往東走，講一下兩岸。國民黨主席鄭麗文上週在北京人民大會堂跟習近平會面了。這是二零一五年馬習會之後，兩岸最高層級的政治接觸。習近平講了「和平發展」跟「一家人」，鄭麗文暗示會放緩台灣的軍事擴張。民進黨當然砲火猛烈，批這是「投降外交」。但國民黨的論述是這是「和平之旅」，目的是降低台海緊張。有意思的是，中國的四十天空域預留還是沒有解除，軍事壓力跟對話管道同時並存，這個矛盾本身就很耐人尋味。

接下來，美國國內也有大事。二十三個州加上 DC 聯合提告 Trump 的郵寄投票行政命令。三月底簽的這道命令要求國土安全部跟社會安全局建一份「合格選民名冊」，美國郵政只能寄選票給名冊上的人。加州領頭在麻州聯邦法院提告，核心論點就是：總統無權片面改變各州的選舉制度，這違反憲法。

好，非洲有兩件事值得講。第一，教宗 Leo 十四世今天啟程了，他要花十天時間訪問阿爾及利亞、喀麥隆、安哥拉跟赤道幾內亞。最有意義的是阿爾及利亞站，他會成為史上第一位訪問阿爾及利亞的教宗，而且會參訪世界最大的清真寺之一。在基督教與伊斯蘭對話這個主題上，這是一個非常有力的象徵。第二，蘇丹的難民危機持續惡化，查德境內一百三十萬蘇丹難民面臨四億兩千八百萬美元的援助缺口，WFP 跟 UNHCR 警告食物、飲水、醫療都將大幅削減。二零二六年蘇丹估計有三千三百七十萬人，也就是百分之六十的人口，需要人道援助。

大洋洲這邊有一個值得關注的動向。紐西蘭跟庫克群島簽了一份安全宣言，庫克群島正式承諾紐西蘭是「國防安全首選夥伴」。背景是去年庫克群島總理 Brown 跑去北京跟中國簽了全面戰略夥伴協議，而且事前沒有告知紐西蘭內容，搞得惠靈頓非常不高興，直接凍結了發展援助。現在這份安全宣言算是修補了裂痕，紐西蘭也恢復了援助。但有意思的是，雙方都刻意迴避評論庫克群島跟北京的那份協議到底怎麼處理。太平洋島國在中美之間的平衡外交，正在變得越來越微妙。

非洲還有一件事要提。吉布提總統 Guelleh 以百分之九十七點八的得票率連任第六任期，反對派杯葛了這次選舉，投票率百分之八十點四。Guelleh 從一九九九年執政到現在，吉布提同時是美國、中國、法國和日本軍事基地的所在地，戰略地位特殊。

另外印度阿薩姆邦的選舉已經在四月九號投完票了，投票率高達百分之八十五，創歷史新高。七百二十二個候選人搶一百二十六個席次，BJP 要拚三連任，國大黨要翻盤。結果五月四號才出來，到時候再跟你報告。菲律賓那邊，ICC 預計四月二十二號要對杜特蒂的管轄權上訴做出裁定。杜特蒂去年三月被移送海牙，他的律師主張菲律賓都已經退出 ICC 了，法院沒有管轄權。但預審庭說犯罪發生在會員期間，管轄權成立。這個判決很關鍵，可能決定杜特蒂到底要不要受審。

好，科技時間。

今天科技圈最有意思的一篇文章，是 Gary Marcus 寫的。他說 Claude Code 是「LLM 以來最大的進展」，但原因不是你想的那樣。他的論點是，Claude Code 的核心有一個三千一百六十七行的 print.ts，裡面用的是古典 symbolic AI 的 pattern matching，不是純神經網路。所以他說 Anthropic 其實走的是 neurosymbolic 路線，跟 AlphaFold、AlphaProof 一樣。Marcus 當然是在幫自己背書，他倡導 neurosymbolic AI 已經二十五年了。但批評者認為他過度簡化了 hybrid 架構的定義，把所有不純是 LLM 的東西都歸類為 neurosymbolic 有點太寬泛了。不管你站哪邊，這個討論本身很有價值，因為它問的問題是：當我們說 AI 進步了，到底是哪個部分在進步？

開源圈有個大版本。SQLite 三點五三釋出了。跳過被撤回的三點五二，這個版本修復了一個 WAL reset 導致的資料庫損壞 bug，這很嚴重。新增了 ALTER TABLE 可以直接增刪 NOT NULL 跟 CHECK 約束、新的 json_array_insert 函式，還有一個全新的 Query Result Formatter 讓 CLI 輸出漂亮很多。Simon Willison 評價說這是一個「積累了大量改進的重要版本」。如果你的專案用 SQLite，建議升級。

Ollama 出了零點二零點六的 RC1，主要是同步 Google 更新的 Gemma 4 模板，修了平行 tool call 的 index 遺失問題。對在本地跑 LLM 的開發者來說是必要的穩定性修正。

LangChain Core 一點三點零的第一個 alpha 也出了，減少串流 metadata 開銷提升效能。OpenClaw 新版支援 ChatGPT 對話匯入跟 Memory Palace 功能。

加密貨幣監管方面，美國 CFTC 主席 Selig 撰文主張對預測市場的「專屬監管權」，正值 Polymarket、Kalshi 快速成長。SEC 下週三的圓桌會議要討論 CLARITY Act，這可能決定數位資產到底歸誰管。另外 Justin Sun 跟 Trump 旗下的 WLFI 公開翻臉了，Sun 指控 WLFI 代幣合約藏有黑名單後門，WLFI 直接回嗆「法庭見」。兩邊都不是省油的燈。

資安方面快速帶一下。Fortinet 的 FortiClient EMS 有一個很嚴重的預認證繞過漏洞，CISA 已經列入必修名單。三月底就偵測到野外攻擊了，未經認證就能繞過 API 認證執行惡意程式碼。如果你的公司有用 FortiClient，趕快更新。另外 WordPress 用戶要注意，Smart Slider 3 Pro 外掛的更新系統被劫持了，攻擊者在更新管道裡植入後門，供應鏈攻擊又一例。

生技方面，Insilico Medicine 跟 Eli Lilly 簽了一筆重磅交易，總值二十七億五千萬美元，預付一億一千五百萬。Lilly 拿到了 Insilico 用 AI 發現的臨床前候選藥物的開發和商業化權利。這是 AI 藥物開發領域迄今最大的單筆交易之一，信號很明確：大藥廠不再只是觀望 AI，而是真金白銀砸下去了。

全球智慧型手機第一季出貨微增百分之一，Apple 靠 iPhone 17 拿下百分之二十一市佔居冠，Samsung 百分之十八靠中低價機型撐住，小米百分之十一攻新興市場。但 DRAM 跟 NAND 短缺的壓力持續，因為供應商把產能優先給 AI 資料中心。另外 Apple 的折疊 iPhone 確認採用 Samsung 獨家 OLED，三年獨家合約，定價大概兩千到兩千四百美元，預計九月發表。

市場快報。今天是週日，全球股市休市。週四的股市收盤數據已經超過二十四小時了，我們不報。匯率方面，週六的最新報價，美元兌日圓一百五十九點三九，微漲零點一八個百分點；歐元兌美元一點一七，漲零點二個百分點；美元兌台幣三十一點七七。加密貨幣，比特幣七萬一千一百九十，跌兩個半百分點；以太坊兩千兩百零三，跌三個半百分點。荷莫茲封鎖的消息明天亞洲開盤後會怎麼反映，值得密切觀察。

最後補充一下，Artemis II 已經圓滿完成了，太空人安全返回地球。這是一九七二年 Apollo 17 以來人類首次飛越月球。NASA 的 Roman 太空望遠鏡也完成組裝了，四月二十一號開放媒體參觀。太空探索在沉寂了半個世紀之後，真的在加速回來了。

好，今天就到這裡。復活節假期，但今年的復活節一點都不平靜。烏克蘭的停火名存實亡，荷莫茲海峽的封鎖令讓全球能源市場繃緊神經，匈牙利可能正在見證一個時代的結束。明天亞洲開盤的時候，市場會怎麼反映封鎖消息，非常值得關注。我們明天見，祝你有個平安的假日。
