早安，今天是四月十號，星期五。我是你的新聞主持人，歡迎收聽 EP39。

今天有幾件大事要跟你聊。國際局勢方面，美伊停火才簽了兩天就快撐不住了，以色列在停火生效當天對黎巴嫩發動了史上最大規模空襲，一天炸死三百多人，這讓整個停火協議搖搖欲墜。另外匈牙利後天就要投票了，奧班執政十六年的政權可能要走入歷史，這場選舉的結果會直接影響歐盟的外交跟國防政策。太空方面，Artemis II 太空人今晚就要濺落太平洋了，人類五十四年來第一次從月球軌道回到地球，這是一個歷史性的時刻。科技圈呢，美伊停火推動加密貨幣暴漲，但也有人潑冷水；另外 Gary Marcus 寫了一篇很有分量的文章批評 Anthropic 的 Mythos 過度渲染，GLM-5.1 的開源更是讓整個 AI 社群震動。我們一個一個來聊。

先從中東說起。兩天前，也就是四月八號，在巴基斯坦的斡旋下，美國跟伊朗終於達成了一個兩週的停火協議。你可能會想，這不是好消息嗎？是的，但問題是停火墨水都還沒乾，事情就已經開始不對勁了。這是一個充滿裂縫的停火。

我給你講三個主要的問題。第一，荷姆茲海峽。Trump 說停火的前提是伊朗「完全、立即、安全地」重新開放荷姆茲海峽。但到今天為止，海峽還是沒有真正重新開放。更讓人擔心的是，有報導說水面上發現了疑似水雷。你要知道，全球大約百分之二十的石油供應要經過荷姆茲海峽，海峽不開，這個停火就等於打了折扣。

第二個問題，也是最大的問題——以色列。Netanyahu 非常明確地說：這個停火「不包含黎巴嫩」。你沒聽錯，以色列的立場是，美伊停火歸美伊停火，我打黎巴嫩是我自己的事。然後在停火生效的第一天，四月九號，以色列就對黎巴嫩發動了他們稱為「永恆黑暗行動」的大規模空襲。十分鐘之內打了超過一百個目標，涵蓋南黎巴嫩、貝魯特跟貝卡谷地的真主黨指揮控制中心。黎巴嫩衛生部統計那一天至少三百零三人死亡、一千一百五十人受傷。這是自從以色列三月二號開始對黎巴嫩動手以來，單日最慘重的傷亡。黎巴嫩宣布了全國哀悼日。

第三個問題是，伊朗跟以色列對停火範圍的理解根本不一樣。伊朗的十項停火條款裡面明確要求停火必須涵蓋黎巴嫩跟所有地區衝突，結果以色列說不包含，然後在停火第一天發動最大規模轟炸。伊朗當然立刻跳出來，說美國必須在「戰爭跟停火之間做選擇」，不能讓以色列成為例外。

你看，這就是為什麼我說這個停火搖搖欲墜。Al Jazeera 的分析很直接——以色列這波轟炸就是要製造「停火不含黎巴嫩」的既成事實。如果這個定義站得住腳，伊朗就有理由退出整個停火協議。而 CBS 的報導標題更直白，說黎巴嫩已經成為美伊停火的最大引爆點。

不過在外交方面也不是完全沒有進展。巴基斯坦總理 Sharif 邀請美伊雙方代表團今天到伊斯蘭堡繼續談判，各國領導人也紛紛讚揚巴基斯坦在這次斡旋中的關鍵角色。Netanyahu 也說他願意跟黎巴嫩展開直接談判，討論解除真主黨武裝跟建立和平關係。但真主黨在被炸了三百多人之後，會不會坐下來談，我覺得你心裡也有數。

另一個值得注意的是，數千名伊朗民眾參加了悼念已故最高領袖哈梅內伊的活動。哈梅內伊在戰爭初期的美以聯合打擊中喪生，他統治伊朗將近四十年。這場悼念進一步凝聚了伊朗國內強烈的反美情緒，對接下來的和談絕對不是好兆頭。

在經濟影響方面，IMF、世界銀行跟世界糧食計畫署聯合發出了一份很嚴肅的警告。他們說中東衝突已經觸發了重大的能源衝擊，這個衝擊正在向全球糧食供應鏈傳導。特別是那些依賴進口的低收入國家，糧食安全問題會越來越嚴重。WTI 原油現在是九十八塊四七，雖然停火了但因為荷姆茲沒開，油價反而還在漲。

好，接下來我們看看歐洲。匈牙利後天，四月十二號，就要舉行國會大選了。這場選舉為什麼重要？因為奧班已經執政十六年了，這是他面臨的最大挑戰。反對黨 Tisza 在幾乎所有獨立民調中都大幅領先。最新的綜合民調顯示 Tisza 百分之四十九，Fidesz 百分之四十點七。但如果你看獨立機構 Median 的數據，差距更大，Tisza 可能拿到三分之二的國會多數，也就是一百九十九席裡面的一百二十九席。

但有趣的是，民調之間的差異大到不可思議。親政府的 Nézőpont 民調居然還是顯示 Fidesz 領先六個百分點。一位政治學者說，政府跟非政府民調之間的巨大差距，在匈牙利政治史上前所未見。EUobserver 的標題寫得很好，叫做「壓倒性的 Tisza 勝利——還是民調機構的滑鐵盧？」。華盛頓郵報寫了一篇評論，認為奧班的困境可能是 Trump 跟 MAGA 運動的預兆。不管結果如何，後天就見分曉了。

俄烏方面，Putin 宣布了東正教復活節週末停火，但你也知道，說停火跟實際停火是兩回事。同一天，烏克蘭就記錄了一百六十四次戰鬥接觸、七十五次空襲，俄軍夜間還發射了一百一十九架無人機，其中近七十架是 Shahed 型。一個讓人擔心的數字——俄羅斯計畫把無人機部隊從現在的十萬一千人擴編百分之六十，到十六萬五千五百人。這意味著無人機戰爭的規模還會繼續擴大。烏克蘭這邊也沒閒著，用無人機攻擊了俄羅斯 Krasnodar Krai 的一座油泵站，引發大火。

東亞這邊，停火消息對市場的影響非常明顯。四月八號亞洲開盤後全面暴漲。日經漲了百分之五點三九，這個漲幅是非常驚人的。台灣加權指數漲了百分之四點六一，一天之內從三萬三跳到三萬四。恆生漲了百分之三點零九，上證漲了百分之二點六九。這些漲幅基本上反映了市場對全球風險溢價下降的預期。但分析師警告，市場可能對這個脆弱的停火過度樂觀了。

台灣方面，國民黨主席鄭麗文訪中已經進入第四天。她前天在南京拜謁了中山陵，這個動作充滿歷史象徵意義。Foreign Policy 有一篇非常好的分析文章，說習近平選擇在 Trump 下個月訪中之前跟鄭麗文會面，是在發出一個穩定的信號，同時也是透過國民黨這個管道，預先佈局兩岸跟美中議題。民進黨批評鄭麗文的行程是「朝聖式」的，損害台灣利益。賴清德繼續重申願意對話但捍衛自主決策權的立場。

然後要跟你聊一件讓人很振奮的事情。Artemis II，人類五十四年來第一次繞月任務，今晚就要完成最後一步了。四名太空人——三位美國人 Reid Wiseman、Victor Glover、Christina Koch，加上加拿大太空人 Jeremy Hansen——將在美東晚上八點零七分，以接近兩萬四千英里的時速重返大氣層，濺落在聖地牙哥外海。隔熱罩跟十一具降落傘會負責保護他們安全著水。他們在這次任務中創下了人類最遠飛行紀錄，飛了二十四萬八千六百五十五英里，超越了一九七零年 Apollo 13 的紀錄。回收團隊會在 USS John P. Murtha 上接應太空人。NASA 會從美東下午六點半開始直播。如果你有空的話，這真的是一個值得看的歷史時刻。自從一九七二年 Apollo 17 以來，人類再也沒有從月球軌道回到地球過。今晚就是。

好，聊完國際局勢跟太空，我們來看看科技圈今天有什麼動靜。

首先最大的消息當然是美伊停火對加密市場的衝擊。停火消息一出來，比特幣直接飆到三週高點，接近七萬三千美元，漲幅超過百分之四點六。以太坊、Solana 也都跟著大漲。整個加密市場二十四小時內增加了大約一千億美元的市值。這個數字聽起來很厲害，但我要提醒你兩件事。第一，Fear and Greed Index 現在還是在十一，也就是「極度恐懼」的等級。第二，停火到現在荷姆茲都還沒真的重開，以色列又在轟炸黎巴嫩，這個停火能撐多久真的很難說。所以衝高之後比特幣已經有點回調，目前大概在七萬一左右。

另一個有趣的數據是 CME 的比特幣期貨活動量跌到了十四個月新低。主要原因是基差交易在大規模解套，機構投資人在地緣風險面前選擇撤離套利倉位。諷刺的是，CME 兩週前才剛宣布五月底要把所有加密期貨改成二十四小時七天交易。機構在撤，交易所卻在擴張，這個矛盾很值得玩味。

Tornado Cash 開發者 Roman Storm 的案子也有新進展。法官對檢方的指控範圍表示「太多了」，這個案子被整個加密社群視為開源開發者法律責任的風向標。如果 Storm 最終被定罪，以後寫智能合約的開發者可能都要擔心法律風險了。

科技模型方面有兩個重要的進展。第一個是 Gary Marcus 寫了一篇很犀利的文章，標題就叫「三個理由認為 Anthropic 的 Mythos 公告過度渲染」。他的論點很有力。首先，Anthropic 做測試的時候把沙盒關掉了，這等於是讓模型在一個比真實環境寬鬆得多的條件下發揮。其次，Marcus 的團隊實測發現，用開源的小模型在簡化環境中就能復現 Mythos 多數的漏洞分析能力，所以 Mythos 的能力可能沒有看起來那麼獨特。第三，如果你把基準分數正規化來看，Mythos 其實只比 GPT-5.4 略高一點點，並沒有呈現什麼指數級的突破。Anthropic 目前還沒有正式回應這篇文章。不過公平地說，參與 Project Glasswing 的十家公司，包括 AWS、Apple、Microsoft、Google、NVIDIA 這些大廠，他們確實認為 Mythos 在網路安全方面有實質突破。所以到底是 Marcus 對還是這些公司對，可能需要更多時間來驗證。

第二個重要進展是智譜 AI 的 GLM-5.1。這個模型的完整技術規格終於公開了。七百四十四億參數的 Mixture of Experts 架構，每個 token 使用四百億活躍參數，二十萬 token 的上下文窗口，十三萬一千 token 的最大輸出長度。最關鍵的是它用 MIT 授權完全開源，這是最寬鬆的開源授權之一，你可以任意商用。在 SWE-Bench Pro 這個編程基準上，GLM-5.1 拿了百分之五十八點四，超過 Claude Opus 4.6 的百分之五十七點三。這是開源模型第一次在主流編程基準上登頂。而且整個模型全程是在華為的昇騰晶片上訓練的，完全不依賴 NVIDIA。智譜今年一月在香港上市，成為全球第一家上市的基礎模型公司，市值大約三百一十三億美元。

AI Agent 方面，Microsoft 發布了 Agent Framework 1.0。這跟上週他們發布的 Agent Governance Toolkit 是不同的產品。Framework 1.0 提供穩定的 API 跟長期支援承諾，同時兼容 MCP 跟 A2A 兩大協定。這是第一個同時支持這兩個協定的企業級框架，對想要做多代理協作的開發者來說是個好消息。另外 EY 也宣布在他們的審計平台嵌入多代理 AI 框架，服務大約十三萬名審計員，目標是二零二八年所有審計活動都由 AI Agent 輔助。這算是四大裡面最積極的 Agent 部署了。

網路安全方面要提醒大家注意。Chrome 又修了一個零日漏洞，CVE-2026-5281，這已經是今年第四個了。這次是 WebGPU 實作層 Dawn 裡面的 use-after-free 漏洞，已經被實際利用了。如果你還沒更新 Chrome，趕快更新。Docker Engine 也披露了一個 CVSS 八點八的高危漏洞 CVE-2026-34040，可以繞過授權外掛。如果你有用 Docker 的授權外掛，也要注意更新。

最後來看看市場數據。今天是星期五，全球股市多數有交易。美股昨天繼續漲，S&P 500 收在六千八百二十四，漲百分之零點六二。Nasdaq 漲百分之零點八三。亞股四月八號的數據非常亮眼，日經五萬六千三百零八，台股三萬四千七百六十一，這些都是停火消息帶動的。油價方面比較矛盾，雖然停火了但荷姆茲沒開，WTI 原油九十八塊四七，Brent 九十六塊六三，反而還在漲。黃金四千七百八十五，避險需求略降但還在歷史高位附近。加密貨幣方面，BTC 七萬一千一百二十三，衝高之後有點回調。
