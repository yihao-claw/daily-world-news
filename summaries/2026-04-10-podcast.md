# EP40 每日新聞 Podcast — 2026年4月10日

嘿，早安。今天是四月十號，週四，歡迎回到每日新聞 Podcast 第四十集。今天最重要的兩件事——美國跟伊朗的和談今天在巴基斯坦正式登場了，但停火還沒滿四十八小時就已經出現嚴重裂痕；然後 Artemis II 的四位太空人今晚就要回到地球了，人類五十二年來第一次繞月飛行即將畫下句點。

好，直接進入時事。

先從中東說起。我們前兩天一直在追的美伊停火，有了重大進展——但也有重大危機。巴基斯坦總理 Sharif 居中斡旋，美伊兩邊在四月八號達成了兩週停火協議。這件事本身就充滿戲劇性。你想想看，巴基斯坦在這場衝突裡面原本不是主要角色，但他們成功搶到了和平談判的主持權，某種程度上把 Trump 的風頭都搶走了。Al Jazeera 有一篇很好的分析文章在講巴基斯坦是怎麼做到的，核心就是利用了跟伊朗的地理鄰近性和宗教連結。

但問題是，停火墨水根本還沒乾，裂痕就出現了。伊朗議長 Ghalibaf 在停火宣布後不到二十四小時，就公開指控美國已經違反了伊朗提出的十點方案裡面的三個條款。具體違反了哪三條，目前細節還不完全清楚，但這個訊號非常不妙。CNN 的分析用了一個很好的標題叫做「搖搖欲墜的停火為險峻的和談蒙上陰影」。

今天在伊斯蘭堡的談判陣容值得注意。美方這次是由副總統 JD Vance 領隊，Trump 的女婿 Kushner 跟中東特使 Witkoff 也都到場了——這個規格很高，說明美方是認真的。伊朗這邊則由議長 Ghalibaf 率團，外長 Araghchi 一起參加。但雙方在一個最根本的問題上就有分歧，就是這個停火到底涵蓋多大的範圍。巴基斯坦的 Sharif 很明確地說，停火涵蓋所有戰線，包括黎巴嫩。但 Trump 說黎巴嫩是「另外一回事」。以色列也明確拒絕把黎巴嫩納入停火範圍。

這直接帶到了昨天最令人震驚的消息——就在停火宣布後短短幾個小時，以色列對黎巴嫩發動了開戰以來最大規模的空襲。以色列國防軍把這次行動命名為「永恆黑暗行動」，在十分鐘之內精準打擊了一百個他們所稱的真主黨目標。結果是至少兩百五十四人死亡，超過一千人受傷——這是黎以戰爭開打以來最血腥的單一日。黎巴嫩政府立即宣布了全國哀悼日。

更讓人不安的是，這些空襲不只是打偏遠的軍事目標。貝魯特市中心的密集住宅區和商業區都遭到了無預警的轟炸。Democracy Now 報導的標題是「十分鐘，一百次空襲」，你可以想像那個衝擊力。國際社會的反應是一面倒的譴責——Al Jazeera 用了「殘暴」這個詞來描述各國的反應。

但這裡有一個值得注意的轉折。即使是 Trump 本人，都出面要求 Netanyahu 縮減對黎巴嫩的攻擊規模，並且敦促以色列進入跟黎巴嫩的直接談判。Netanyahu 的回應是——指示他的內閣「儘速展開與黎巴嫩的對話」，焦點會放在真主黨的解除武裝和建立和平關係上。所以未來幾天你可能會看到兩條平行的談判線：一條是美伊在巴基斯坦的大框架和平談判，另一條是以色列跟黎巴嫩可能開始的直接對話。這兩條線能不能匯流，我覺得是接下來中東局勢最關鍵的問題。

同時，伊朗外長 Araghchi 也跟俄羅斯外長 Lavrov 通了電話，強調美國有責任確保停火延伸到所有地區，包括黎巴嫩。俄羅斯在這裡的角色很微妙——他們既是伊朗的盟友，也在自己的戰場上（烏克蘭）消耗中。

說到烏克蘭。俄羅斯所謂的春季攻勢，進展可以說是非常有限。過去四週大約只推進了十七平方英里。ISW，就是戰爭研究所，他們的評估很直接——俄軍今年不太可能突破烏克蘭建立的「堡壘帶」防線。反而是烏克蘭那邊傳來了久違的好消息。二月份成為了二零二四年以來第一個烏克蘭月度淨奪回領土的月份，南線反攻已經收復了四百平方公里。Zelensky 說前線狀況是十個月以來最好的。當然，我們要謹慎看待這些數字——戰場上局部的推進和後撤是常態，但整體趨勢確實在向有利於烏克蘭的方向傾斜。

歐洲還有一件大事——匈牙利的國會選舉後天，四月十二號，就要投票了。Orbán 執政十六年以來面臨最嚴峻的挑戰。反對派 Tisza 黨——由前 Fidesz 圈內人 Peter Magyar 創立——在最新的 PolitPro 民調趨勢中以四十九比四十一領先執政聯盟。Median 的預測甚至認為 Tisza 可能拿到三分之二超級多數，就是一百三十八到一百四十二席。如果這成真，那不只是政權更替，而是可以修憲的壓倒性多數。

但這裡有個很重要的但書。匈牙利的選舉制度對執政黨有結構性的優勢——選區劃分、媒體控制、國家資源的動員，這些都是 Fidesz 經營了十六年的護城河。而且不同民調機構的結果差異很大，有些甚至還顯示 Fidesz 領先。所以後天的結果到底是反對派的歷史性勝利，還是民調界的集體翻車，我們很快就會知道了。EUobserver 的標題下得很好——「七十二小時後，是 Tisza 的壓倒性勝利，還是民調的崩潰？」

東亞方面有幾件事值得一提。

首先是日本。日銀四月升息的市場預期正在快速升溫。一位前 BOJ 理事公開表示本月可能加息。背後的邏輯很清楚——中東衝突推高油價，日圓持續走弱讓輸入性通膨壓力不斷加大。IMF 最新的第四條款磋商報告把日本二零二六年 GDP 增速從百分之一點三下修到零點八，但同時卻敦促 BOJ 繼續漸進升息。這有點矛盾，但 IMF 的邏輯是，通膨風險比成長放緩的風險更迫切。BOJ 四月二十八號的利率決議會議將是觀察重點。不過從 BOJ 最新發布的地區經濟報告來看，措辭非常謹慎，九大地區都只用了「溫和復甦」這類模糊表述。他們顯然不想在會議前三週就把市場預期推太高。

台灣方面，國民黨主席鄭麗文的「和平之旅」進入第四天。她四月七號抵達上海，八號去了南京的中山陵，九號飛到北京，習鄭會預計就在今明兩天舉行。如果成功，這會是十年來第一次國共最高領導層的正式會面。Foreign Policy 有兩篇很有深度的分析。第一篇的觀點是北京在 Trump 下個月訪中之前，刻意安排這場會面，向華府傳遞一個訊號——「我們有能力透過民間和反對派力量來穩定台海，不需要你來插手」。第二篇則指出，這場會面代表北京仍然把台灣問題視為「可以解決的爭端」，而不是不可避免的衝突。賴清德政府的回應是一貫的「對話與自主並行」，但台灣島內的輿論對這趟訪問看法嚴重分歧——有人認為是為和平鋪路，也有人認為這就是統戰的一部分。

南亞那邊，印度阿薩姆邦的議會選舉昨天完成投票了，投票率高達百分之八十四點四二，比上一屆還高。BJP 的執政聯盟在爭取第三任期，國大黨力圖翻盤。不過計票要到五月四號才會公布，所以目前只能等。同一天，喀拉拉邦跟本地治里也完成了投票。

好，時事先到這裡。接下來看科技。

今天科技圈最大的新聞是 Microsoft 正式發布了 Agent Framework 一點零。如果你有在關注 AI Agent 這個領域，你就會知道 Microsoft 之前一直有兩套各自發展的框架——Semantic Kernel 是比較企業級的，AutoGen 則偏研究導向。現在他們把這兩個合併成一個統一的、生產就緒的開源 SDK。這不只是把程式碼搬在一起而已，它內建了 MCP 也就是 Model Context Protocol 的工具發現功能，還支援 A2A 一點零的跨框架代理協作。同一時間，Linux Foundation 底下的 Agentic AI Foundation 也正式成為 MCP 加 A2A 的治理機構。

這代表什麼？代表多代理架構正在從群雄割據的階段，進入標準化的時代了。就像 HTTP 之於 Web、REST 之於 API 一樣，MCP 加 A2A 很可能成為 AI Agent 世界的基礎通訊協議。如果你是做 Agent 開發的，這個框架值得認真評估。

另一個有趣的更新是 Ollama 出了 v0.20.5。這次有一個特別的功能——你可以用 ollama launch openclaw 一個指令就連上 WhatsApp、Telegram、Discord 這些訊息平台。等於說你跑一個本地模型，就能直接變成一個聊天機器人。另外 Gemma 4 的 Flash Attention 支援也改善了，工具呼叫修復了一些問題。

大模型方面，DeepSeek 發布了 R2 推理模型——在 AIME 二零二五拿到了超過九十二分、MATH 五百拿到了接近九十分，基本上已經逼近 OpenAI o3 的水準。開源陣營在推理能力上的追趕速度真的很快。Mistral 也發了 Large 3，主打結構化輸出的改善和歐盟資料落地，瞄準那些對 GDPR 資料本地化有嚴格要求的歐洲企業客戶。不過我要提醒一下，這兩個模型目前都還缺乏獨立第三方驗證的安全評估報告，在實際生產環境部署之前，幻覺率跟對抗性輸入的韌性都還需要進一步驗證。

HuggingFace 也出了 Transformers v5.5.3 的熱修版本，專門修了 Gemma 4 在多 GPU 環境下 device map 的分配問題。如果你在跑 Gemma 4 的分散式推理，記得更新。

Agent 生態系統方面，OpenClaw 出了二零二六點四點九版，這次的改動主要在記憶系統——新增了所謂的 grounded REM backfill 功能，讓歷史日記可以回放到長期記憶裡面，不需要額外的記憶堆疊。然後 EY，就是四大會計師事務所之一的安永，他們把一個多代理 AI 框架直接嵌入了 EY Canvas 這個審計平台，目標是二零二八年實現百分之百代理輔助審計。這是四大裡面最激進的 Agent 落地案例。

資安要注意幾件事。Docker Engine 被發現了一個 CVSS 八點八的高危漏洞，攻擊者可以繞過授權外掛。這個漏洞更麻煩的是，它是之前一個二零二四年漏洞的不完整修復。Chrome 也修了今年的第四個零日漏洞，Google 緊急推出了 Chrome 一四六的帶外更新。然後 React2Shell 這個漏洞被大規模利用，已經有七百六十六個 Next.js 主機的各種憑證被竊取——包括 AWS secrets、SSH 金鑰、Stripe API key、GitHub token，基本上能偷的都偷了。如果你有在用 Next.js 跑生產環境，拜託趕快去查一下。

最後看市場。

美股昨天收盤繼續上漲——S&P 五百漲了百分之零點六二，收在六千八百二十四點；Nasdaq 漲了百分之零點八三；道瓊漲了百分之零點五八。停火帶來的風險偏好改善還在持續。

不過油價因為停火出現裂痕，昨天又再度攀升了。WTI 原油漲了超過四個百分點，來到九十八塊半。Brent 也漲了快兩個百分點。中東地緣政治的不確定性，尤其是以色列在黎巴嫩的軍事行動，讓市場對能源供應的擔憂又升溫了。黃金小漲零點七，收在四千七百八十二——在不確定性這麼高的時期，黃金表現其實有點溫吞。

匯率方面，今天最新報價——美元兌日圓一百五十九點一一，歐元兌美元一點一七，美元兌台幣三十一點七三，基本持平。

加密貨幣，比特幣昨天收在七萬一千八百，漲了不到一個百分點。但恐懼貪婪指數只有十一——這是極度恐懼的水準。以太幣兩千一百八十九，幾乎沒動。雖然停火利好讓比特幣一度衝上七萬三千的三週高點，但 Circle 跟 Bullish 遭到分析師降評大跌，市場信心不足。整體來說，加密市場還在消化前一波的漲幅，多頭動能不夠。

歐洲跟亞洲股市的數據因為超過二十四小時未更新，今天就不報了。

好，這就是今天的新聞。最後快速回顧一下重點：美伊和談今天在伊斯蘭堡登場，但停火已經出現裂痕，關鍵分歧是黎巴嫩；以色列的「永恆黑暗行動」造成兩百五十四人死亡，是開戰以來最血腥的一天；匈牙利後天投票，可能迎來十六年來首次政權更替；Artemis II 的太空人今晚回家。科技方面，Microsoft Agent Framework 一點零標誌著多代理標準化時代的到來，DeepSeek R2 在推理上逼近了 OpenAI o3。我們明天見。
