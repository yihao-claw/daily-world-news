各位晚安，歡迎收聽每日世界新聞，我是你的夜間主持人。今天是 2026 年 4 月 17 日，星期五，這是第四十四集。今天最值得你停下來想一下的，是兩件同時發生、但方向完全相反的事。一個是川普在華府宣布以色列和黎巴嫩談成十天停火，三十三年來兩國大使第一次坐在同一個房間裡。另一個，是俄軍在烏克蘭夜裡丟出將近七百架無人機和近二十枚彈道飛彈，把基輔、Odesa、Dnipro 炸得像聖誕節的火焰樹，一個十二歲的孩子死在家裡。這兩件事都是在同一個二十四小時內發生的，你就會知道，地緣政治從來不會真正給你一個安寧的夜晚。

我們今天就從華府那場會談開始說起。

川普四月十六日在白宮宣布，以色列和黎巴嫩談成十天的停火，生效時間是美國東岸時間四月十八號下午五點。這表面上聽起來像一個外交突破，但你如果把耳朵貼近一點，會聽到很不一樣的旋律。停火條件寫得很有意思：以色列在「正當自衛」的前提下，還是可以隨時打進黎巴嫩。換句話說，這個十天停火是有開關的，而那個開關是握在特拉維夫手上。

黎巴嫩總理在 X 上發文，說「這是我們從開戰第一天就在追求的訴求」。聽起來是慶祝，但你要記得，真主黨同一天出來公開呼籲黎巴嫩政府退出談判，說這是「無效的努力」。所以這個十天停火實際上撕開了黎巴嫩內部：一邊是政府，希望透過華府拉開一段距離，讓自己脫離德黑蘭的影響圈；另一邊是真主黨，覺得政府出賣了他們。這個裂痕本身就比停火還要重要。

美國國務院則把這場停火包裝成「美以黎戰略再定位」的開端，強調這是把黎巴嫩從伊朗的軌道上拉出來的歷史機會。你可能會想，十天能做什麼？答案是——給沙烏地、埃及、土耳其三個中東大國同步介入的時間窗。

接下來我們看另一個戰場。就在美國把外交馬力都投在黎巴嫩和伊朗身上的時候，俄軍在烏克蘭上空掀起了今年最兇的一波夜襲。幾乎七百架無人機、十九枚彈道飛彈、還有巡弋飛彈。基輔四個人死亡，其中一個是十二歲的男孩。他躺在自家公寓裡，被一架無人機直接撞穿西側外牆。Odesa 死了九個人，Dnipro 三位女性，Zaporizhzhia 一位。基輔市內被打中的，有十七棟公寓、十棟住宅、一間旅館、一個商場、車行還有加油站。澤倫斯基說，這是「今年最黑暗的一夜」。

更值得你注意的是，烏軍擊落了大約六百三十六架無人機，和「一部分」飛彈。注意「一部分」這個字。這句話的言外之意，是防空飛彈快不夠用了。為什麼？因為美國現在把 Patriot 防空飛彈的出口優先級，從烏克蘭挪到了波斯灣。Iran 戰爭把全球防空武器的庫存結構整個翻過來。烏克蘭防空網的每一次「只擊落一部分」，背後都是華府那張優先級清單的順序被重寫。

然後我們談中國。同一天，北京公布第一季 GDP 年增百分之五，比 Reuters 分析師預估的四點八還高了一截，季增百分之一點三。中國把這個數字解讀成「面對川普關稅還能挺住的證明」。但你要看的是背後的劇本——川普在四月十三號威脅對中國加徵百分之五十關稅，說中方正在幫伊朗運武。如果這刀子真的落下，中國整體對美關稅會重新逼近去年五月前的八成水準。北京到現在沒有正式回應，只是把 GDP 數字擺出來，就像在說「我們撐得住」。

同一天還有另一個畫面：IEA 總幹事出來告訴 AP，說歐洲的 jet fuel 庫存「大概只剩六週」。歐洲百分之七十五的航空燃油都是從中東進口的，現在荷莫茲封鎖不解除，SAS 已經宣布四月取消一千班飛機、Ryanair 警告如果危機再拖，整個夏季都要減班。德國四月十三號已經對汽油柴油減稅兩個月來救通膨。這代表什麼？代表中國、美國、歐洲，三個全球最大經濟體，現在同時被一條狹窄的海峽牽著走。五百年前的海權論，在二零二六年用航空燃油的方式重新上演。

我們再把視角拉到印度。莫迪政府這禮拜啟動三天的國會特別會期，四月十六號到十八號。核心議案有兩個：擴大下議院 Lok Sabha 的席次，還有對中東危機的能源補貼預算。莫迪跟馬克宏通了電話，兩個人講的重點是「緊急恢復荷莫茲航行自由」。世界銀行預估印度 FY27 成長百分之六點六，仍然是全球最快的主要經濟體之一。但這個故事的重點不是數字，是地緣。因為印度是全球少數，可以同時跟伊朗、沙烏地、美國、俄羅斯保持正常溝通的國家，這個斡旋空間比 GDP 更稀有。

然後是台灣。今天台股終於突破三萬七千點。盤終收三萬七千一百三十二點零二，漲四百零九點八八，連八紅，四月漲了大約百分之十六。台灣股市的總市值已經超越英國，躍居全球第七大。權值股台積電收在新台幣二千零八十五元，單日只漲了百分之零點二四，但盤中再創多項新高。驅動這波上漲的是兩件事：第一，美伊第二輪談判的樂觀情緒；第二，台積電公布第一季獲利年增百分之五十八，並上修全年營收預估到超過百分之三十。

接下來我們換個氣口，聊聊非洲，因為這個大陸今天同時出了兩個故事。第一個是蘇丹。聯合國救援總幹事說「世界正在辜負蘇丹」——戰爭進入第四年，三千四百萬人、全國百分之六十五需要人道援助。去年十月二十七號 RSF 拿下 El Fasher 之後，聯合國估計三天內有六千個平民被殺，被定性為「具種族滅絕特徵」的暴行。更糟的是，二零二六年聯合國蘇丹人道援助計畫二十八億美元，到現在只到位百分之十七。達爾富爾和 Kordofans 地區進入饑荒確認階段，嚴重急性營養不良人數要從五十五萬升到八十萬。這不是一場大家眼中的「大戰爭」，但它是這個星球上最大的人道危機。

第二個非洲故事，是教宗 Leo XIV 的非洲行。這位史上第一位美籍教宗，這幾天在喀麥隆、阿爾及利亞、安哥拉、赤道幾內亞巡迴。他在 Bamenda 主教堂說了一句很狠的話：「戰爭大師們假裝不知道——毀滅只要一瞬間，重建卻要一生。」這話是直球衝川普。川普四月十四號在 Truth Social 罵教宗「對犯罪軟弱、對外交政策軟弱」，還貼了一張 AI 生成、把自己變成耶穌的圖片，後來刪了。教宗今天回應，只說「世界需要聽這段和平的訊息」。你會發現，教宗跟白宮之間，用語言和圖像打了一場二零二六年最有宗教感的政治戰。這場戰的雙方，一邊是宗教象徵，一邊是世俗權力，在 AI 生成圖像這個新媒介上正面交鋒，這件事本身就是我們這個時代獨有的場景。第一位美籍教宗跟川普之間這種公開對槓，是教廷史上罕見的畫面。

大洋洲今天也不平靜。紐西蘭北地發了紅色暴雨警報，三十六小時內累積雨量可能到三百二十毫米，Whangārei 一帶的居民被建議往高處撤。巴布亞新幾內亞中部 Bougainville 的 Asiko 村山崩，八個人死亡。這些都是很小的數字，但在一個台灣人生活的島嶼上，暴雨和山崩從來不是小事。

好，時事就先到這邊。我們休息一下，接著進入你今天關心的另一個世界——科技的二零二六年四月十六號，其實跟上面這些國際新聞一樣熱鬧。你會發現，今天最重要的科技事件，幾乎都在同一天發生。

我要先講的，是 Anthropic 把 Claude Opus 四點七正式放出來的事。這是他們自己的旗艦模型，直接跳過四點六的延伸，進到一個新的版本號。SWE-bench Verified 跑到八十七點六，比前代的八十點八高了將近七個百分點，也壓過 Gemini 三點一 Pro 的八十點六。更狠的是 SWE-bench Pro 這個更難的基準，Opus 四點七拿下六十四點三，GPT 五點四只有五十七點七。CursorBench 從 Opus 四點六的五十八，一躍到七十。

新增的功能裡面，最值得一提的是兩個：第一個叫「xhigh 推理檔位」，這是比現有 high 還要更高的推理深度，代表 Anthropic 開始把推理時間當成一個滑桿，讓你決定要花多少 token 去想同一個問題。第二個是 agentic loop 的「任務預算」——你可以明確告訴模型「這個 agent 最多花十個 step、三萬 token」，避免它跑飛。定價是一美元輸入五塊、一百萬 token 輸出二十五塊，已經上線在 AWS Bedrock、GCP Vertex AI、還有 Microsoft Foundry。

但我要給你留個反方視角。Simon Willison 是英國開源社群的老戰友，他今天 blog 做了一個「鵜鶘騎腳踏車」的視覺基準實測——把 Anthropic 剛發的 Opus 四點七，跟 Alibaba 今天剛開源的 Qwen 三點六三十五 B A3B 放在一起比。Qwen 三點六是一個本地可以跑在 MacBook Pro M5 上的量化模型，大小才二十點九個 GB。結果出來，Willison 說 Qwen 畫的鵜鶘「驚人地接近、甚至超過」Opus 四點七。這個實驗本身不是什麼科學基準，但它的符號意義很強——你可以用一台筆電跑贏旗艦雲端。

講到 Qwen，Alibaba 今天同步把 Qwen 三點六三十五 B A3B 開源。這是一個 MoE 混合專家模型，總參數三百五十億、active 三十億，Apache 2.0 授權，context 二十六萬二千 token，支援顯式思維鏈。在 Terminal-Bench 二點零 agentic coding 基準上，它拿到五十一點五分，對手 Gemma 四三十一 B 四十二點九，高了快九分。coding 能力排名第十五。你要注意一件事——今天開源中文模型的性價比曲線又被推高一次，Anthropic 跟 Alibaba 同一天發大招，不是巧合。

另一邊，Ollama 今早 release 了 v0.21.0 的第一個 RC，從 0.20.8 直接跳版。這次更新主要是 MLX backend 修 gemma4 cache 的邏輯視圖，Apple Silicon 跑 MoE 模型的顯存佔用會明顯下降。這是給那些拿 MacBook 跑 Qwen 和 Gemma 的人一個小小的福音。

然後 LangChain 今天出了 core 1.3.0 的第三個 alpha。重點是 chat model 跟 LLM invocation 參數終於進入 traceable metadata，意味著你在 LangSmith 上追蹤 agent 的每一步，終於可以看到模型呼叫的超參。同一天搭配 OpenAI、HuggingFace、text-splitters 三個子套件全部做 SSRF-safe transport 升級——提醒你，SSRF 是去年底整個 agent 生態圈踩過的坑，大家在補。

有一個很「內部」的發布你可能會關心：OpenClaw 今天發了二零二六點四點十五 beta 二。這個版本把整個框架的預設模型切到 Claude Opus 四點七，Anthropic selections、Claude CLI defaults、圖像理解全部一起升級。另外，Google plugin 第一次加入 Gemini TTS 語音合成支援，你可以直接在 OpenClaw 裡面用 Gemini 的聲音做語音輸出。

再來我們講加密貨幣。今天有三條線值得聽。

第一條是 Morgan Stanley 的 Bitcoin ETF。它的代號是 MSBT，四月八號上市，到今天四月十六號首週 inflows 突破一億美元，首日資金就有三千零六十萬。費用率零點一四，是整個現貨 BTC ETF 市場最低的一檔。Morgan Stanley 數位資產負責人 Amy Oldenburg 跟 Bloomberg 說，crypto 已經是他們的「每日業務常態」。這不再是十年前那句「Bitcoin is going to zero」了。這是華爾街真正的轉身。

但你要留個看空視角。費用率一檔比一檔低，BlackRock 的 IBIT 現在管了五百三十億美元，但費用殺到零點一四的時候，每一家 ETF 的毛利都在被蠶食。長期來看，如果基金公司賺不到錢，他們推動 BTC 基礎建設的意願就會下降。零費用這件事，對投資人是好事，但對整個機構 crypto 的資本重分配會有結構性影響。

第二條線是 CFTC 主席 Mike Selig 今天在眾議院農業委員會被質詢的戲劇性畫面。民主黨眾議員 Jim McGovern 當面質問他：白宮有沒有要求 CFTC 停止對 Polymarket 的調查？背景是 Donald Trump Jr. 是 Kalshi 的顧問、也是 Polymarket 的投資人。McGovern 還當著 C-SPAN 鏡頭講了一句：「這聞起來就是腐敗。」Selig 的回應是，他對預測市場的詐欺、操縱、內線交易「用盡法律力量追到底」，並且重申 CFTC 對預測市場有獨家管轄權——這是他同時對州政府劃下的戰線。

第三條線是 Charles Schwab 今天啟動現貨 BTC 跟 ETH 的交易平台，叫 Schwab Crypto。首階段只支援 BTC 和 ETH，未來幾週要鋪到它數千萬的零售客戶。這家老牌券商的入場，比 Morgan Stanley 還有象徵意義——因為 Schwab 代表美國中產階級的主要股票帳戶市場。

還有一條不能漏的：Bitcoin Core v31.0 final 今天正式 tag。昨天才出 RC4，今天就打 final，核心協議層完成年度大版本升級。這次的治理協議改動預期會在接下來三個月，被礦池和交易所逐步採納。

最後我們快速過一下幾個前沿。Andon Labs 今天公布一個長期實驗：給一個 AI agent 一份實體零售店面的三年租約、一筆本錢，讓它自主採購、定價、排班、行銷、報稅。測試在有物流、有現金流、有監理壓力的現實市場裡，agent 能不能維持正現金流。HN 上一百五十一分、兩百一十八個留言。另外一個是 YC W26 冬季班發射的 Kampala，做 MITM 代理，把舊系統的 UI 操作流程「反向工程」成 API，讓你不用 RPA、不用 computer-use agent，也能做到工作流自動化。
