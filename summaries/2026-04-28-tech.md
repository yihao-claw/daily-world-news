## 🔥 科技頭條（Top 3）

• 🔥10 **微軟-OpenAI 重塑合作：AGI 條款正式失效** — 兩家公司今天宣布合作進入「新階段」，最大重點是過去 6 年那條怪規定「OpenAI 達成 AGI 後微軟 IP 權即時失效」被刪除。改寫後微軟對 OpenAI 模型 IP 的權利延長到 2032 年；AGI 與否改由獨立科學委員會認定，不再讓 OpenAI 自己宣告。等於拿掉了過去 Sam Altman 對微軟的最大籌碼。
  🔻 The Information / Sam Lessin：微軟賠了「定義 AGI」的權力，但換到更長 IP 護城河 + OpenAI IPO 優先股；OpenAI 則犧牲「自我宣告 AGI 即解約」的逃生口。Simon Willison 把 openai.com 6 年來這段文字的版本演進全列出，可看到字面的轉折時間軸。
  📎 來源：[OpenAI Blog](https://openai.com/index/next-phase-of-microsoft-partnership) | [Simon Willison 追蹤](https://simonwillison.net/2026/Apr/27/now-deceased-agi-clause/)

• 🔥10 **GitHub Copilot 全面改 Usage-Based Billing** — GitHub 宣布 6 月起把 Copilot 個人版與 Business 版改成按用量計費：月費仍在，但超出基線 token 配額後額外計費；企業可在 Azure 帳單集中結算。HN 留言區 290 則以上抨擊「實質漲價」，但 GitHub 強調 OSS 維護者與學生帳號免費額度不變。
  🔻 開發者社群普遍認為這是被 Anthropic Managed Agents（每 session-hour 0.08 美元 + tokens）與 Cursor 改用量定價倒逼的結果，不是「給用戶選擇」這麼好聽。
  📎 來源：[GitHub Blog](https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/)

• 🔥10 **Gemini 推 Agentic Trading：AI 直接下單** — Coinbase 旗下 Gemini 美國交易所今天上線「Agentic Trading」，把 GPT、Claude、Grok 模型直接接進客戶帳戶，agent 可下單、調倉、停損，號稱「第一個在受監管美國交易所內運作的代理人交易工具」。風控設定：每 24 小時最多 25% 資產可轉外部錢包，agent 不可跨帳戶移資金。
  🔻 鏈上分析師警告：合規外殼下，散戶把 API key 加上錢包授權同時交給第三方 agent，等於整個 wallet 暴露在 prompt injection 攻擊面。
  📎 來源：[The Block](https://www.theblock.co/post/399001/gemini-rolls-out-agentic-trading-allowing-ai-bots-to-directly-manage-crypto-exchange-trading-accounts)
