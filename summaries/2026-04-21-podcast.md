各位聽眾，今天是二〇二六年四月二十一號，星期二。我是你的主持人，這是每日世界新聞的第四十八集。

今天的節目只有兩個字要放在你耳邊：倒數。一個倒數在波斯灣——美伊為期兩週的停火，明天晚間就到期，川普親口說「延長極不可能」；另一個倒數在開發者社群——比特幣 Core 終於在停擺三年後跳到 v31.0，這代表什麼，我們後半段一起拆。接下來一路講過去的，是菲律賓前總統 Duterte 明天上午在海牙的命運、賴清德明天啟程飛非洲、匈牙利 Orban 時代正式閉幕、印度的女性保障名額硬生生被推遲到二〇二九年以後，還有北韓 Lazarus 一夜之間把 DeFi 拖到十二個月新低。我們進入第一段。

先從波斯灣講起。川普週一對外明說，兩週停火到週三晚間結束，再延長「可能性極低」。這句話的份量，你得放到幾個數字旁邊才看得出來——海運公司的紀錄顯示，過去兩週，通過荷莫茲海峽的船流量比戰前暴跌了九成，大概有每天一千萬桶的原油被卡在峽谷兩端進退不得。更具體一點：OPEC 的四月產量月減幅衝到百分之二十七，是自一九八〇年代以來最大的一次。沙國自家的東西向輸油管被伊朗擊中後，容量直接掉了七十萬桶每日，整個沙國從每天一千零一十萬桶掉到七百八十萬桶，少了將近百分之二十三。阿聯酋、伊拉克、科威特同步下挫。這些數字擺在一起，你就知道為什麼布蘭特原油已經收在九十五塊五、WTI 也挑戰八十七塊。

昨天週末的時候還發生了一件事——美國海軍在阿曼灣攔截、接收了一艘懸掛伊朗國旗的貨輪，理由是違反了華府新劃的封鎖區。德黑蘭外長斥責這是戰爭行為、誓言報復；德黑蘭外交部的 Baqaei 昨天下午還特別澄清「沒有要去巴基斯坦伊斯蘭瑪巴德談第二輪」。換句話說，外交管道基本上空了。

而這場戰爭的第二圈外溢，正好落在華府和北京身上。川普把話說得很直白：「只要有國家供武給伊朗，對美出口一律加百分之五十關稅、沒有豁免。」《華盛頓郵報》前幾天爆料，說中國有意向伊朗運防空系統，川普隔幾小時就把五十趴關稅威脅推到 X 上，還順便加了一句「China has agreed not to send weapons to Iran」。北京否認前提，但官媒這週連發三篇「必要反制」社論。外貿學者 Eswar Prasad 提醒我們一件事——對中再加五十趴這件事，早就寫進了川普二.〇 關稅體系裡算過的帳，真正先被打到的，不是中國，而是美國消費電子價格，選情往哪偏，還難說。

好，這是波斯灣的版圖。我們把鏡頭搬到海牙。

明天，海牙時間上午十一點，國際刑事法院的 Appeals Chamber 會宣判菲律賓前總統 Duterte 的管轄權上訴。這不是最終審，但是決定他會不會正式進入「戰犯審判」程序的一個大關卡。去年十月，Pre-Trial Chamber I 已經駁回過他的第一次管轄權挑戰——當時裁定，即使菲律賓二〇一九年才正式退出羅馬公約，但案件犯罪時間落在二〇一一到二〇一九年，法院依然有管轄權。辯方隨後上訴，十一月中遞交意見書，今天終於到裁決前夕。

Duterte 本人，昨天透過首席律師 Nicholas Kaufman 書面表態，放棄親自出席。信裡他說得很乾淨——「我把話交給律師替我說」。為什麼這樣？因為他被羈押在海牙看守所已經好幾個月了，避免給對手一個「認輸畫面」，也避免自己情緒失控。馬尼拉那頭，副總統 Sara Duterte 陣營正在發動「和平示威」；現任的小馬可仕政府則保持司法尊重的距離，但同時悄悄強化安全配合。美方國務院雖然還是把 ICC 列為非美利益機構，但魯比歐說會配合荷蘭押解。

這個案子的意義不在於 Duterte 本人，而在於 ICC 有沒有辦法用這個先例、把東南亞其他強人的作為納入管轄。去年這個時候，你我一起聊過，很多觀察家一度認為 ICC 在亞洲根本沒牙——那麼明天早上的判決，就是把這顆牙齒補上、還是再次被拔掉的分水嶺。

講完菲律賓，我想讓你跟我一起把地圖拉回東亞，看兩個同時進行的「領袖外交」。

一個是賴清德。明天四月二十二號，他會搭機前往史瓦帝尼，這是任內第一次出訪、也是把「非洲唯一邦交國」這個標籤演到最高潮的一趟。賴辦沒有選擇走中美洲過美國的舊路線，而是直飛南非，因為史瓦帝尼被南非包著，這條航線繞開中東空域、也不用經過關島、夏威夷，最大優勢是不會激怒北京。這次行程是慶祝國王 Mswati 三世登基四十週年和生日，賴清德會簽一份聯合公報、視察一個兩億新台幣的工業園區預定地。

同時間，另一邊——南韓的李在明，四月十九號就從首爾出發，今天抵達河內，明天要跟越南的蘇林坐下來談。韓國首爾這趟大動作，三星、LG、SK、現代四家巨頭 CEO 全體隨行，談題聚焦造船、AI、稀土、LNG。首爾總統辦公室昨天還順勢發了一份白皮書，正式把「印度—越南—澳洲」定調為南韓在 Hormuz 斷鏈後的「戰略替代三角」。

你看這兩條腿：台灣押的是象徵性外交、把價值觀、邦誼、島國地位這些軟籌碼往非洲倒；南韓押的是實質經濟整合，用供應鏈綁定。兩邊都在為同一件事準備——波斯灣斷鏈之後，東亞製造業要有備胎。一個偏向形象、一個偏向鏈條，兩條都很合理，但也都有自己的脆弱點：賴清德的非洲外交，被淡江大學戰略所的黃介正點出「同時把最高層級的政要送出國」在戰爭邊緣不夠穩；南韓的白皮書被 Nikkei 質疑「印度—越南在真實製造能量上，短期還填不滿美日兩個主回路」。

換個大陸。歐洲這週最大的懸念，是匈牙利的交接倒數。四月十二號，Magyar Péter 領的 Tisza 黨拿下一百三十八席、壓過 Orban 的 Fidesz 只剩五十五席——結束了長達十六年的 Orban 時代，而且是一次絕對多數、直接跨過修憲三分之二門檻。Magyar 的下一步很明確：第一、把 Sulyok 拉下元首寶座，因為 Sulyok 是 Orban 時代欽點的；第二、搶在八月之前把被 EU 凍結的一百億歐元救援金給解鎖——這筆錢的時間窗要關上了；第三、把 Orban 之前塞進最高法院、審計院、國家銀行的人，一個一個「自願離開或者我來請你走」。

Magyar 自己預估最快要等到五月十二號前，國會首次會期才能正式召開，那個時候他才會真的交棒上台。但 LSE 的 Zsolt Enyedi 提醒：Tisza 贏選舉是靠「不是 Orban」這單一主軸，黨內其實還沒長出制度化的派系和路線，未來十二個月內很可能爆發「權力真空式內戰」。換句話說，Orban 結束，不代表 Orban 主義結束——這點跟我們昨天在聊的時候立場一樣，但今天多了一個新訊號：斯洛伐克外長 Blanár 週一再度公開放話，繼續封鎖歐盟對俄第二十輪制裁，除非基輔書面保證修復 Druzhba 輸油管。Blanár 這次把話講得更滑溜——他強調斯洛伐克不反對九百億歐元援烏貸款，只是卡在能源。也就是說，Orban 倒了、Fico 和 Orban 的聯盟也裂了，但留下來的俄油依存問題，不會因為換了政府就自動解決。

快速帶一下南亞。印度的 131 修憲案，四月十七號那個晚上以 298 對 230 翻盤的事情，今天進入後續。這個法案原本要把女性保障名額直接綁定「以 2011 人口普查做 delimitation」，由南印和西孟加拉的議員聯手擋下。這件事在印度憲政史上，是 12 年來莫迪政府第一次修憲敗選。官方昨天承認，女性 33% 保障名額的實質實施時間，會被推遲到二〇二九年大選以後——也就是說，原本上週四剛通過的《106 號憲法修正案》雖然紙面上生效，但它的實施條件需要等下一次普查和選區重劃完成，在沒有 131 修憲案的情況下，那個條件仍然存在、但沒人可以跳過它。莫迪政府也同步宣布暫停提交《Delimitation Bill 2026》。Rahul Gandhi 在記者會上酸得很狠：「以女性之名，行違憲捷徑」。莫迪的反對陣營，這次是站穩了。

非洲方面，有兩條線值得你記一下。第一條是蘇丹戰爭邁入第四年。美國在週末公開呼籲薩赫勒軍和 RSF 雙方接受「三個月人道停火」。紅十字、聯合國兒童基金會、Intersos 的聯合報告最新數字：蘇丹戰爭三年來一千四百萬人流離失所、兩千六百萬人面臨嚴重糧食不安全、三千三百七十萬人需要人道援助——幾乎是美國半個總人口的規模。RSF 口頭接受停火，SAF 堅持「先繳械、再停火」。國際危機組織 Africa 部門的 Alan Boswell 老實講了一句：「人道停火」過去兩年已被提過至少五次，如果沒有持續外部壓力，這類口頭協議不會落地。

第二條是剛果民主共和國。DRC 政府和 M23 在瑞士連續五天談判之後，週六發出聯合聲明：承諾十天之內開始交換囚犯，M23 手上三一一人、DRC 政府手上一六六人，並正式啟動由國際大湖區會議和聯合國 MONUSCO 提供後勤支援的「驗證機制」，第一輪驗證行動一週內啟動。這不是終戰協議，但是二〇二四年十二月多哈框架之後，第一個有具體時程和人數的落地文件。

好，時事就說到這。先深呼吸一下，我要把你帶進科技的世界，但主題其實沒差太多：都是「系統級的權力重新分配」。

科技這邊，我想先說一個你可能以為只是工程師話題、但其實和你我錢包很有關的事情——比特幣 Core v31.0 正式發布。bitcoin/bitcoin 這個 repo 在時隔三年之後跨過了 v30 整個世代，直接跳到 v31。重點不在版號，而在打包的內容：block propagation 協議升級、UTXO 集合的記憶體結構整個被重寫、Taproot 之後的 mempool policy 重新校準。這三件事合在一起告訴你一件事：比特幣底層在熊市這兩年，悄悄完成了一輪效能最佳化的硬核準備。為什麼這個節點要注意？因為我們過去三個月聊的 Saylor、Strategy、現貨 ETF 的巨量進場，全都是「應用層」的故事——當底層協議終於把吞吐、UX、手續費邏輯收斂進來，你等於看到一個底座工程好了、樓可以蓋更高的關鍵時刻。

順便說，Saylor 昨天又加碼了三萬四千一百六十四枚比特幣，動用二十五億美元，總持倉首次衝破八十萬枚，大概占比特幣兩千一百萬顆硬上限的百分之三點八。他在 X 上放話「Think even bigger」。但華爾街老空頭 Kerrisdale Capital 的邏輯還是那套：Strategy 本質是「高波動率槓桿 BTC ETF」，溢價估值不合理，如果比特幣真的跌破六萬五，保證金觸發點會進場放大熊市下行。這邊我想提醒你——不論是看多的 Bitmine 本週再吃下十萬一千六百二十七枚以太幣，還是 Saylor 的八十萬枚比特幣，機構這一波的買盤密度，確實是過去三年沒見過的。

但同一天，DeFi 圈也傳來壞消息。LayerZero 把 Kelp DAO 上週被駭的 post-mortem 公開了，直接點名是北韓 Lazarus 集團。整個 DeFi 板塊一週損失超過六億美元，TVL 回到一年新低。Kelp、Aave、LayerZero 三方目前還在推諉損失怎麼分攤。這件事值得你記：它不是一個獨立事件，它是整個跨鏈橋「單點故障」在壓力下被北韓社交工程擊穿的最新證據。DefiLlama 的 0xngmi 為此提出一個不同的視角——他擔心「Lazarus everywhere」會稀釋掉 Kelp 本身多簽權限設計失敗的反省。換句話說：怪北韓很方便，但改不了橋架構。

從 crypto 換到模型世界。昨晚 Simon Willison 寫了一篇很技術但很重要的 post：Claude Opus 4.7 是史上第一個更換 tokenizer 的 Claude 模型。過去 Claude 家族 Opus、Sonnet、Haiku 共用 tokenizer，token 數量、計費、prompt cache 命中率可以通約。現在 4.7 換了，這個「通約」的便利性在某種意義上宣告結束。他為自己的 Claude Token Counter 工具加了跨模型比較，結果讓很多工程師眼睛睜很大——cache 跨版本命中率會被稀釋、API 成本模型要重寫。而 Anthropic 目前還沒公開 tokenizer spec。這是小事嗎？如果你在企業裡要估算全年 token 費用，不是小事。

同時 OpenAI 的動作完全對著幹。OpenAI 昨天宣布，Hyatt 集團全球員工導入 ChatGPT Enterprise，核心模型 GPT-5.4 搭 Codex。這是 2026 年繼沃爾瑪之後，第二個覆蓋超過十五萬員工的 enterprise 案例。OpenAI 藉這個機會，把 hospitality 產業從 Microsoft Copilot 的生態拉回自家。不過 Gartner 的 Bern Elliot 有一個很現實的 reminder：全員部署，不等於實際使用率。早年的 Microsoft Copilot 全域授權，12 個月後重度使用率大多落在百分之十五到二十五。這個數字是我們這個行業要長期盯著的真相。

接下來是 agent 這一塊的 housekeeping。LangChain 昨晚放出 langchain-openai 1.1.15，修了兩個雷點：streaming 回應裡 dict 型 response items 的兼容、以及依模型名稱自動推斷 Azure chat profile。對跨 Azure 和原生 OpenAI 的企業團隊來說，這是把好幾個月堆積的 profile hack 收斂進官方。OpenClaw 2026.4.19-beta.2 再補一刀，強制把 stream_options.include_usage 送給本地的 OpenAI-compatible 後端，原本顯示 0% 用量的 bug 徹底壓掉；他們新上的「nested lanes」設計把巢狀 agent 的作用域收斂到單一 target session，避免跨任務污染。CrewAI 1.14.3a1 支援 checkpoint 和 fork，獨立 agent 第一次可以斷點續跑、分叉還會發 task_started 事件。順便，Coinbase 把自家 x402 協議變成「AI bots 的 App Store」——代理每次調用 per-use 付費，Erik Reppel 放話 agentic commerce 會徹底重塑 SaaS 啟用成本。同時 Coinbase 還有個更瘋的實驗：CEO Brian Armstrong 在公司內測兩個以前共同創辦人 Fred Ehrsam、a16z Crypto 合夥人 Balaji Srinivasan 為原型訓練的 persona agent，任務是「給員工提供高層級回饋」。這是 persona agent 進企業內部治理層的早期案例——好玩，但風險視角也值得注意：用前高管語氣訓練的 agent 會強化 bias，責任歸屬該怎麼切，暫時沒人說得清。

再快速拉一條中國科技的線。華為昨天發布了第一款鴻蒙 AI 眼鏡，對標 Meta Ray-Ban 和 Apple Vision 中間的價格帶，主打 Pangu 模型端雲協同。更勁爆的是鋰礦——天齊鋰業 Q1 淨利預估年增一千五百三十到一千八百一十八個百分點，簡單說就是大爆發。同日盛屯礦業、創維數字一季度淨利全部爆噴。背景很清楚：SQM 鋰礦聯營加上鋰輝石精礦價格從每噸九百五十反彈到一千六百美元，中國資源股在 Q1 整體高於市場共識。野村中國研究提醒，這波有低基期效應，Q2 如果 CATL、BYD 的下游庫存回補見頂，毛利率在下半年會再度承壓——所以這不是結構性牛市，是補漲行情。

最後一個冷知識——螞蟻集團的「靈光」應用昨天上線了「靈光圈」，允許一般使用者一句話生成的「閃應用」直接發到朋友圈式社交流。最近爆紅的 SBTI 人格測試，就是一位二〇〇三年出生的金融系女生，花四五天手搓出來的，結果刷爆微信。一人應用時代，用社交分發當引擎，這件事值得你我認真看。

好了，時間到。今天從荷莫茲海峽聊到海牙法庭、從史瓦帝尼的非洲邦誼聊到布達佩斯的接班、從蘇丹的飢荒聊到比特幣 Core 的升級、從 Opus 4.7 的 tokenizer 聊到華為的鴻蒙 AI 眼鏡。這些故事看似零散，但其實是一張正在重組的世界秩序圖：能源、供應鏈、貨幣、運算底座，每一條都在被同一股壓力擠壓著重排。明天我們會拿到兩個關鍵答案——美伊停火會不會爆，Duterte 會不會正式進入審判程序。無論結果如何，我們明天準時見。我是你的主持人，這集到這裡。晚安。
