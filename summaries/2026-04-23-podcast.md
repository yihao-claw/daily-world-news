## 時事

### 一、杜特蒂最後一張法律牌被打掉

我們先從海牙說起。

時間倒回去一點點，你記得嗎？去年 10 月 23 日，ICC 的預審庭 I 已經裁定，法庭對杜特蒂在 2011 年 11 月到 2019 年 3 月這段「反毒戰爭」時期涉犯的反人類罪擁有管轄權。杜特蒂那邊不服，提了四項抗告——大概意思是說「菲律賓 2019 年就退出羅馬規約了，你海牙憑什麼管我」。這個抗告已經拖了快六個月。

今天，4 月 22 日海牙時間上午十一點，上訴庭開庭宣讀判決。結果你大概猜得到：四項抗告——全部駁回。

這個判決有幾個細節你需要抓一下。第一，票數是 4:1，一位法官只在其中一項理由上提了部分異議，意思是多數意見在每一個關鍵點都站得很穩。第二，ICC 這次把邏輯寫得特別直白：管轄權的基礎不是「你現在還是不是羅馬規約會員」，而是「你在當會員期間做了什麼」。杜特蒂 2011 到 2019 確實是會員，所以 ICC 的管轄權一直延續，哪怕他後來退出也不能追溯拔掉。

這個邏輯聽起來很技術，但對整個國際刑事司法體系來講是一次非常關鍵的背書。因為過去幾年，從俄羅斯、以色列到美國，只要有官員被 ICC 盯上，他們的第一招都是去撤會員身份、切割管轄。如果 ICC 讓杜特蒂抗告成功，那條路就等於被打開了。今天這個判決，是說「對不起，這招不行」。

接下來是什麼？是「確認起訴」這一關。預審庭預計在 4 月 28 日之前——也就是下週二之前——決定是否正式把案件送進審判。如果過了，那就是前菲律賓領導人第一次真的被送上國際戰犯審判台。

菲律賓國內反應我也講一下。Marcos 政府的官方說法是「尊重判決、不會要求釋放」，算是低調收尾；但菲律賓參議院那邊的 Imee Marcos——Marcos 總統的姊姊，她的陣營和杜特蒂派系反批「這是司法殖民」。你可以看到，菲律賓國內的政治分裂，其實跟著這個案子在同步撕裂。

我自己的觀察是：ICC 這個判決拖很久才出來，不是因為法官在猶豫——而是因為它一定要把每個論點都堵死，才不會被後面的案件拿來引用當漏洞。杜特蒂陣營現在基本沒牌了。

### 二、Trump 延長美伊停火，但伊朗國會說這是騙局

從馬尼拉轉到德黑蘭。

美伊停火原本是昨天晚上到期。這份停火從 2 月底的美伊戰爭一路拖下來，今天走到第 53 天。原本 Trump 週二還撂狠話說「極不可能延長」，市場都在準備開戰情境。結果昨天深夜他又翻盤。

翻盤的直接原因是巴基斯坦。巴基斯坦陸軍參謀長 Asim Munir 和總理 Sharif 打電話給白宮，正式請求美方「在伊朗領導層能拿出統一方案前」不要開火。Trump 把這通電話當台階下，宣布停火無限期延長，而且這次——你注意聽——沒有訂新的到期日。

他給的理由很妙：伊朗政府「嚴重分裂」，沒有人能代表德黑蘭談判。換句話說，美方不是不想談，是「連談的對象都找不到」。同時，美軍對伊朗港口的海上封鎖不撤。

伊朗那邊的反應怎麼樣？國會發言人顧問 Mohammadi 第一時間跳出來說：「這什麼都不算，這是美軍準備突襲前爭取時間的騙局。」國會鷹派老大 Ghalibaf 接著警告「德黑蘭不會投降」。更關鍵的是，副總統 Vance 昨天補了一句——第一輪停火談判之所以破局，是因為伊朗拒絕承諾放棄擁核。這把真正的核心爭議端上檯面了。

這個故事最弔詭的地方在哪裡？在於每一方都覺得自己在爭取時間。Trump 爭取時間，是希望伊朗內部換人上台、出現一個能談的對口。伊朗爭取時間，是等待以色列、美國之間出現裂縫，或等到美軍封鎖成本高到撐不住。巴基斯坦爭取時間，是想把自己塑造成不可取代的中東斡旋人。

我自己比較擔心的是第三個風險：只要沒有明訂期限，「擦槍走火」的機率反而會上升。美軍 4/22 登檢伊朗貨船，伊朗軍方已經揚言報復；荷莫茲海峽的民間航運成本這個月飆了 30% 以上。你這幾天看油價新聞會看得眼花撩亂，但核心邏輯就是：沒有和平協議、只有「停火」，就意味著任何一枚導彈落錯地點都可能引爆新一輪衝突。

### 三、EU 第 20 輪制裁 + 900 億歐元貸款一次過關，斯洛伐克換到 Druzhba 復油

我們從中東拉回歐洲。

布魯塞爾昨天 4 月 22 日有一場關鍵會議。歐盟成員國大使一次批准兩件大事：第 20 輪對俄制裁、以及對烏克蘭 900 億歐元的貸款案。這個結果之所以重要，是因為整整兩週前，大家都以為這兩案要卡在斯洛伐克和匈牙利手上。

你記得嗎？斯洛伐克外長 Blanár 上週還在喊話「沒有 Druzhba 管線的書面復油保證，斯洛伐克絕對封殺」。匈牙利的 Szijjártó 也在同一條線上。結果這兩週內，烏克蘭實質恢復了 Druzhba 管線的運油——對，就是那條從俄羅斯跨烏克蘭輸往匈牙利、斯洛伐克的老舊油管。一旦油開始流，兩國的政治成本就瞬間改變。昨天在布魯塞爾，Blanár 和匈牙利代表前後腳解除否決。

第 20 輪制裁裡面有幾條重點我講給你聽。第一，從 4 月 25 日起，歐盟禁止對俄羅斯旗幟的冰級船和 LNG 船提供技術與金融服務——這是對俄羅斯北極航線能源出口的精準打擊。第二，七家俄羅斯煉油廠和兩家油商被列黑名單。第三，新增 20 家俄羅斯銀行進交易禁令。這些都不是象徵性制裁，每一條都會直接影響俄羅斯 2026 年的能源現金流。

然後是那 900 億歐元。這不是純補助，是「可重複流用信貸」，烏克蘭可以分階段提取、用於財政赤字和軍備採購。基輔這邊算是在「保留能源槓桿」和「確保 900 億歐元入帳」之間選了後者。

我自己的觀察是——這輪通過其實有一個更深層的意義。斯洛伐克和匈牙利都不是真的親俄，他們是想在歐盟內部保留「最後一張否決牌」當政治籌碼。這次他們成功把 Druzhba 復油寫進書面條件、換到制裁通過，證明小國還是有在結構內討價還價的空間。但同時這個籌碼消耗掉之後，歐盟下一輪對俄施壓的門檻又降低了。

### 四、墨西哥 Sheinbaum：考慮制裁 Chihuahua 州

接著聊聊墨西哥，這個故事有點像懸疑劇。

上週末兩名 CIA 幹員在 Chihuahua 州的一次搗毀毒品實驗室行動結束後返程時車禍身亡，還有兩名墨西哥調查員一起。CIA 身份是 4 月 21 日才被曝光的。這直接把 Sheinbaum 這位墨國元首 逼上火線。

昨天 4 月 22 日的晨間記者會，Sheinbaum 講話很重：「任何美方機構都不得在墨西哥領土執行任務。」她已經致函美國駐墨大使要求完整說明，並且——這是重點——她正在考慮對 Chihuahua 州政府祭出制裁，因為省府允許 CIA 加入行動而沒有向聯邦政府報准。

墨西哥軍方的立場更尷尬。他們承認有支援這個地方反毒行動，但堅稱「中央完全不知道 CIA 會到現場」。白宮另一邊則放話說「墨方一直都知情配合」。這兩個版本直接對撞。

這件事放在 Trump 重返白宮以後對墨西哥毒品戰爭的施壓路線來看特別敏感。Trump 兩個月前才公開威脅要「派美軍進墨西哥打擊卡特爾」，Sheinbaum 當時的回應就是「不准美軍入境」。結果這次事件出現，等於把那個政治界線——「到底美方能不能在墨西哥土地上執行任務」——重新推到最前線。

墨西哥國內的反應你可以想像：左派在罵州長親美，右派在罵中央政府無能、連 CIA 進了家門都不知道。Sheinbaum 在中間要兩邊都安撫，所以她選擇把怒火對準 Chihuahua 州，而不是直接跟華府對嗆。這是很精準的政治操作。

### 五、台灣賴清德訪史瓦帝尼延期：北京跨域脅迫三國撤飛越許可

從墨西哥再拉回亞太。

這故事對台灣聽眾來說特別有感。原本，賴清德總統 4 月 22 日要啟程訪問史瓦帝尼——那是台灣在非洲唯一邦交國，任內首次。結果總統府昨晚正式宣布：延期、改派特使代行。

為什麼延期？官方說法很直白：塞席爾、模里西斯、馬達加斯加這三個非洲島國，在中方經濟脅迫下陸續撤回對「玉山 2 號」專機的過境飛越許可。改航線規劃過於複雜，基於安全顧慮，總統府決定延期。

你如果打開地圖看就會懂。從台北飛史瓦帝尼，最合理的南線會經過印度洋上這一串島國。一旦這三國同時拒絕飛越，替代路線要麼繞東非大陸內側、要麼從南美洲那邊繞過去——後者等於是繞了半個地球。

史瓦帝尼政府表態很有風度，說「對無法按原計畫接待感到遺憾，但這不影響雙邊關係」。北京外交部昨天的例行記者會沒有正面回應「撤許可」指控，只重申「一中原則」。塞、模、馬三國公開立場含糊，私下被指是在換取中方的雙邊貸款展延或基建項目。

這件事情的意義在哪？台北把它當作「國際社會認清北京壓力外交」的證據；北京則把整件事重新包裝為「各國遵守國際共識」。

我自己比較在意的是一個技術面問題：台灣之後每一次出訪邦交國，都要先處理「北京能不能在半路上拔掉你的飛越許可」這個新風險。這代表國家領袖出訪的成本——從航油、機組到外交協調——每一次都會比上一次高。它不是一時的阻撓，是一個結構性的新困難。

### 六、韓越河內峰會：核電、關鍵礦物、2030 年 1,500 億美元貿易目標

最後一個時事段落，我給你一個比較正面的消息。

李在明這位韓方元首 4 月 22 日在河內會晤蘇林，也就是越南的國家元首，簽了 12 項合作協議。最重磅的是兩項：越南寧順二號核電廠的建設可行性與資金協議、以及關鍵礦物供應鏈合作。

為什麼核電這麼重要？因為 2011 年福島核災後，越南原本凍結了民用核電計畫。寧順二號重啟等於是越南第一次回到核電賽道，而韓國在這個項目上前期砸了大錢想打掉俄羅斯和中方的競爭。李在明這次簽下協議，對韓國核電出海是戰略級的里程碑。

另一個看點是蘇林在會後公開表態「支持首爾恢復兩韓對話」。越南在東亞外交上一直玩「戰略自主」，不公開介入朝鮮半島議題。這次蘇林主動開口，被韓國解讀為對李在明路線的信任票。

雙邊目標也公布了：2030 年雙邊貿易衝到 1,500 億美元。這個數字聽起來驚人，但如果你知道 2025 年兩國貿易已經摸到 1,000 億美元，你就會發現這個 50% 成長，其實是合理外推。

背景我補一下——李在明這趟出訪是 4 月 19 日啟程的，先到印度，再到越南，四大財閥——三星、LG、SK、現代——的 CEO 全員隨行。這種大型經濟外交團，是典型的「後 Hormuz 時代」韓國版本：把供應鏈多元化從口號變成具體合約。

## 科技

### 一、Qwen3.6-27B：開源模型壓縮戰進入新階段

先從開源模型講起。

阿里雲昨天在 Hugging Face 放出 Qwen3.6-27B 的權重。這件事為什麼值得你花三分鐘聽？給你兩個數字：前一代 Qwen3.5-397B-A17B，這是一個混合專家模型，總參數 3,970 億、每次啟動 170 億；權重檔上傳到 Hugging Face 是 807GB。這一代 Qwen3.6-27B 是稠密模型，270 億參數，檔案只有 55.6GB。

關鍵是：阿里雲自己的測試裡，Qwen3.6-27B 在幾個主要程式碼 benchmark 上——包括 SWE-bench Verified、HumanEval、aider polyglot——全面超越上一代的 397B MoE。

這兩個數字放在一起的意思是什麼？同級能力的部署成本，被壓下一個數量級。你原本要一台八卡 H100 才跑得動 397B，現在單張 A100 甚至消費級 4090 透過 Unsloth 的 Q4_K_M 量化就能吃下——16.8GB 的 GGUF 版本，普通工程師家裡的機器就能跑 llama-server。

Simon Willison 昨天做了完整實測，confirm 了這個性能。但我必須給你一個反面視角。Gary Marcus——NYU 那位認知科學家——這週連發兩篇文章示範 OpenAI Images 2.0 連「攪拌器跟手肘」這種人體結構圖都畫不出來，他的論點是「這些 benchmark 分數越來越高，不是理解力變強，是訓練資料涵蓋度變高」。這個批評同樣適用於 Qwen。你測 SWE-bench 表現好，不代表模型真的懂軟體工程，可能只是它看過的 GitHub issue 比較多。

但不管怎麼說，從「開源模型能不能單卡跑」這個實際落地的角度，Qwen3.6-27B 是這幾個月裡最重要的一次壓縮。

### 二、GitHub Copilot + Claude Code 訂閱價同日重洗

從開源模型跳到商業模型——昨天一天之內，開發者端兩家龍頭都在同一個方向移動：漲價或收緊。

第一件事：GitHub 正式宣佈 Copilot 個人方案結構調整。第一，暫停個人版新註冊——請注意，是暫停，不是取消。第二，把 Claude Opus 4.7 移到 39 美元一個月的 Pro+ 方案，原本的 Pro 方案只能用到較弱的模型。第三，先前的 Claude Opus 4.5 和 4.6 直接下架。

第二件事：同一天，Anthropic 官網 pricing 頁面被發現「偷偷改版」，Claude Code 被標註為只有 100 美元一個月的 Max 方案才能用。網路上瞬間炸鍋，Internet Archive 上有人翻出前一天快照還是 Pro 勾選的狀態。Anthropic 後來又默默改回來，沒有任何公告。Simon Willison 直接寫了一篇「Is Claude Code going to cost $100/month?」的稿子，結論是「可能不會，但整件事極度混亂」。

這兩個動作背後的邏輯其實很相似：agentic coding 產品的單次推論成本太高，訂閱制已經撐不住了。Claude Code、Copilot 這類工具，用戶一個 session 可能消耗幾千萬到上億 token，但訂閱費只有 20 美元。這個單位經濟模型早就破裂，只是誰先喊出來、喊多少的問題。

你是個開發者的話，這件事對你的直接影響是：短期內，用 OpenAI 或 Claude API 直接付費、搭配開源 IDE 工具，性價比會比訂閱方案好。中期來看，下一季可能還會有第二波調價。

### 三、紐約州預測市場監管升溫：加密立法面前的「州府大戰」

從開發者工具跳到加密與監管。

昨天紐約州長 Hochul 簽了一道行政令：禁止州政府員工利用內部資訊下注預測市場。這個背景是前日紐約州總檢察長 Letitia James 起訴 Coinbase 和 Gemini，指它們的預測市場產品「實質上是非法賭博」，COIN 當天盤中跌 6%。現在 Hochul 的行政令把打擊面從「交易所」擴大到「公務員自己」。

為什麼這件事重要？因為它是「聯邦 vs 州政府在加密監管上分裂」的最清楚案例。聯邦這邊，美國加密立法今天有另一個消息——參與 Market Structure Act 談判的關鍵參議員表示「穩定幣獎勵條款已經在好位置」。但是投行 TD Cowen 同一天發報告，列出五道仍未解的路障：CFTC 委員缺額、World Liberty Financial 的利益衝突、伊朗用加密結算、跨州稅務口徑、以及 DeFi 協議歸類。這意味著聯邦立法救援短期到不了，但州政府的監管攻勢——紐約只是第一個開火的——會在接下來幾個月持續推進。

Polymarket、Kalshi 這些預測市場平台，在紐州基本會被打到退出。這對 Web3 新創的合規成本是實質壓力。看多的一方會說：Circle 股價月漲 30%、市場已經 price in 穩定幣立法局部勝利；看空的一方會說：整包法律今年 Q3 前不可能成，州級監管會先把生態修理一輪。

### 四、OpenAI Workspace Agents、Zed Parallel Agents：agentic IDE 新一輪競賽

最後聊一下 agent 工具這塊。

OpenAI 昨天正式發表「Workspace Agents in ChatGPT」。這是把 ChatGPT 升級成可以接 Google Workspace 或 Microsoft 365 的 OAuth 權限，然後讓它直接幫你開檔、改簡報、跑郵件歸類。Hacker News 上 81 分、27 留言，大家討論的重點是「這會不會取代 Microsoft Copilot Workspace」。

幾乎同一天，Zed Editor 推出「Parallel Agents」。Zed 是一個比較新的程式編輯器，這次加的功能是讓你同時跑多條 agentic 任務，每條 agent 在獨立 git worktree 裡跑，寫完合併 diff。HN 上 128 分，定位直接對上 Claude Code 和 Cursor 的 autonomy 模式。

我自己覺得這兩件事值得放在一起看。過去 agent 這個詞在 2024 年還在「能不能真的跑完一個任務」的階段，現在大家已經在比「能不能同時跑多個、合起來還能 review 得過」。這是一個質變。

但這裡也有一個反面視角你要記住：開第三方雲端 OAuth 權限給通用 agent 的資料外洩面，比傳統 LLM chat 大一個數量級。企業 IT 部門現在正在制定 policy，基本上短期都會要求這類 agent 要先走 DLP 或 SIEM。這意味著 OpenAI 的 Workspace Agents 要進大企業，門檻其實非常高。

### 五、Firefox 150 + Claude Mythos Preview：271 個漏洞一次修完

講個我自己覺得很有啟發性的冷故事收尾。

Mozilla 工程師 Bobby Holley 昨天公開一篇文章：這週釋出的 Firefox 150，有 271 個漏洞是靠 Anthropic 早期 Mythos Preview 模型協助評估找出的。這是史上單一 Firefox 版本修補漏洞最多的紀錄之一。Holley 寫道——他們團隊「把其他所有事情都停下來，單一目標、全力衝刺」，才看到隧道盡頭的光。

這是個很重要的信號。第一，AI-assisted security triage 從「概念性 demo」走到「真的能量產補丁」這個臨界點。第二，這個合作不是 Anthropic 的公關稿，是 Mozilla 自己的工程師跳出來講。

我給你一個批評視角。271 個漏洞聽起來驚人，但 CVE 嚴重度分布沒有公開，大多數可能是低風險記憶體誤用。AI 幫你挖出一堆 issue 是容易的，但真正的瓶頸是在「誰來 review、誰來寫 patch、誰來測試」。如果這一步還是人，那 AI-assisted triage 的 throughput 提升空間是有限的。
