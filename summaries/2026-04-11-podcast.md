# EP40 — 2026年4月11日

嘿，早安，歡迎回來，這裡是每日新聞 Podcast 第四十集。今天是四月十一號，週五，我是你的主持人。

今天最重要的兩件事。第一，美國跟伊朗的停火正在崩潰邊緣，以色列在停火宣布當天就對黎巴嫩炸死超過三百人，然後今天在巴基斯坦的伊斯蘭堡，美伊代表終於要第一次面對面坐下來談。第二件事，Artemis II 的四位太空人今晚就要回到地球了，五十四年來人類距離地球最遠的旅行，即將在聖地牙哥外海畫下句號。

好，我們先來聊中東，因為這真的是牽動全球局勢最大的一條線。

你還記得前天跟你說的美伊停火嗎？當時大家都鬆了一口氣，油價瞬間暴跌超過百分之十四，全球股市也跟著反彈。但才過兩天，這個停火就快撐不住了。問題出在兩個地方。

第一個問題是荷姆茲海峽。這條海峽是全球石油運輸的咽喉，每天大概有兩千一百萬桶原油要經過這裡，占全球海運石油的三分之一。伊朗在戰爭期間封鎖了海峽，美國停火的條件就是「立即、完全、安全地重新開放」。但到現在，船還是過不去。伊朗說這是他們的主權，荷姆茲海峽的控制權不容談判。你看到衝突的核心了嗎？美國覺得開放海峽是停火的前提條件，伊朗覺得那是主權問題，根本不該被放在停火談判的框架裡。

第二個問題更致命。以色列在停火宣布的同一天，對黎巴嫩發動了他們自己命名為「永恆黑暗行動」的大規模軍事打擊。一天之內，至少三百零三人死亡、一千一百五十人受傷。這不是小規模衝突，這是二〇二六年黎巴嫩戰爭開打以來，單日最高的傷亡數字。以色列國防軍說這次行動針對的是真主黨的基礎設施，但在停火才剛宣布的時間點做這件事，政治訊號非常強烈。

這就帶出一個根本性的問題：停火的「範圍」到底涵蓋什麼？Netanyahu 說得很清楚，黎巴嫩不在停火範圍內。Trump 也附和，說黎巴嫩是「另一回事」。但調停人巴基斯坦完全不同意，巴國總理 Sharif 堅持他斡旋的停火涵蓋黎巴嫩。伊朗也引用這一點，指控美國和以色列已經違反了停火協議。所以你看到了，停火才第三天，各方對停火本身的定義就已經南轅北轍了。

更讓國際社會震驚的是，半島電視台的記者 Mohammed Wishah 在加薩的 Deir el-Balah 被以色列無人機擊殺，當時他正在參加一場葬禮。新聞自由在這場衝突中持續受到嚴重威脅。英國首相 Starmer 的反應也值得注意，他公開說他對 Trump「受夠了」，並且批評以色列對黎巴嫩的攻擊影響了英國民生。這種措辭在英美特殊關係的歷史裡是非常非常罕見的。

今天的重頭戲就是伊斯蘭堡的美伊談判。伊朗帶著他們的十點方案過來，核心爭議還是荷姆茲海峽。美國能不能接受某種形式的共管或者分階段開放？伊朗能不能在主權論述上做出彈性？這是接下來幾天全世界都在看的焦點。油價目前在九十五到一百零五美金之間劇烈震盪，每一條來自伊斯蘭堡的消息都會直接反映在油價上。

把視角轉到歐洲。Putin 宣布了復活節停火，從今天莫斯科時間下午四點開始到明天半夜。但老實說，烏克蘭對這種停火承諾的信任度已經降到零了。就在停火宣布前的二十四小時，俄軍還發射了一百一十九架無人機攻擊烏克蘭，其中大約七十架是 Shahed 型攻擊無人機。奧德薩地區的兩座重要能源設施被嚴重破壞，這對烏克蘭的電力供應又是一次打擊。

從戰場態勢來看，過去一週俄羅斯反而丟失了大約一平方英里的領土，這跟之前幾週小幅推進的趨勢是逆轉的。但 Zelensky 警告說俄羅斯正在從戰略預備隊調兵進入烏克蘭，所以戰局可能又會有新的變化。累計到現在，俄軍的戰損人數已經超過一百三十萬了。光是昨天一天就增加了一千一百三十人。

再來看匈牙利。明天就投票了，四月十二號。奧班執政十六年，Fidesz 黨長期一黨獨大，但這次他遇到了真正的挑戰者。反對黨 Tisza 的領袖 Magyar Péter 在過去一年迅速崛起，民調數字非常有意思。PolitPro 的彙整趨勢顯示 Tisza 領先，拿到大約四十九個百分點，Fidesz 大概四十個百分點。但其他機構的數字差異大到不可思議：Medián 說 Tisza 領先二十到二十一個百分點，但二十一世紀研究所反而說 Fidesz 領先五個百分點。政治學者 Gábor Török 直接說，這種程度的分歧「在學術研究的方法論上無法解釋」。匈牙利的選舉制度有利於執政黨，所以即使 Tisza 在全國得票率領先，也不一定能拿到國會多數。明天晚上的開票結果，對歐盟政治生態、對烏克蘭援助、甚至對歐俄關係都會有連鎖反應。

台灣的部分，國民黨主席鄭麗文這次的中國之行走到了最高潮——她在北京見到了習近平。這是二〇一五年馬英九在新加坡與習近平會面之後，跟習近平見面的最高層級台灣政治人物。鄭麗文的訪問從四月七號持續到十二號，走訪了南京、上海和北京。她在會後呼籲「和解」，說這是兩岸關係「新春天」的起點。

但執政黨民進黨完全不領情。賴清德政府的國安官員說，這個訪問「帶有高度政治意圖」，在中國對台灣持續施加軍事壓力的背景下——記得台灣以北的四十天空域預留到現在還在進行——這種時候去北京跟習近平握手，傳遞的信號是非常有問題的。島內輿論也在激辯：這到底是「和平」還是「投降」？

非洲快速帶兩件事。吉布提昨天投票了，七十八歲的 Guelleh 總統面對一個幾乎沒有存在感的對手，預料會繼續連任。吉布提雖然是個小國，但位置非常關鍵，扼守紅海的 Bab al-Mandab 海峽，那裡有美國在非洲唯一的永久軍事基地，法國最大的海外基地也在那裡。喀麥隆的 Biya 總統則是做了一個引發爭議的決定，把自己的兒子 Franck Emmanuel Biya 拉上來做副手，同時兼任武裝部隊首長。Biya 已經執政超過四十一年了，這個任命讓世襲政治的疑慮更加明確。另外，查德境內有一百三十萬蘇丹難民正面臨援助斷裂的危機，糧食、用水和基本服務都因為國際資金削減而岌岌可危。

好，來聊科技。

今天科技圈最讓人興奮的消息跟太空有關。Artemis II 的四位太空人——NASA 的 Reid Wiseman、Victor Glover、Christina Koch，加上加拿大太空總署的 Jeremy Hansen——預計在美東時間今晚八點零七分濺落太平洋。他們四月一號升空，在太空中待了十天，繞了月球一圈。四月六號的時候，他們距離地球二十四萬八千六百五十五英里，打破了一九七〇年 Apollo 13 創下的人類最遠距離紀錄。他們還拍到了從未有人類親眼見過的月球區域，以及罕見的太空日蝕畫面。回收船 USS John P. Murtha 已經在聖地牙哥外海就位，天氣條件也符合標準。太空人濺落後兩小時內會被直升機接送到軍艦上，進行身體檢查後飛回休士頓的 Johnson Space Center。這是一個歷史性的時刻，五十四年來人類第一次從月球軌道安全返回。

開發者工具圈今天有幾個重要更新。Ollama 發布了零點二十點五版，最大亮點是整合了 OpenClaw 頻道設定，現在只要一行指令就能把 Ollama 接上 WhatsApp、Telegram、Discord 等訊息平台。在相容的 GPU 上也為 Gemma 4 啟用了 flash attention 加速推論。LangChain 這邊動作更大，langchain-core 直接從一點二跳到一點三點零 alpha，這是一個大版本升級。重構了 streaming metadata 的處理方式以提升效能，也加強了 template 的安全清理和 symlink 保護。這些都是在為更複雜的 Agent 使用場景打基礎。

CrewAI 也出了新版，一點十四點二 alpha 二，最讓人眼睛一亮的功能是 Checkpoint TUI，支援樹狀檢視、fork 和可編輯的 inputs 跟 outputs。還加入了 reasoning tokens 和 cache creation tokens 的追蹤統計，讓你更清楚知道每次 Agent 運行到底花了多少 token 在哪裡。

AI 安全這個主題今天有好幾條值得關注的消息。微軟開源了 Agent Governance Toolkit，這是第一個覆蓋 OWASP Agentic AI 全部十項風險的開源工具包，用 MIT 授權。裡面包含 Policy Engine 做策略管理、Trust Mesh 做信任驗證、Execution Sandbox 做沙箱隔離，而且已經整合了包括 LangChain、CrewAI、AutoGen 在內的十二個以上主流框架。對於正在部署 Agent 系統的團隊來說，這是一個很有參考價值的起點。

但同時，AI 工具鏈本身的安全問題也在浮現。開源 AI 流程編排平台 Flowise 被發現有一個 CVSS 滿分十分的程式碼注入漏洞，而且已經在野外被攻擊者利用了。如果你有在用 Flowise 來串接 LLM 工作流，這是最高優先級的更新。另外 Fortinet 的 FortiClient EMS 也有一個嚴重的預認證繞過漏洞，攻擊者不需要任何登入憑證就能透過特製的 HTTP 請求執行惡意程式碼，已經被 CISA 列入必須修補的 KEV 清單。

Anthropic 最近動作頻繁。他們花了四億美金收購了生技新創 Coefficient Bio，正式跨足 AI 輔助的藥物發現和蛋白質工程領域。這代表 AI 公司的野心已經不止於軟體，開始觸及生物科技這種更硬核的科學領域了。另一方面，他們的 Mythos 模型持續引發社群辯論。Gary Marcus 列了三大理由說 Anthropic 的公告過度渲染了 Mythos 的能力，認為開源小模型其實可以復現大部分功能。但 Anthropic 的 Project Glasswing 網安計畫已經拉進了超過四十家企業，包括 AWS、Apple、Cisco，用 Mythos Preview 來做防禦性的漏洞獵捕。這到底是真正的技術突破還是行銷話術？市場還在觀望。

Simon Willison 寫了一篇有趣的觀察文。他指出 ChatGPT 的語音模式其實跑的是一個比文字版弱很多的舊模型，知識截止日期只到二〇二四年四月，還停留在 GPT-4o 的時代。Andrej Karpathy 也呼應這個觀點，說根據使用者接觸 AI 的介面不同，對 AI 能力的認知正在產生越來越大的落差。你用語音跟 AI 講話，感覺好像在跟最聰明的 AI 對話，但實際上你拿到的是一個遠不如最新文字模型的舊版本。

加密貨幣這邊有幾件事。OKX 的創辦人徐明星公開稱幣安前 CEO CZ 是「騙子」，兩家全球最大加密貨幣交易所的創辦人公開對罵，這在業界很罕見。歐洲最大資產管理公司 Amundi，管理規模兩兆三千億歐元，跟 Spiko 合作在 Ethereum 和 Stellar 上推出了代幣化基金 SAFO，由 Chainlink 驅動，三週內就衝上四億美金的管理規模，創下代幣化基金史上最快增速。這說明傳統金融對鏈上資產的接受度正在加速。美國 SEC 排定四月十六號要開圓桌會議討論 CLARITY Act，這個法案如果通過，有望終結多年來 SEC 跟 CFTC 之間對數位資產監管權的爭議。比特幣開發者 Olaoluwa Osuntokun 也做了一件未來感很強的事——建構了第一個抗量子攻擊的比特幣錢包原型，為未來量子電腦可能對區塊鏈加密造成的威脅預做準備。

消費科技快速帶一下。Samsung 跟 Apple 簽了三年獨家 OLED 面板合約，供應折疊 iPhone 使用。面板採用 CoE 技術，就是把色彩濾光片直接整合到封裝層裡面，可以減少光損失、提升亮度和色彩表現。不過 Apple 對折疊機的銷量預期比較保守，生產量從原本預估的一千萬台下修到三百萬台，定價可能落在兩千到兩千四百美金之間。全球智慧型手機第一季出貨量年減百分之六，主要原因是 DRAM 和 NAND 記憶體短缺——因為記憶體供應商把產能優先分配給了利潤更高的 AI 資料中心。Samsung 還宣布七月要在美國淘汰原生的 Messages App，引導用戶轉到 Google Messages。

好，最後來快速過一下市場數字。

美股昨天收盤，四月十號的數字。S&P 500 收在六千八百一十九點，基本持平，微跌不到十分之一個百分點。Nasdaq 表現好一些，漲了百分之零點三五，收在兩萬兩千九百零三點。道瓊工業指數稍微弱一點，跌了大約半個百分點，收在四萬七千九百二十五。歐洲 STOXX 600 漲了百分之零點三七。亞股的部分，日經、恆生、上證、台股目前都是九號的收盤價，超過二十四小時了，要等亞洲開盤才有最新數據。

匯率方面，美元兌日圓一百五十九點二九，日圓繼續走弱。歐元兌美元一點一七，漲了百分之零點六三。新台幣三十一點七五，基本持平。

油價還是在停火消息後的震盪區間裡。WTI 昨天收九十六點三六美金，跌了大約一個半百分點。Brent 收九十四點六二，也是類似跌幅。黃金四千七百九十美金，幾乎沒動。比特幣七萬兩千八百八十美金，漲了百分之一點五五；以太坊兩千兩百四十美金，漲了超過兩個百分點。CPI 報告今天就要出來，經濟學家預估通膨率可能在三點三到三點四個百分點，比上個月高了大約一個百分點，如果數字太高，可能會對加密貨幣和股市都帶來壓力。
