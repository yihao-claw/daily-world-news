嗨，大家好，今天是四月二日，星期四。

今天世界上發生的事情，密度非常高——特別是中東方向，可以說每隔幾個小時就有新的進展。讓我先預告一下今天的大事：時事方面，美國今晚要對全國發表演說，宣布伊朗戰爭的下一步；特朗普說他「絕對在考慮」讓美國退出 NATO；以色列則在黎巴嫩的地面入侵中宣示要長期佔領南部。科技圈這邊也不消停——Ollama 發布了一個重大架構更新，讓 Mac 用戶在本機跑大型模型這件事變了一個等級；另外 Anthropic 那個被洩漏出來的新模型 Claude Mythos，今天有了更清楚的輪廓——但好消息是，你大概短期內還用不到它。還有一個讓 DeFi 圈心驚肉跳的消息：Solana 最大的衍生品交易所 Drift 被駭了，而且規模大到不像話。

好，我們來一件一件聊。

先說伊朗這邊。

美東時間今晚九點，特朗普要向全國發表演說。白宮說這是「重要更新」。在演說前的幾個小時，特朗普在 Truth Social 上發了一篇文，伊朗總統佩澤希齊揚已經向美方請求停火。他的原話大概是——「停火？當然可以，但前提是荷姆茲海峽必須開放、自由、暢通，否則我們會繼續把伊朗炸回廢墟。」

這個說法放出去以後，伊朗總統府第一時間強力否認，說這是「完全虛假、毫無根據」的說法，強調伊朗決心奮戰到底。

那這種局面怎麼理解？你會發現，特朗普用的是一個非常典型的他式外交手法——先拋出一個「對方已經在談了」的說法，再設出一個條件，讓自己看起來是在主導局勢。這給他今晚的演說留下了兩種可能的劇本：一，他宣布伊朗「跪了」，美國打贏了，接下來進入某種談判框架；二，他繼續升級施壓，把荷姆茲開通設為一切的前提。

戰場上的實際情況呢，美軍中央指揮部已經確認，從開戰到現在，他們對伊朗境內超過一萬兩千三百個目標執行過打擊，美軍陣亡十三人，受傷數百人。這個數字聽起來好像很克制，但如果你看整個中東局勢，傷亡擴散的方式遠不止這些。

油價是另一個告訴你現在局勢的指標。特朗普發出停火訊號之後，布倫特原油從昨天的高點一百一十八美元跌到今天的大約一百零一美元。這聽起來是個好消息——但別忘了，戰前油價是七十美元。換句話說，現在跌回來的那一段，只是從「大漲」變成「中漲」，市場還是在高度戒備的狀態。更壞的預測是什麼？Stratas Advisors 的首席分析師直接說，如果荷姆茲海峽再關閉一個月，布倫特可能衝到一百九十美元。那個數字一旦成真，全球通膨的影響會是另一個級別的問題。

說到這裡，我想讓你感受一下這件事對亞洲的衝擊，因為其實跟台灣、日本、韓國都非常直接相關。

韓國今天宣布提升資源安全危機警報等級。說具體一點——他們進一步收緊公務部門的駕車限制，而且財政部說，如果油價突破每桶一百二十美元，他們可能要把駕車管制擴大到一般民眾。這是韓國三十五年來考慮的第一次大規模民間駕車限制，上一次是一九九一年波灣戰爭時期的事。韓國總統李在明已經通過了一個大概等於一百九十億美元的補充預算，正在執行中。

日本那邊，分析師估計日本可以用煤炭補充大概百分之七十的天然氣發電缺口，但前提是情況不能拖太久。韓日兩國大使最近在媒體上說，「我們需要緊密協商，一起應對伊朗危機」——你幾乎是公開承認，兩國在能源安全上都非常脆弱。

紐約時報有一篇分析標題叫「亞洲被夾在油價和美元之間」，說的是因為油是用美元計價的，當油價漲的時候，亞洲各國不只要付更多錢買油，本幣還同時在貶值，等於是雙重壓力——你的錢在縮水，你的油價帳單在漲。從印度到東南亞到韓國，都在承受這個壓力。

這個時候回過頭看，為什麼北京在密切盯著荷姆茲的動態？因為穿過那個海峽的石油，有大概百分之二十六是要去中國的。這個數字告訴你，解鎖荷姆茲對中國的重要性，幾乎和對美國一樣大。

好，讓我們從中東稍微往北走一下，說說 NATO 這個大炸彈。

特朗普今天在兩個不同的採訪裡說，他是「absolutely considering」，也就是他「絕對在考慮」讓美國退出 NATO。理由就是，他認為歐洲盟友拒絕派遣艦艇協助打通荷姆茲海峽，等於是把美國一個人丟在那邊打仗。他說 NATO 是「紙老虎」，說歐洲在真正需要展示承諾的時候退縮了。

歐洲的反應是震驚加憤怒。法國、德國都有官員強烈反對，他們的論點是，NATO 的集體防衛機制對歐洲安全是根基性的存在，動搖它的代價不是一個戰爭就能衡量的。法律學者也指出，美國退出 NATO 在國內法層面有爭議，國會那邊可能不會輕易讓過。

但更讓人擔心的是戰略層面的隱憂。前美軍將領在接受 Stars and Stripes 採訪時說，如果美國真的退出 NATO，最大的受益者是俄羅斯，普丁在烏克蘭的底氣會立刻增加好幾個倍。

說到烏克蘭——昨天我跟你聊過烏克蘭前線的基本情況，今天有一個值得注意的細節：澤連斯基的「合作夥伴」，也就是西方盟友，已經在詢問烏克蘭能不能「減少攻擊」俄羅斯的石油設施。你現在理解為什麼了——因為伊朗戰爭已經讓油價很高了，如果烏克蘭繼續打掉俄羅斯的石油出口，歐洲能源成本會雪上加霜。這是一個非常現實但也非常尷尬的外交壓力——你一邊在支援烏克蘭，一邊又因為能源利益告訴他們手下留情。

前線方面，烏軍在 Q1 累計收復超過四百平方公里——這個數字，在這場戰爭的規模下算是有意義的進展，但遠還沒到可以宣告戰情逆轉的程度。俄方今天昨天的損失報告是一千零六十人，這只是烏克蘭國防部的官方數字，需要保留一定的判斷空間。

來說以色列黎巴嫩這條線。

今天貝魯特南郊遭到以色列空襲，至少七人死亡。以色列陸軍同時宣布在黎巴嫩南部新增部署了一個師，也就是說地面入侵在擴大。

但今天最值得你注意的新聞，不是空襲的死亡人數，而是以色列防長 Israel Katz 的一個聲明。他說得很直接：等到跟真主黨的戰鬥結束以後，以色列計劃佔領利塔尼河以南的區域，也就是距離以黎邊境大約三十公里的範圍，作為永久的「緩衝帶」。

這是以色列迄今為止，對自己領土意圖最清楚的一次表態。黎巴嫩政府的用語很激烈，叫「公然吞併」。當地南部居民面對的，是一個非常殘酷的現實——利塔尼河以南是黎巴嫩什葉派社區的主要聚居地，也有基督徒社區。現在已經有當地人公開說他們擔心自己永遠回不了家。

從以色列的角度來說，這個邏輯在安全上有其脈絡——二○○六年以黎戰爭之後，國際社會試圖用聯合國部隊維持緩衝，但最終真主黨還是在南部重新武裝。以色列這次顯然決定不再走那條路。但從國際法的角度，這就是對別國領土的軍事佔領，是另一條完全不同的線。

美國白宮今天的重心在伊朗演說，對以色列入侵黎巴嫩相對沉默——這個沉默本身，某種程度上也是一種態度。

讓我們把目光移到台灣海峽。

今天台灣這邊最大的新聞，鄭麗文正式確認，她將在四月七日到十二日率代表團訪問中國，行程包括江蘇、上海，最後到北京，這是應中共中央委員會和習近平的邀請。這是近年來最重要的兩岸政治接觸之一。

鄭麗文昨天在出發前接受採訪，說「世界現在這麼動盪，我們不需要再有一個台灣危機」，她說她去是為了尋求和解。

台灣行政院長卓榮泰的回應很有意思，他用了一個字：「神經」。他說這個時間點、這個背景下選擇去訪問北京，「需要不少神經」。這個批評說的是，在美國正在和伊朗打仗、全球注意力都在中東的時刻，北京主動發出邀請，很難不讓人聯想到背後的戰略考量。

分析師的解讀是這樣的：這次訪問在 Trump 原本計劃訪問北京的前一個月發生——不管 Trump 那次訪問後來有沒有延期，北京都可以透過國民黨的到訪，製造出一個「台灣內部也有和平聲音」的外交敘事。在全球各地都在應對伊朗危機的背景下，這個訊息的傳播效果會特別好。

再說一件讓人揪心的事，但通常在大新聞的背景下不太被注意到——

今天，也就是四月二日，孟加拉難民營裡數十萬羅興亞難民的食物援助正式削減。NPR 的報道說，程度「令社區絕望」。這件事的背景，是 USAID 在特朗普政府主導的全球人道援助縮減之後，多個 UN 機構的預算都在縮水。同時，伊朗戰爭造成的糧食和能源通膨，讓這些本來就在危機邊緣的人，又被往下推了一步。沒有人在報導這件事，因為所有鏡頭都在中東——但它真實地發生著。

好，說完這些嚴峻的時事，我們換個頻道，來看看科技圈今天的動靜。

先說一個讓很多 Mac 用戶眼睛一亮的消息。

Ollama v0.19.0 今天發布了，這是他們歷來最重要的架構改變之一。在 Apple Silicon，也就是 M 系列晶片上，Ollama 把推理引擎從原本的 llama.cpp 全面換成了 Apple 自家的 MLX 機器學習框架。

你可能會問，這對我有什麼影響？如果你在用 MacBook 本機跑語言模型，影響是很直接的——初步測試顯示，在 M3 或 M4 Pro 以上的裝置，推理速度可以提升三到五成，同時記憶體效率大幅改善。這意味著你現在可以在 MacBook 上跑更大的量化模型，而不會出現之前那種過熱降速的情況。

批評的聲音也有——有開發者指出，切換到 MLX 意味著 Ollama 的技術棧更深度綁定 Apple 生態了，如果 Apple 哪天修改 MLX API，Ollama 就得跟著改，維護成本上升，跨平台通用性也降低了。但對大多數個人用戶來說，如果你是 Mac 用戶，這是今年 Ollama 最值得更新的版本。

接著說 Anthropic 的 Claude Mythos。

這個模型在大概一週前從某個地方洩漏出來，洩漏內容顯示它是 Anthropic 目前最強的模型，代號 Capybara，在編程、推理、網路安全防禦方面有顯著進展，被描述為比現有 Opus 系列更上一層。

今天，Anthropic 確認了這個模型確實在進行早期存取測試，但他們選擇了一個非常謹慎的部署策略：目前只向「特定企業客戶」開放，而且主要集中在網路安全防禦這個領域，沒有近期公開計畫。

你有沒有注意到這和他們以前的發布節奏有多不同？過去 Claude 系列的新版本，基本上是一個月或幾個月就讓一般用戶用上。這次顯然是刻意放慢了——部分可能是安全考量，因為這個模型的網路安全能力太強了，如果被用於攻擊性用途會很麻煩；部分可能也是一個市場策略，先限制存取、製造稀缺感，再正式發布。

Polymarket 的預測市場顯示，Q2 也就是六月底前公開發布的概率大約是百分之三十五。我猜你如果想用到 Mythos，可能還要等幾個月。

不過科技圈今天有一個壞消息，需要認真跟你說一下。

Drift Protocol，就是 Solana 上最大的去中心化衍生品交易所，昨天也就是四月一號遭遇了重大駭客攻擊。The Block 確認，損失金額至少達兩億美元。Drift 官方在發聲明的時候，特別強調「這不是愚人節玩笑」——這個細節你想一下就很心酸，因為他們的駭客事件剛好發生在愚人節，所以不得不先澄清這是真的。

Drift 已經暫停提款，正在進行鏈上取證調查，攻擊者利用的是智能合約的漏洞，系統性地提取了多個金庫的資金。鏈上取證公司說追蹤進展迅速，有可能追回部分資金，但目前沒有確定的結果。

這是 Solana 生態今年規模最大的單一安全事件。如果你有資金在 Drift 或者 Solana 的 DeFi 協議上，現在可能是一個應該認真審視自己風險敞口的時候。

另一個今天的網路安全警示，是針對開發者的。Axios，就是那個你幾乎每個 JavaScript 項目都會用到的 HTTP 套件——每週下載量超過五千萬次——今天被曝出供應鏈攻擊，有惡意依賴套件被短暫發布到 npm registry 上。雖然很快就被下架，但在那個時間窗口裡，已經有一些自動化的 CI/CD 流程把它拉取下去了。這是繼 XZ Utils 事件之後，開源供應鏈安全的又一個警示——你引用的那些套件，不一定都是安全的。

順帶提一下加密貨幣的幾個值得注意的動態。比特幣 Core 的 v31.0 第二候選版本今天發布了，包含 P2P 網路優化和記憶體池管理的改進，正式版應該再過幾週就出來。以太坊主要客戶端 go-ethereum 也發布了 v1.17.2，修復了一個可能導致節點資料損壞的關鍵漏洞，如果你在跑以太坊節點，這是一個建議盡快升級的版本。

監管面，美國財政部今天就《GENIUS Act》，也就是穩定幣監管法案，發布了正式的規則提案，尋求對州層級穩定幣監管的公眾意見。這是美國在加密立法方面今年最具實質進展的動作之一，明確法律框架理論上對機構入場是好事，但 DeFi 倡導者在擔心法規適用範圍太廣，可能波及去中心化協議。

我想在結束之前，多說一件你可能沒注意到的事，但它值得被提起。

今天，孟加拉難民營裡的數十萬羅興亞難民，食物援助正式削減了。不是要削減，不是在討論要不要削——是今天開始真正削減了。NPR 的報道說，「令社區絕望」。這件事的背景，你要理解兩層：第一層是美國 USAID 在特朗普政府的全球援助縮減下，聯合國難民署、世糧計劃署等機構的預算都在縮水；第二層是伊朗戰爭造成的糧食和能源通膨，讓本來就在生存邊緣的人，又被往下推了一步。

你有沒有發現，現在所有的攝影機都對著中東？那片土地上當然有真實的苦難，但有些苦難是沒有攝影機的。孟加拉難民營裡的羅興亞人，從緬甸流離失所以後，在這些臨時的帆布屋子裡住了好幾年了。他們今天的糧食又少了。

這不是今天最大的頭條，但在我心裡，它是今天最沉重的那一筆。

今天股市方面，受到特朗普停火訊號的刺激，美股三大指數都收漲，標普五百漲了將近百分之零點七，納斯達克漲了超過百分之一，道瓊也漲了超過百分之零點六。歐股漲幅更大，STOXX 600 漲了超過百分之二點五，英國富時一百甚至創下了年度單日最大漲幅——歐洲在這個問題上對停火訊號反應更強烈，可能因為他們在能源上更脆弱，任何停戰的可能性對他們來說都是直接的好消息。

亞股昨天也跟著強勢反彈，台灣加權指數大漲百分之四點五八，日經更是飆漲超過百分之五點二四——可以看到停火訊號對亞洲市場的提振力道非常強。

能源市場，油價大幅回落——布倫特從昨天的一百一十八美元跌回到大約一百零一美元，WTI 跌到九十九美元附近。但這兩個數字，都還是比戰前的七十美元高出三成到四成。黃金繼續在歷史高位附近，快到四千八百美元，地緣不確定性讓避險需求持續存在。

加密貨幣方面，比特幣在六萬八千美元附近，漲跌不大，整體情緒還算穩定。但 Drift 駭客事件讓 Solana 生態的 TVL，也就是鎖倉總值，存在外流風險，這個影響短期之內還會持續。以太坊漲了一個多百分點，部分反映的是 go-ethereum 漏洞修復帶來的基礎設施信心回升——小事情，但市場確實注意到了。

再說一件加密圈的監管消息：美國財政部今天就《GENIUS Act》發布了正式規則提案。這是美國在穩定幣監管上，去年立法討論之後最具實質進展的一步——它在嘗試釐清聯邦和各州對穩定幣的監管分工到底是怎麼回事。看多的人說，法律框架清晰了，機構才有信心入場；看空的人，尤其是 DeFi 社群，擔心法規一旦定下來，適用範圍可能太廣，波及去中心化協議的運作方式。這個故事接下來幾個月會繼續發展。

好，今天就說到這裡。

你今晚可以繼續追蹤的事情：特朗普的全國演說，美東時間九點，這會決定伊朗這條線在接下來一兩週的走向；匈牙利大選還有十天，奧班還在選，對手的民調優勢還在——那邊的選情值得繼續關注；科技圈的話，就是等 Drift 的追款結果，還有 Claude Mythos 什麼時候真的讓一般人用到。
