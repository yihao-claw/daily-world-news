# EP45 每日新聞 Podcast — 2026年4月18日

嗨，週六早安，歡迎回來。今天是四月十八號，我是你的每日新聞主持人。

今天最重要的兩件事：第一，中東停火連環炮——以色列跟黎巴嫩停火了，伊朗也宣布荷莫茲海峽重新開放，油價一天暴跌超過一成；第二，科技圈大事，DeepSeek 的 R2 推理模型正式發布，一個三百二十億參數的小模型，居然在數學考試上打到九成二以上的分數。我們等一下慢慢聊。

先從中東講起。

你知道，過去這幾個禮拜，中東局勢就像是同時在下好幾盤棋。以色列跟黎巴嫩這盤棋，週四終於走出了一步——川普宣布雙方同意十天停火。貝魯特的民眾聽到消息，直接在街頭鳴槍慶祝，你可以想像他們有多壓抑。不過呢，停火墨水都還沒乾，黎巴嫩軍方就說以軍砲擊了南部好幾個村子。所以這個停火到底能撐多久，坦白說沒人敢打包票。

但真正讓全球市場震動的，是另一盤棋。伊朗外長 Araghchi 在社群媒體上發了一則聲明：荷莫茲海峽在停火期間全面對商船開放。你要知道，荷莫茲海峽是全球石油運輸的咽喉，全世界大概兩成的石油要經過那裡。它封了好幾個禮拜，全球油價飆到九十幾塊。所以這句話一出來，油價直接崩了。WTI 原油單日跌了將近百分之十二，從九十幾塊跌到八十三塊多，布蘭特也跌了超過八個百分點。這是六個禮拜以來最大的單日跌幅。

但你仔細看 Araghchi 的措辭，他講得非常清楚——海峽開放是有條件的，停火一結束就關回去。而美伊停火四月二十一號就到期了，只剩三天。第一輪伊斯蘭堡和談已經破裂了，二十一個小時的密集談判，副總統 Vance 最後出來說伊朗「拒絕接受我們的條件」。

核心僵局是什麼？就兩個字：鈾和海峽。美國要伊朗出清所有高濃縮鈾的庫存、永久開放荷莫茲海峽。伊朗呢，要美國解除所有制裁、解凍被凍結的數十億美元資產。雙方都不願意先讓步。

更讓事情複雜的是，國務卿 Rubio 在四月十八號直接喊話歐洲國家，說你們要趕快做決定、重新對伊朗制裁，因為伊朗正在逼近核武器的門檻。這話等於是在給歐洲施壓，要歐洲在美伊之間選邊站。

所以結論是什麼？這一波停火帶來的油價下跌和市場反彈，本質上是暫時的喘息。四月二十一號是真正的關鍵日——如果談判沒有突破，荷莫茲可能再度關閉，油價又要飆回去，全球的通膨壓力也會跟著回來。

往歐洲看。

匈牙利的故事真的很戲劇化。Péter Magyar 帶著他的 Tisza 黨拿下國會三分之二的絕對多數，終結了 Orbán 整整十六年的執政。他已經說新政府最快五月初就能上台，法律上新國會最遲五月十二號要召開並選出新總理。

但最有趣的地方在哪裡？Magyar 本身就是 Orbán 體制裡出來的人。他曾經是 Orbán 的忠實支持者，是體制內部人。所以他現在喊著要「清除 Orbán 體制安插的所有傀儡」，要總統 Sulyok 辭職、要改組國家媒體、要設立獨立的衛生部和環保部——整個歐洲都在問：這是真心改革，還是換一批人做同樣的事？

對歐盟來說，匈牙利過去十六年一直是最刺頭的成員國，動不動否決歐盟決議。如果 Magyar 真的「重返歐洲」，那歐盟的決策效率可能會有明顯提升。但對俄羅斯來說，這等於是在歐洲失去了最親密的朋友。Orbán 一直是莫斯科在歐盟裡面最好的盟友，他的下台對歐洲的能源政治跟對俄立場都會有連鎖反應。

俄烏那邊呢，復活節停火已經過期了，而且幾乎可以說是名存實亡。三十二小時的停火期間，烏克蘭軍方記錄了兩千兩百九十九次違規——包括砲擊和 FPV 無人機攻擊。俄羅斯說烏方違規一千九百七十一次。反正雙方都說是對方先動手。Zelensky 呼籲延長停火，但克宮直接拒絕，發言人 Peskov 說除非烏方接受俄方「眾所周知的條件」。戰爭已經進入第四年了，一千兩百公里的前線依然是軍事僵局，看不到盡頭。

東亞這邊有一個很值得你注意的長期趨勢。

日本跟韓國的核武裝討論正在認真加速。這不再是邊緣議題了。峨山政策研究院的民調顯示，百分之七十六的韓國民眾支持自主發展核武——四個人裡面有三個說贊成。日本方面也不是沒有動作，日本手上有大約四十五噸的鈽，技術上幾個月內就能製造核武器。

背後的邏輯很直白：美國的「美國優先」政策讓亞洲盟國越來越不確定核保護傘還靠不靠得住。如果美國連伊朗都搞得焦頭爛額，到時候中國真的對台海或者朝鮮半島出手，美國會不會真的用核武器保護盟友？這個問題以前大家覺得是假設性的，現在越來越多人認真在想了。

中國已經警告說可能經濟報復加軍事反制，北韓指控韓國走向「準核國家」。分析師開始認真討論東北亞「核骨牌效應」的可能性——一個國家核武裝，周邊國家就會跟進。這個趨勢值得持續關注。

台灣方面，北京在鄭麗文訪中之後推出了一系列善意措施，包括農漁產品銷售便利化和部分恢復赴台旅遊。但同一天，十六架中國軍機在台灣周邊活動。軟硬兩手並行，這個策略沒有變。另外巴西駐台代表公開說「巴西視台灣為中國一部分」，台灣外交部嚴正抗議。在國際外交空間越來越窄的情況下，這種公開宣示對台灣來說特別刺眼。

非洲有一個我覺得最需要被關注、但最容易被忽略的新聞。

蘇丹的戰爭進入第四年了。我每次講到蘇丹都覺得這些數字很沉重——一千四百萬人流離失所，兩千六百萬人面臨急性糧食危機，估計已經有四十萬人死亡。El Fasher 光是三天之內就有六千人遇難。聯合國的二十八億美元人道主義援助呼籲，到現在只募到百分之十六。國際社會在四月十五號的會議上承諾了十五億美元，但跟需求相比簡直是杯水車薪。UN 的高級人道救援官員直接用了「被遺棄的危機」這個詞。

跟烏克蘭、跟中東的國際關注度相比，蘇丹得到的資源和媒體版面極度不成比例。這是一個值得我們每個人反思的問題——為什麼有些危機能引發全球動員，有些就被遺忘了？

教宗 Leo XIV 正在進行他上任以來最大規模的非洲巡訪。他已經去了阿爾及利亞，現在在喀麥隆，接下來還會去安哥拉跟赤道幾內亞。他是第一位踏足阿爾及利亞的教宗。在 Bamenda 主持了一場跨宗教和平集會。整趟旅程聚焦和平、移民跟宗教對話。

還有一件讓人很心痛的事——土耳其連續兩天發生校園槍擊。四月十四號 Siverek 高中槍擊造成十六人受傷，隔天 Kahramanmaraş 的一所中學又發生槍擊，一名十四歲學生使用他父親——一名前警察——的槍枝，殺害了九人、傷了十三人。一名數學老師試圖用身體擋住學生，不幸殉職。土耳其之前幾乎沒有過校園槍擊事件，這兩天的悲劇對整個社會衝擊非常大。教師工會宣布全國罷工三天。

好，時事講到這裡，接下來聊科技。

今天科技圈最大的新聞，我覺得是 DeepSeek R2。

這是一個三百二十億參數的開源推理模型，用 MIT 授權釋出，意思是你可以拿去做任何事。它在 AIME 2025 數學基準上達到百分之九十二點七的正確率。AIME 是什麼？就是美國的高中數學邀請賽，每題都是多步驟的複雜推理。十五題裡面答對大概十四題。重點是，這個模型小到可以在一張二十四 GB 的消費級 GPU 上跑——就是你家的遊戲顯示卡就行了。API 定價比西方同類模型便宜大約七成。

之前洩漏的消息說 R2 會是一個一點二兆參數的巨型 MoE 架構，結果最後出來是三百二十億的 dense 模型。規模縮小了幾十倍，但表現反而更精準。DeepSeek 的做法是把智慧灌在後期訓練階段——特別是他們自己研發的 GRPO 強化學習管線——而不是靠堆參數。這再次證明：你不需要最大的模型才能有最好的推理能力。不過也有人質疑，這是不是意味著大規模預訓練其實遇到瓶頸了？這個問題目前沒有答案。

第二個大新聞是 Ollama v0.21.0 正式發布。

這次的亮點不只是版本號升級，而是它直接整合了 Nous Research 的 Hermes Agent。你只要在終端打一行指令 ollama launch hermes，就能在你的電腦上啟動一個會自動學習、自動建立技能的 AI 代理。它會根據你的工作方式來進化自己的能力。這是本地 LLM 工具第一次內建自我進化的 Agent 能力。對於重視隱私、不想資料離開電腦的使用者來說，這是一個里程碑。

OpenAI 那邊呢，代號「Spud」的新旗艦模型已經在三月底完成預訓練了，但到現在還是只聞樓梯響不見人下來。Sam Altman 說這是「非常強的模型」，Polymarket 上七成八的人押注四月底前發布。但一個月過去了，沒有任何技術細節。社群開始質疑：如果真的有所謂的代際飛躍，你為什麼不先秀基準成績出來？是不是結果沒有預期中那麼驚艷？我們只能繼續等。

Agent 生態方面，兩個重要進展。微軟正式發布 Agent Framework 1.0——這是一個穩定版，不是預覽版了——同時支援 .NET 跟 Python，內建 MCP 支援跟瀏覽器端的 DevUI 視覺化工具。Google 的 A2A 協議也滿一歲了，超過一百五十個組織加入，GitHub 星星數破兩萬二。微軟、AWS、Salesforce、SAP 都已經在生產環境部署。Agent 的基礎設施正在快速成熟，這是一個很健康的信號。

Snap 裁員一千人的新聞也值得一提。CEO Spiegel 直接說理由是 AI 快速進步。而且他給了一個數字——AI 現在已經生成了 Snap 超過百分之六十五的新程式碼。百分之六十五。這不再是「AI 會不會取代程式設計師」的假設性討論了，而是正在發生的事。

加密貨幣的部分，Bitcoin Core v31.0 正式版終於從 RC 階段畢業了。最大亮點是 Cluster Mempool，重新設計了交易池的排序方式，讓手續費估算更精確、區塊打包更有效率。這是 Bitcoin 核心協議近年最重要的基礎建設升級。另外 X 平台的 Cashtags 交易功能累計創造了超過十億美元的交易量，World ID 整合了 Tinder 的真人驗證跟 Zoom 的深偽檢測——身份認證從加密貨幣的小圈子跨出來，進入主流應用了。

資安方面簡短帶過：微軟四月 Patch Tuesday 修了一百六十八個漏洞，其中 SharePoint 的零日漏洞已經在被利用，四月二十八號前一定要更新。Fortinet 的 FortiClient EMS 也有一個從三月底就被攻擊的漏洞，請盡快修補。

來看一下市場。

美股在停火利多帶動下全面上漲——道瓊漲了將近百分之一點八，S&P 500 來到七千一百一十八點，Nasdaq 兩萬四千四百一十六點。歐洲 STOXX 600 也漲了百分之一點五六。亞洲今天週六休市，就不報了。

商品方面最大的焦點是原油暴跌——WTI 跌了將近百分之十二，Brent 跌了超過百分之八。黃金繼續創歷史新高，來到四千八百七十九塊七毛，避險需求持續強勁。

加密貨幣，比特幣站上七萬七千二，漲了接近百分之三，以太坊兩千四百二十二塊，漲超過百分之三。停火利多帶動風險資產全面反彈。

好，今天就到這裡。下週最重要的事：第一，四月二十一號美伊停火到期——如果談不攏，荷莫茲可能再度關閉，油價又要飆回去；第二，四月二十二號 ICC 對杜特蒂的管轄權裁定，決定這位前總統是否要面對國際審判。這兩件事都可能在下週帶來重大轉折。
