嗨，歡迎收聽每日世界脈動，我是你的主持人。今天是二零二六年四月十四號，星期二，EP41。

今天最重要的兩件事，一個是荷莫茲海峽封鎖正式生效，全球能源供應鏈面臨斷裂風險；另一個是匈牙利變天，Orbán 十六年執政畫下句點。我們等下會深入聊這些，還有科技圈這週炸裂的 AI 模型大亂鬥。

好，我們先從中東開始。

各位，這個週末真的是驚心動魄。上週三美國跟伊朗在巴基斯坦伊斯蘭堡坐下來談，大家都以為這次和談有戲，畢竟停火才剛生效不久。結果談了整整二十一個小時，到了週六，副總統乃磊出來宣布：談判破裂了。

然後川普的反應非常川普，他直接宣布美國海軍封鎖荷莫茲海峽，禁止一切船隻進出伊朗的港口。而且他還加碼說，任何伊朗的快攻艇敢接近封鎖線，美軍就直接擊沉。

這個決定的影響有多大？你要知道，全球大概百分之二十的石油跟天然氣都是經過荷莫茲海峽運輸的。封鎖一宣布，布蘭特原油期貨馬上跳漲了百分之十到十三，現在已經逼近一百美元一桶了。更嚇人的是聯合國糧農組織發出警告，說如果航運無法恢復，全球糧食危機恐怕幾週內就會爆發。因為化肥、尿素、石油這些農業關鍵物資全部被卡住了。摩根大通的分析師甚至說，西方國家的石油庫存可能在五月初就會觸及營運最低水位。

國際反應也很分歧。外長王毅說封鎖不符合世界的共同利益，呼籲國際社會加強和談。英國首相施凱爾明確表態不會參與封鎖行動，他說英國的重點是盡快重啟海峽。伊朗軍方則直接說這是海盜行為，違反國際法。

這件事的後續發展會怎樣？坦白說沒人知道。但你可以觀察幾個指標：一是油價有沒有真的突破一百美元，二是有沒有其他國家加入封鎖，三是伊朗會不會試圖突破封鎖線。如果這三個問題的答案都是「是」的話，那情況會升級得很快。

說到伊朗戰爭，有一個很有意思的支線故事。教宗 Leo XIV，也就是天主教史上第一位美國出生的教宗，他本名 Robert Prevost，芝加哥人，這幾天跟川普槓上了。Leo 在聖伯多祿大殿的守夜祈禱中，公開譴責川普對伊朗人民的威脅「真正不可接受」，批評所謂的全能妄想正在助長戰爭。川普的回應是什麼？他在社群平台上說教宗做得不好，然後分享了一張用人工智慧生成的圖片，把自己畫成耶穌的樣子在治癒病人。Leo 的回應非常硬氣，他說我不怕川普政府，也不怕大聲說出真相。

這其實反映了一個更深層的現象。當總統跟教宗公開交鋒的時候，你可以看到美國社會在戰爭議題上的撕裂有多嚴重。而且這不只是宗教和政治的衝突，更是兩種截然不同的世界觀在碰撞。一邊是以軍事力量為優先的強硬派，另一邊是以對話跟和平為核心的理想主義。

好，我們把目光轉向歐洲。

匈牙利在週六舉行了國會大選，結果是一場地震。Peter Magyar 領導的 Tisza 黨以百分之五十三點六的得票率，拿下一百九十九席中的一百三十八席，壓倒性的勝利。Orbán 的 Fidesz 只拿到五十五席，得票率百分之三十七點八。投票率超過百分之七十七，創下匈牙利在後共產時代的最高紀錄。

Magyar 在多瑙河畔對著數萬名支持者說了一句很有力量的話：今晚，真相戰勝了謊言。

這個結果對歐洲的影響是巨大的。Orbán 執政十六年來一直是歐盟內部最大的麻煩製造者。他擋烏克蘭的加入談判，擋對俄制裁，最近還擋住了歐盟九百億歐元的烏克蘭貸款。現在他下台了，布魯塞爾終於鬆了一口氣。烏克蘭方面，澤倫斯基第一時間就表示準備跟匈牙利發展合作關係。

說到烏克蘭，情況還是很嚴峻。他們的國庫可能在五月就見底，就是因為 Orbán 之前擋住了那筆九百億歐元的貸款。而且俄軍在復活節期間完全不顧基輔的停火呼籲，持續攻擊切爾尼戈夫州的能源設施，導致一萬兩千戶斷電。歐盟到目前為止已經對烏克蘭提供了將近一千九百五十億歐元的援助，這是一個天文數字。

東亞這邊，今天有一個比較正面的消息。中日韓文化交流年在東京開幕了，三國外長都出席。表面上是文化活動，但在地緣政治緊張的背景下，任何三方對話的機會都是值得關注的。

外長王毅上週訪問了北韓，北京和平壤在試圖修復疫情以來冷淡的關係。有趣的是，趁全球目光都聚焦在荷莫茲的時候，北韓同步測試了一系列新武器系統。日本這邊，國會議員在呼籲追加一兆日圓的預備金來應對能源危機，主要是擔心伊朗衝突會進一步推高全球能源價格，對日本這個能源進口大國來說衝擊特別大。

台灣的情況非常經典。北京的胡蘿蔔加大棒策略又來了。習近平跟國民黨的鄭麗文會面之後，中國宣布了一系列善意措施，包括促進台灣農漁產品銷往大陸、恢復部分赴台旅遊。但就在同一天，十六架共軍軍機繞台飛行。你說這是善意還是威嚇？大概兩個都是吧。這種軟硬兩手的策略北京玩了好幾十年了，台灣社會對此已經相當熟悉，但每次發生的時候還是讓人不太舒服。另外，賴清德總統準備訪問史瓦帝尼，那是台灣在非洲的最後一個邦交國了，外交空間持續受到壓縮。

南亞跟東南亞方面，世界銀行最新報告說南亞經濟成長今年會放緩到百分之六點三，去年是百分之七，主因就是中東能源市場的擾動。各國都在想辦法因應，越南在積極推太陽能發電，印尼則趁荷莫茲海峽關閉的機會向三個國家出口尿素，化危機為轉機。越南領導人蘇林今天開始訪問中國四天，這是他就任國家主席後的首次外訪，顯示越南在美中之間的平衡外交策略。

非洲有兩則需要關注的新聞。奈及利亞軍方在博爾諾跟約貝邊境打擊博科聖地的行動中，空襲誤中了一個市場，國際特赦組織說超過一百人罹難，三十五人重傷。這種所謂的附帶傷害在反恐行動中反覆出現，凸顯了軍事行動與平民保護之間的兩難。另一方面，埃及啟用了非洲最長的單軌鐵路，五百六十五公里，連接開羅及周邊地區，這是一個值得注意的基建里程碑，也是埃及經濟改革的一部分。

好，時事就聊到這裡。接下來我們來聊科技，這週科技圈真的是太精彩了。

首先最轟動的消息，Anthropic 發布了 Claude Mythos Preview。這個模型厲害在哪？它能夠完全自主地發現並利用所有主流作業系統和瀏覽器的零日漏洞。exploit 成功率達到驚人的百分之七十二點四，而上一代 Opus 4.6 的成功率幾乎是零。它甚至找到了一個存在了二十七年的 FreeBSD 遠端程式碼執行漏洞。

但 Anthropic 做了一個很不尋常的決定：他們不打算公開發布這個模型。取而代之的是一個叫 Project Glasswing 的計畫，把模型分享給 AWS、Apple、Google、Microsoft、NVIDIA 等十二家科技巨頭，讓他們優先修補漏洞。

這引發了很多討論。支持者說這是負責任的做法，先讓防守方有時間補洞。批評者則說，誰給了一家私人公司權力來決定什麼該發布什麼不該發布？而且這些被選中的合作夥伴全是美國公司，這到底是負責任的科技治理，還是另一種形式的技術壟斷？

然後 Google 這邊也放了一個大招：開源了 Gemma 4 模型家族。最大的版本只有三百一十億參數，卻在很多 benchmark 上打敗了參數量大十倍以上的封閉模型。而且它用的是 Apache 2.0 授權，完全免費商用。有四個版本可以選，從手機到雲端都能跑。支援兩百五十六K 的 context window，原生視覺和音頻處理，一百四十多種語言。這對開發者社群來說真的是一份大禮。

中國的智譜 AI 也不甘示弱，發布了 GLM-5.1。七百四十四億參數的混合專家架構，但實際運算只用四百億。用 MIT 授權開源，據稱在 SWE-Bench Pro 上超越了 Claude Opus 4.6 和 GPT-5.4。不過我要提醒一下，獨立復現的結果還沒出來。我們在看 benchmark 排名的時候要保持一點懷疑精神，因為有些模型會針對特定 benchmark 做優化，實際使用的體驗未必一樣。

Meta 也來了，發布了 Llama 4 Scout，十七億參數的視覺語言模型，只要一張二十四 GB 顯存的消費級 GPU 就能全速跑。OpenAI 則是擴展了 Responses API，加了 shell 工具、agent 執行迴路跟 context compaction，很明顯在往 agentic workflow 的方向全力推進。Mistral 也發了 Large 3 旗艦模型。四月真的是 AI 模型的爆發月。

Agent 框架方面，Microsoft 正式發布了 Agent Framework 1.0 的 production-ready 版本，支援 .NET 和 Python，內建了 A2A 和 MCP 協定的互操作性，可以讓不同廠商的 agent 互相溝通。Google 也推出了兩個學術 AI Agent：一個自動畫論文圖表，一個自動評審論文。但有一個數據很值得深思：OutSystems 的調查顯示，百分之九十四的企業對 AI Agent 的擴散表示擔憂，主要是擔心 agent sprawl 帶來更多複雜度、技術債和安全風險。

資安方面也有不少事情。Citrix NetScaler 的零日漏洞正在被積極利用，Chrome 今年已經修補了第四個零日漏洞。北韓的 APT37 駭客團體在 Facebook 上用社交工程手法散佈木馬程式，手法越來越精細。而 Anthropic 的 Mythos 模型發現零日漏洞的能力，更是讓整個資安社群都在重新思考 AI 在攻防對抗中的角色。

太空方面有個好消息。NASA 的 Artemis II 太空人在四月十號安全返回地球了。這是半個世紀以來，第一批前往月球的太空人，寫下了太空飛行的新紀錄。天文學家也發表了新的哈伯常數測量結果，精度突破百分之一，數值是每秒每百萬秒差距七十三點五公里。但弔詭的是，測量越精確，宇宙膨脹的謎團反而越深了，因為不同方法測出來的數字就是對不上。

生技領域，Gilead Sciences 今年砸了將近一百五十億美金做了三筆收購，全都聚焦在抗體藥物複合體這個癌症治療的熱門領域。Anthropic 則用四億美金收購了一家不到十人的生技 AI 公司，顯示 AI 大廠正在把觸角伸向藥物研發。

最後來看一下市場的數字。

加密貨幣方面，比特幣在美伊和談破裂後一度下跌，但昨天已經反彈到七萬四千六百七十五美元，日漲超過百分之五。以太幣表現更好，漲了超過百分之八，到兩千三百七十七美元。

美股方面，昨天也就是週日的期貨交易，S&P 500 在六千八百八十六點，漲了大約百分之一。Nasdaq 漲了百分之一點二。不過亞洲股市跟歐股因為週末休市，數據要等今天正式開盤才會更新。

匯率方面，今天的即時報價，美元兌日圓一百五十九點二三，歐元兌美元一點一八，美元兌台幣三十一點六四。

商品方面，WTI 原油在九十六點八美元，Brent 原油九十七點三二美元。黃金維持在四千七百八十一美元的高位，避險需求明顯。

好，最後幫你快速整理一下今天的重點。第一，荷莫茲海峽封鎖已經生效，油價逼近一百美元，糧食供應鏈面臨壓力。第二，匈牙利選出新政府，歐盟對烏克蘭的支持有望加速。第三，教宗跟川普的公開衝突反映了美國社會的深層分裂。科技方面，Anthropic 的 Mythos 模型讓資安世界震動，Google 跟中國智譜的開源模型則在推動 AI 民主化。今天的節目就到這邊。荷莫茲封鎖接下來怎麼演變、油價會不會突破一百美元、匈牙利新政府上台後歐盟對烏克蘭的政策調整、還有 AI 模型的軍備競賽會走向何方，這些都是接下來幾天要密切關注的重點。感謝收聽，我們明天見。
