# EP43 — 2026年4月16日

嗨，早安，歡迎收聽每日新聞快報第四十三集。今天是四月十六號，星期四。美國跟伊朗的停火倒數剩不到一個禮拜，匈牙利經歷了一場歷史性的政治地震，以色列跟黎巴嫩在華盛頓的和談第一天就陷入僵局。今天內容很多，我們直接開始。

先從最緊迫的事情講起——美國跟伊朗之間的停火協議，四月二十一號就要到期了。現在全世界都在看一件事：第二輪和談到底能不能在到期之前重啟？

先幫你回顧一下背景。上一輪談判是在巴基斯坦的伊斯蘭堡舉行的，副總統 Vance 帶著特使 Witkoff 跟 Kushner 親自上陣。但談判在上週末破裂了，核心分歧非常清楚也非常根本：美國要求伊朗完全終止核計畫，交出將近一千磅的濃縮鈾，而且必須無條件開放荷莫茲海峽的通行權，不能收過路費。伊朗當然不可能接受這種他們眼中的「投降式」條件。德黑蘭的立場是，你要我讓步可以，但美國必須先解除制裁，而且這必須是一場對等的協商，不是一場最後通牒。

那現在有什麼新進展呢？巴基斯坦的元帥 Asim Munir 已經飛到德黑蘭了，跟他一起去的還有巴基斯坦的內政部長 Moshin Naqvi。你可以看到伊斯蘭堡在這件事上投入的外交資源有多大——元帥加部長級的陣容，目的就是要說服伊朗回到談判桌。白宮那邊的態度滿樂觀的，川普自己也放話說新一輪的面對面談判可能這兩天就會安排。

而華爾街呢？華爾街顯然已經先投票了。S&P 500 昨天收在七千零二十三點，漲了將近一個百分點。Nasdaq 更猛，收在兩萬四千零十六點，漲了一點六個百分點，雙雙創歷史新高。你想想看，這代表什麼？代表市場已經把伊朗戰爭以來所有的恐慌跌幅全部吃回來了。投資人在賭和平。不過我要提醒你，道瓊指數其實小跌了零點一五個百分點，所以不是所有指數都在歡呼——工業股跟傳統產業的投資人還是有些保留的。

接下來聊一個讓整個歐洲都在討論的事——匈牙利大選。

這場選舉的結果，用「政治地震」來形容一點都不誇張。Viktor Orbán，執政十六年的強人，終於被選民趕下台了。贏家是 Tisza 黨的領袖 Péter Magyar，他拿下了百分之五十三點六的選票。你可能會說，五成多，贏得很漂亮但也不算特別離譜吧？關鍵在席次：在一百九十九個國會席位裡面，Tisza 黨拿了一百三十八席，Fidesz 只剩五十五席。一百三十八席代表什麼？代表修憲門檻的三分之二絕對多數。也就是說，Magyar 不只是要組閣，他理論上有能力修改憲法。

投票率是另一個值得注意的數字——百分之七十七點八，是匈牙利一九九零年民主化以來的最高紀錄。上一次接近這個數字是二零零二年，當時才百分之七十點五。這麼高的投票率說明什麼？說明匈牙利的選民真的被動員起來了，他們對改變有非常強烈的渴望。

歐盟執委會主席 von der Leyen 第一時間發了一條推文，就五個字——「匈牙利選擇了歐洲」。這句話背後的意思很深。過去十六年，Orbán 基本上是歐盟裡面最不聽話的那個——跟普丁眉來眼去，在移民問題上跟布魯塞爾對著幹，不斷挑戰歐盟的法治原則。匈牙利有一大筆歐盟發展基金因為法治爭議被凍結了好幾年。現在 Magyar 上台，這些錢有望解凍，但 Orbán 留下的制度遺產——控制的媒體、被安插的司法系統、各種有利於 Fidesz 的選舉法規——這些不是一天兩天能拆的。而且別忘了，Russian 方面也在看這件事——莫斯科剛失去了在歐盟裡面最親密的朋友。

第三個頭條是以色列跟黎巴嫩在華盛頓的和談。四月十四號，美國國務卿 Rubio 主持了一場三方會談，出席的有以色列大使、黎巴嫩大使，還有白宮的 Needham 跟美國駐聯合國大使 Waltz。聽起來陣容不小，但坦白講，這場會談的第一天就已經碰壁了。

為什麼？因為雙方對「第一步應該做什麼」的想法完全相反。以色列的立場非常明確也非常強硬：不談停火。先把真主黨的武裝解除，確立和平關係的框架，然後才能討論其他事情。而黎巴嫩呢？黎巴嫩要求的是立即停火跟人道走廊。這不難理解，到目前為止，這場衝突已經在黎巴嫩造成超過兩千一百二十四人死亡，一百一十萬人被迫離開家園。

更讓黎巴嫩方面憤怒的是四月八號發生的事情。當天，在停火宣布之後不久，以色列對黎巴嫩發動了被稱為「黑色星期三」的大規模空襲行動。一天之內，三百五十七人喪生。黎巴嫩政府稱之為「屠殺」。你可以想像，在這樣的背景下，黎巴嫩要坐下來跟以色列談和平，國內的政治壓力有多大。

再來看歐洲的另一個戰區。俄烏之間的東正教復活節停火已經正式結束了。普丁宣布了三十二小時的停火，從四月十一號下午四點到十二號午夜。結果呢？雙方互相指控對方數千次違規。烏克蘭的總參謀部統計，俄軍在停火期間違反了兩千兩百九十九次，包括各種無人機攻擊、小型迫擊砲跟步兵突擊。俄羅斯那邊也不甘示弱，說烏軍違規了一千九百七十一次，其中包括兩百五十八次砲擊、一千三百二十九次 FPV 無人機攻擊。

克里姆林宮的發言人佩斯科夫直接拒絕了延長停火的提議，說除非澤倫斯基接受俄方「眾所周知的條件」。白話翻譯就是：你先承認克里米亞是俄羅斯的、承認頓巴斯的現狀，我們再來談。所以這條路基本上就是死胡同。

把視線轉到亞洲。兩岸關係這幾天又有新的發展。國民黨主席鄭麗文上週訪問中國之後，北京宣布了一系列「善意措施」。具體來說，恢復更多城市到台灣的直航——包括西安跟烏魯木齊這種以前沒有直飛的城市——還有重新開放台灣水產品進口。你可能記得，中國在二零二二年因為佩洛西訪台而中斷了這些往來，現在算是恢復了一部分。

不過台灣政府的反應非常明確。陸委會直接用了「政治交易」這個詞來批評，說這是繞過政府的黨對黨私下安排，強調所有涉及公權力的兩岸事務都必須由兩個政府在對等跟尊嚴的基礎上來協商。美國在台協會也表態了，說中國應該放棄對台灣的軍事威脅，跟台灣的民選領導人直接對話，這樣才能避免誤判、穩定兩岸關係。

另外一個值得關注的發展——台灣正在準備歷史上第一次反封鎖護航演習。這次是內政部跟國防部聯合主導的跨部會演練，模擬的情境是：如果中國真的對台灣實施海上封鎖，怎麼護送載著天然氣跟石油的運輸船安全抵達港口。這個演習的政治訊號非常清楚——台灣正在認真準備應對最壞的情境。

東北亞安全局勢也在重組。日本跟韓國決定把兩國的安全對話升級為「二加二」副部長級會談機制，預計五月在首爾首次舉行，外交加國防高層都會出席。同時日本跟澳洲四月八號也重申要深化防衛合作。背後的驅動力很明顯：中東衝突讓能源供應變得不穩定，而美國的「美國優先」外交路線讓亞太盟國開始思考——萬一哪天美國的安全保護傘不那麼可靠了，我們自己該怎麼辦？日韓兩國內部關於獨立核武能力的辯論也因此升溫了。

再看非洲。教宗 Leo 十四世正在進行他就任以來最長的一次海外之旅——十天，橫跨四個國家：阿爾及利亞、喀麥隆、安哥拉和赤道幾內亞。他已經結束了在阿爾及利亞歷史性的兩天訪問——這是史上第一位到訪阿爾及利亞的教宗——昨天轉抵喀麥隆。今天的重頭戲是在喀麥隆西北部的巴門達舉行一場跨宗教和平集會，傳統酋長、長老會牧師、伊斯蘭教伊瑪目和天主教修女都會一起出席。週五在杜阿拉的彌撒預計會有六十萬人到場。整趟旅程總計一萬八千公里、十八趟航班、十一座城市，主題圍繞和平、移民、環境、青年跟家庭。

同時在非洲的另一端，蘇丹的戰爭已經進入第四年了。聯合國最高人道援助官員用了一個很沉痛的詞來形容——「被遺棄的危機」。注意，不是「被遺忘」，是「被遺棄」。因為全世界其實知道蘇丹在發生什麼，只是選擇不去管。數字是這樣的：一千四百萬人流離失所，三千三百七十萬人——也就是蘇丹三分之二的人口——需要人道援助，估計有四十萬人已經死亡。今年的二十八億美元援助呼籲只到位了百分之十六。查德這個鄰國收容了超過一百三十萬蘇丹難民，面臨四億兩千八百萬美元的資金缺口。百分之八十的流離家庭被迫減少進食。這是當今世界規模最大的人道危機，但它得到的關注遠遠不夠。

大洋洲也有一則值得注意的外交新聞。紐西蘭跟庫克群島簽署了一份防衛跟安全宣言，算是化解了之前因為中國因素引發的緊張關係。去年庫克群島總理 Brown 跑去北京簽了一份全面戰略夥伴協議，但沒有事先告知紐西蘭內容，搞得惠靈頓非常不高興，甚至暫停了對庫克群島的發展援助。現在這份新宣言的核心條款是：庫克群島承諾紐西蘭是他們在「防衛跟安全事務上的首選夥伴」。換句話說，安全合作這一塊，紐西蘭優先於中國。

好，接下來我們轉到科技新聞。

今天科技圈最值得關注的消息是 Google 正式發布了 Gemini 三點一 Flash TTS。你可能會說，語音合成模型不是什麼新鮮事了，但這次真的不太一樣。三個重點：第一，它支援超過兩百個音訊控制標籤，你可以非常精細地調整語氣、節奏、停頓、甚至情緒表達，比以前的 TTS 模型可控性高了一個量級。第二，原生支援多講者對話。以前你要做一段兩個人對話的語音，得分別呼叫 API 再拼接起來，節奏常常很奇怪。現在直接在一次呼叫裡處理多個角色，對話的節奏跟互動感自然得多。第三，支援七十多種語言。而且所有生成的音訊都內建 SynthID 浮水印，可以偵測是不是 AI 生成的。在語音合成的排行榜上拿到了 Elo 一千兩百一十一分。這對做 podcast、有聲書、或者是 AI 助理對話的開發者來說是一個相當大的升級。

程式語言方面有一個重要的釋出——Zig 零點十六正式版。這個版本開發了八個月，有兩百四十四位貢獻者參與，總共一千一百八十三個 commit。三大改進：第一是非同步 I/O 全面改寫，新的標準函式庫介面在 Linux 上用 io_uring、macOS 上用 GCD，統一了不同平台的非同步模型，而且消除了所謂的「函式著色」問題——就是以前 async 函式跟普通函式不能混用的麻煩。第二是型別解析的重新設計，三月十號才合併進主分支的，大幅減少了不必要的分析步驟，增量編譯變得更快了。第三是套件管理簡化，依賴項改放在本地的 zig-pkg 目錄，搭配全域壓縮快取。對系統程式語言的生態來說，這是一個很紮實的進步。

大型語言模型的競賽又有新動態。Anthropic 的 Claude Opus 四點六在 LMSYS 的 Chatbot Arena 排行榜上登頂了，在人類偏好評估中超過了 OpenAI 的 GPT 五點四和 Google 的 Gemini 三點一 Pro。特別值得注意的是它在 SWE-bench Verified 上達到百分之六十五點三的成績，這是衡量 AI 能不能真正解決真實世界軟體工程問題的指標。另外 Mistral AI 也發表了他們的 Large 3 旗艦模型，主打結構化輸出跟 function calling 精準度的提升，還提供歐盟資料駐留選項。

資安方面，微軟四月的 Patch Tuesday 是今年最大的一次修補行動——一百六十七個漏洞。裡面有兩個零日漏洞需要特別注意。一個是 SharePoint Server 的偽造漏洞，已經被發現在野外被利用了。另一個是 Microsoft Defender 的提權漏洞，編號 CVE-2026-33825，可以讓攻擊者拿到 SYSTEM 等級的權限，已經在四點一八版修復。CISA 也同步把 Fortinet FortiClient EMS 的 SQL 注入漏洞等六個已遭利用的弱點列入強制修補清單，聯邦機構必須在四月二十七號之前完成修補。

另外有一個特別惡劣的案例——有人在 Apple 的 App Store 上架了一款假冒的 Ledger Live 應用程式。如果你不知道 Ledger 是什麼，它是一個很受歡迎的硬體加密貨幣錢包品牌。這個假 app 在短短幾天之內就騙走了五十名受害者總共九百五十萬美元的加密貨幣。Apple 已經把它下架了，但損失已經造成。這再次提醒我們，就算是在 App Store 這種有審核機制的平台上，也不能完全放心。

生物科技領域，整個產業繼續加速擁抱 AI。Novo Nordisk——就是做減肥藥 Ozempic 那家——跟 OpenAI 宣布合作，用大型語言模型來分析複雜的藥物數據集、識別有潛力的新藥分子。GSK 跟 Eli Lilly 也分別跟 AI 新創公司簽了合作協議，GSK 前期就投了五千萬美元。NVIDIA 跟 Lilly 還更進一步，直接成立了一個 AI 藥物共創實驗室。產業觀察家的共識是：大型藥廠已經從「試試看 AI 能不能用」的階段，進入了「全面整合 AI 到研發流程」的階段。
