早安，各位聽眾朋友，今天是四月六號，星期一，歡迎收聽每日新聞 Podcast 第三十七集。

今天有幾件大事要跟你聊。國際局勢方面，美伊戰爭進入第三十七天，川普把荷姆茲海峽的最後通牒又往後延了一天，改成週二晚上。但更戲劇性的是，被擊落的美軍飛行員竟然在伊朗境內躲了超過二十四小時之後被成功救出。中東那邊，以色列在黎巴嫩的軍事行動造成更多傷亡。另外，新任教宗良十四世發表了他第一次復活節文告，呼籲世界放下武器。科技圈這邊呢，今天是 Artemis II 繞月的大日子，還有北韓駭客用了六個月的時間滲透一個 DeFi 協議偷走了將近三億美元。我們一個一個來聊。

先來看美伊戰爭。你可能還記得，上週川普下了一個四十八小時的最後通牒，要求伊朗重新開放荷姆茲海峽，否則要讓伊朗「活在地獄裡」。原本這個期限應該是週一到期，但週日的時候川普接受華爾街日報專訪，又把期限延到了週二晚間八點東部時間。他的原話是：「如果他們週二之前不做點什麼，他們就不會有任何電廠了，也不會有任何橋還站著。」

伊朗那邊怎麼回應呢？高級官員直接說，海峽在戰爭賠償全額到位之前不會重開，而且把川普的最後通牒形容為「純粹的絕望和憤怒」。有趣的是，雙方都在嘴硬，但實際上幕後已經有人在斡旋了。阿曼過去兩週一直在跟伊朗秘密談判一個叫做「安全通行協議」的東西，試圖讓商業船隻恢復通過海峽。這可能是目前最有希望打破僵局的管道。

但這場戰爭最戲劇性的一幕是什麼？上週四被擊落的那架 F-15E，其中一名機組員，武器系統官，在被擊落後竟然在伊朗境內躲了超過二十四小時，最後被美軍特種部隊成功救了出來。川普在社群媒體上大寫字母打了一句：「我們把他救回來了！」你可以想像這對美國國內的士氣來說是多大的提振。但反過來看，一架 F-15E 被擊落這件事本身，對美軍來說是一個相當嚴肅的警訊。

整體來看，這場戰爭已經打了超過五週。荷姆茲海峽通行量從每天一百五十艘船暴跌到十到二十艘，全球百分之二十的石油供應受到影響，WTI 原油已經衝破了一百一十二美元。而且 OPEC+ 昨天在維也納開了一次被稱為「聯盟成立以來最重要」的部長級會議，討論的是要不要維持四月起每日增產二十萬桶左右的計畫。你想想，這個背景下增產的壓力有多大——一方面是海峽封鎖導致全球供應短缺，另一方面是沙烏地和俄羅斯各有各的盤算。這場會議的結果會直接影響接下來油價怎麼走，我們持續關注。

順帶一提，伊朗戰爭的衝擊已經遠遠超出了中東。上週我們提到南亞和東南亞的燃油危機，這個情況還在惡化。全球糧食價格三月份連續第二個月上漲，FAO 的數據顯示主要是因為中東衝突推高的能源成本傳導到了食品供應鏈。非洲多國的燃油價格漲幅動輒百分之十五到百分之三十以上，對當地民眾的生活壓力可想而知。

說到中東，以色列在黎巴嫩的軍事行動也在持續升級。光是昨天一天，以色列對黎巴嫩南部和貝魯特南郊的空襲就造成至少十四人死亡，其中包括南部一家六口全部遇難。有一枚炸彈落在距離拉菲克哈里里大學醫院只有一百公尺的地方，造成三十九人受傷。自從三月二號行動開始以來，黎巴嫩已經有超過一千四百人喪生，超過一百二十萬人流離失所。真主黨宣稱向一艘距離海岸一百二十六公里的以色列軍艦發射了一枚巡弋飛彈，但這個說法沒有被獨立證實。

NBC 報導指出，衛星影像顯示以軍在南黎巴嫩造成的破壞程度，引發了外界對「加薩模式」正在被複製的擔憂。而以色列方面的戰略目標，根據美國智庫 FDD 的分析，已經從原來的「解除真主黨武裝」悄悄轉變為「重建南黎巴嫩安全區」。這兩個目標聽起來差不多，但在實際操作上意味著長期軍事佔領。

轉到歐洲，匈牙利大選只剩下六天了。但有趣的是，各家民調的數據差得離譜。反對派 Tisza 黨在非官方民調中領先多達十九個百分點，但親政府的民調卻顯示執政黨 Fidesz 領先六個百分點。政治學者 Gábor Török 說，政府民調和非政府民調之間出現這麼大的差距，是匈牙利政治史上前所未見的。四月十二號開票的時候，到底哪邊的民調比較準，我們拭目以待。如果奧班真的輸了，對歐盟的外交和國防決策效率會是一個巨大的利好——他的常態性否決權長期癱瘓了歐盟的運作。

另外還有一個值得一提的消息。教宗良十四世在聖伯多祿廣場主持了他就任以來的第一次復活節彌撒。他的 Urbi et Orbi 文告相當直白：呼籲「擁有武器的人放下武器，擁有權力的人選擇以對話實現和平」。他也明確批評了當前的中東戰事，警告世界正在「對戰爭暴力變得麻木」。作為一位新教宗，這個立場算是相當鮮明的。

美國國內也有一則重要人事變動。川普解除了司法部長 Pam Bondi 的職務，據報是因為不滿她處理 Epstein 文件的方式以及沒有足夠積極地起訴他的政治對手。副部長 Todd Blanche 出任代理司法部長——如果你有印象的話，他就是之前撰寫那份加密貨幣執法備忘錄的人，所以加密產業對這個人事變動的解讀是偏正面的。

最後在西半球，古巴宣布特赦兩千零十名囚犯，這是復活節期間的人道主義釋放，也是古巴自二零一一年以來的第五次大規模特赦。但背景是美國持續對古巴施壓，所以這個動作也被解讀為古巴對華盛頓釋出的善意信號。

好，聊完國際局勢，我們來看看科技圈今天有什麼動靜。

首先是今天的重頭戲——Artemis II 繞月！今天是這趟任務最令人期待的一天。四名太空人，NASA 的 Reid Wiseman、Victor Glover、Christina Koch 和加拿大太空人 Jeremy Hansen，他們今天會近距離飛越月球，觀測從未被人類直接目睹的月球背面區域。飛越期間從美東時間下午兩點四十五分到晚上九點四十分，其中大約下午五點四十七分開始，飛船會繞到月球背面，經歷大約四十分鐘的計畫性通訊中斷。更酷的是，在飛越尾聲，太空人還會目睹一場太空日食——從他們的視角看，太陽會在月球後方消失將近一個小時。這是一九七二年 Apollo 十七號以來，人類距離地球最遠的一次飛行，最遠達四十萬六千七百七十三公里。如果你對太空有一點點興趣，今天 NASA+ 有全程直播，非常推薦去看。

科技新聞的第二個大故事是 Drift Protocol 的駭客事件後續。之前我們報導過 Solana 上的 DeFi 協議 Drift 在四月一號被駭了大約兩億八千五百萬美元。昨天 Drift 發布了完整的事件報告，確認這是北韓國家級駭客組織 UNC4736 花了整整六個月精心策劃的滲透行動。

你聽到的沒錯，六個月。這些攻擊者假扮成一家量化交易公司，在好幾個國際會議上跟 Drift 的貢獻者面對面見面、握手、交換名片。他們甚至存了超過一百萬美元的自有資金進入 Drift 的生態系統，建立了一個看起來完全正常的 Ecosystem Vault。然後通過一個惡意的 TestFlight 應用和 VSCode 的漏洞，取得了多重簽名的授權。預先簽好的交易在系統裡潛伏了超過一週，然後在四月一號那天，不到一分鐘就把兩億七千萬美元掏空了。

更驚人的是，那些在會議上跟人面對面的人，並不是北韓國民。根據調查，北韓的這些高階威脅行動者會部署第三方中間人，這些人有完整的假身分、假工作經歷、甚至假的職業人脈網絡，可以通過正常的盡職調查。Drift 在報告中警告，這種長期身分偽造行動暴露了整個 DeFi 多重簽名安全模型的深層弱點。

另一個值得關注的消息是 Microsoft 開源了 Agent Governance Toolkit。這是一個七模組的系統，用來治理自主 AI Agent 的行為，號稱是第一個完整覆蓋 OWASP Agentic AI 全部十項風險的開源框架。核心亮點是亞毫秒級的策略引擎——也就是說每一個 Agent 的動作在執行之前都會先過一道策略檢查，而這個檢查的延遲不到一毫秒。支援 Python、Rust、TypeScript、Go、.NET 五種語言，已經整合了 LangChain、CrewAI、OpenAI Agents SDK 等主流框架。如果你在做 AI Agent 的生產部署，這個工具值得認真看一下。

這裡面有一個大趨勢值得你注意：AI Agent 從去年的概念驗證，到今年已經開始進入大規模生產部署。但問題是，當你讓 AI Agent 自主去執行任務的時候，你怎麼確保它不會做出你不希望它做的事？這就是為什麼像 Microsoft 這樣的公司要推出治理框架。根據產業預測，到今年年底，會有百分之四十的應用程式使用某種形式的任務專用 AI Agent。這個數字聽起來很驚人，但如果你有在關注這個領域的發展速度，其實不意外。

在 LLM 領域，Anthropic 的 Model Context Protocol 在三月份突破了九千七百萬的安裝量。這個數字代表什麼？代表 MCP 已經從一個實驗性的協議，真正轉變成了 AI Agent 生態系統的基礎設施層。所有主要的 AI 供應商現在都在出貨 MCP 相容的工具。另外 Anthropic 也調整了 Claude 的計費方式：從四月四號起，使用 OpenClaw 等第三方工具不再包含在 Claude 訂閱額度內，改為另行計費。

網路安全方面，有兩個值得注意的消息。Cisco 的 Integrated Management Controller 被發現有一個 CVSS 九點八分的最高嚴重度漏洞，未經授權的攻擊者可以透過特製的 HTTP 請求直接更改任何使用者的密碼，包括管理員。另外 Kaspersky 發現有三款偽裝成企業通訊和外送 App 的惡意軟體混進了 App Store 和 Google Play，它們會靜默掃描你手機照片裡的加密貨幣錢包助記詞圖片。如果你有把助記詞拍照存手機的習慣，強烈建議你趕快刪掉。這種攻擊手法其實蠻聰明的，因為很多人為了方便會把助記詞拍照存在手機相簿裡，但他們不知道的是，很多 App 會請求相簿存取權限，而你通常不會多想就按了允許。一旦惡意 App 拿到這個權限，它就可以用 OCR 技術掃描你所有的照片，找到看起來像是助記詞的文字。所以最安全的做法是：助記詞永遠寫在紙上，不要存在任何電子設備裡。
